# <a name="compiler-warning-level-1-cs5000"></a>Advertencia del compilador (nivel 1) CS5000

Opción del compilador '/option' desconocida

 Se especificó una [opción del compilador](../language-reference/compiler-options/index.md) no válida.

> [!NOTE]
> Este error del compilador ya no se usa en Roslyn.
