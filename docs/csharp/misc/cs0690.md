# Error del compilador CS0690
El archivo de entrada 'file' contiene metadatos no válidos.  
  
 Puede abrir el archivo de metadatos, pero debido a algún problema de localización, los metadatos están dañados. Esto es similar al error [CS0009](../../csharp/misc/cs0009.md), salvo que no se puede abrir el archivo de metadatos.  
  
 Un archivo PE \(ejecutable de proceso\) es un archivo ejecutable.
