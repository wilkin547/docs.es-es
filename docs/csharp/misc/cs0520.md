# <a name="compiler-error-cs0520"></a>Error del compilador CS0520
El tipo predefinido 'name' se declaró incorrectamente  
  
 No se encontraron uno o más archivos necesarios para el compilador. Reinstale .NET Framework Common Language Runtime.
