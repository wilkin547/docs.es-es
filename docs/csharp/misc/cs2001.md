# <a name="compiler-error-cs2001"></a>Error del compilador CS2001

No se encontró el archivo de origen 'file'.  
  
 Se pasó un nombre de archivo de origen al compilador, pero no se pudo encontrar. Compruebe la ortografía del nombre de archivo y la ubicación del archivo.
