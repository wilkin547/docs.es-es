# <a name="compiler-error-cs1509"></a>Error del compilador CS1509
El archivo 'archivo' al que se hace referencia no es un ensamblado; use la opción '/addmodule' en su lugar.  
  
 Un archivo de salida (archivo de salida 1), generado en una compilación que usa [/target: module](../language-reference/compiler-options/target-module-compiler-option.md) (no tiene un manifiesto de ensamblado), se especificó como [referencia](../language-reference/compiler-options/reference-compiler-option.md). Por lo tanto, en lugar de agregar un ensamblado al ensamblado para el programa actual, la información de metadatos del archivo de salida 1 se agregará al ensamblado para el programa actual.
