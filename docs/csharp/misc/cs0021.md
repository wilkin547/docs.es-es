## <a name="example"></a><span data-ttu-id="5f049-109">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="5f049-109">Example</span></span>

<span data-ttu-id="5f049-110">El siguiente ejemplo de C++ se compila en un archivo. dll.</span><span class="sxs-lookup"><span data-stu-id="5f049-110">The following C++ example compiles to a .dll file.</span></span> <span data-ttu-id="5f049-111">Tenga en cuenta que el atributo [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) se marca como comentario para generar el error.</span><span class="sxs-lookup"><span data-stu-id="5f049-111">Note that the [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) attribute is commented out in order to generate the error.</span></span>

```cpp
// CPP0021.cpp
// compile with: /clr /LD
using namespace System::Reflection;
// Uncomment the following line to resolve
//[DefaultMember("myItem")]
public ref class MyClassMC
{
        public:
        property int myItem[int]
        {
            int get(int i){  return 5; }
            void set(int i, int value) {}
        }
};
```

<span data-ttu-id="5f049-112">En el siguiente ejemplo de C# se llama al archivo. dll.</span><span class="sxs-lookup"><span data-stu-id="5f049-112">The following C# example calls the .dll file.</span></span> <span data-ttu-id="5f049-113">Intenta tener acceso a la clase a través de un indexador, pero dado que no se declaró ningún miembro como el indexador predeterminado, se genera el error.</span><span class="sxs-lookup"><span data-stu-id="5f049-113">It attempts to access the class via an indexer, but because no member was declared as the default indexer, the error is generated.</span></span> <span data-ttu-id="5f049-114">Puede solucionar el error si quita la marca de comentario de la línea `[DefaultMember("myItem")]` .</span><span class="sxs-lookup"><span data-stu-id="5f049-114">You can address the error by uncommenting the line `[DefaultMember("myItem")]`.</span></span>
