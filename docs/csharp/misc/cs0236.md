# Error del compilador CS0236
Un inicializador de campo no puede hacer referencia al campo, método o propiedad no estáticos 'campo'.  
  
 Los campos de instancia no pueden usarse para inicializar otros campos de instancia fuera de un método. Si está intentando inicializar una variable fuera de un método, considere la posibilidad de realizar la inicialización dentro del constructor de clase. Para obtener más información, consulta [Métodos](../../csharp/programming-guide/classes-and-structs/methods.md).  
  
 El ejemplo siguiente genera la advertencia CS0236:  
  
```  
// CS0236.cs public class MyClass { public int i = 5; public int j = i;  // CS0236 public int k;      // initialize in constructor MyClass() { k = i; } public static void Main() { } }  
```
