# <a name="compiler-error-cs1672"></a>Error del compilador CS1672

Opción 'opción' no válida para /platform; debe ser anycpu, x86, Itanium o x64  
  
 Las opciones especifican el tipo de procesador. Use uno de los valores indicados.
