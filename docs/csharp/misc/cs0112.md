# Error del compilador CS0112
Un miembro estático 'function' no se puede marcar como override, virtual o abstract  
  
 Una declaración de método que utilice la palabra clave `override`, **virtual** o **abstract** no puede usar también la palabra clave **static**.  
  
 Para obtener más información, consulta [Métodos](../../csharp/programming-guide/classes-and-structs/methods.md).  
  
 El ejemplo siguiente genera la advertencia CS0112:  
  
```  
// CS0112.cs namespace MyNamespace { abstract public class MyClass { public abstract void Foo(); } public class MyClass2 : MyClass { override public static void Foo()   // CS0112, remove static keyword { } public static int Main() { return 0; } } }  
```
