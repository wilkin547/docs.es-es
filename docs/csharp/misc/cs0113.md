# <a name="compiler-error-cs0113"></a>Error del compilador CS0113
Un miembro 'function' marcado como override no se puede marcar como new o virtual.  
  
 Es mutuamente excluyente marcar un m√©todo con las palabras clave [new](../../csharp/language-reference/keywords/new.md) y [override](../../csharp/language-reference/keywords/override.md) .  
  
 El ejemplo siguiente genera la advertencia CS0113:  
  
```csharp  
// CS0113.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
  
   public class MyClass2 : MyClass  
   {  
      override new public void Foo()   // CS0113, remove new keyword  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
