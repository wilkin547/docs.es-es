## <a name="example"></a><span data-ttu-id="2ae58-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="2ae58-106">Example</span></span>  
 <span data-ttu-id="2ae58-107">El ejemplo siguiente genera la advertencia CS0540.</span><span class="sxs-lookup"><span data-stu-id="2ae58-107">The following sample generates CS0540.</span></span>  
  
```csharp  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```
