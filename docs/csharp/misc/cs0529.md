 <span data-ttu-id="24fd4-104">La lista de herencia de una [interfaz](../../csharp/language-reference/keywords/interface.md) incluye una referencia directa o indirecta a sí misma.</span><span class="sxs-lookup"><span data-stu-id="24fd4-104">The inheritance list for an [interface](../../csharp/language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="24fd4-105">La interfaz no puede heredar de sí misma.</span><span class="sxs-lookup"><span data-stu-id="24fd4-105">An interface cannot inherit from itself.</span></span>  
  
 <span data-ttu-id="24fd4-106">El ejemplo siguiente genera la advertencia CS0529:</span><span class="sxs-lookup"><span data-stu-id="24fd4-106">The following sample generates CS0529:</span></span>  
  
```  
// CS0529.cs

{  
   public interface a  
   {  
   }  
  
   public interface b : a, c

   {  
   }  
}  
```
