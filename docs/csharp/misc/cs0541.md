 <span data-ttu-id="9796f-104">Una declaración explícita de la [interfaz](../../csharp/language-reference/keywords/interface.md) solo se puede declarar en una [clase](../../csharp/language-reference/keywords/class.md) o una [estructura](../../csharp/language-reference/keywords/struct.md).</span><span class="sxs-lookup"><span data-stu-id="9796f-104">An explicit [interface](../../csharp/language-reference/keywords/interface.md) declaration was found outside a [class](../../csharp/language-reference/keywords/class.md) or [struct](../../csharp/language-reference/keywords/struct.md).</span></span>  
  
 <span data-ttu-id="9796f-105">El ejemplo siguiente genera la advertencia CS0541:</span><span class="sxs-lookup"><span data-stu-id="9796f-105">The following sample generates CS0541:</span></span>  
  
```csharp  
// CS0541.cs  
namespace x  
{  
   interface IFace  
   {  
      void F();  
   }

   {  
      void IFace.F();   // CS0541  
   }  
}  
```
