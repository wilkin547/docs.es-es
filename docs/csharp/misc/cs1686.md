## <a name="example"></a><span data-ttu-id="69b20-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="69b20-106">Example</span></span>  

 <span data-ttu-id="69b20-107">El ejemplo siguiente genera la advertencia CS1686.</span><span class="sxs-lookup"><span data-stu-id="69b20-107">The following sample generates CS1686.</span></span>  
  
```csharp  
// CS1686.cs  
// compile with: /unsafe /target:library  
class MyClass  
{  
   public unsafe delegate int * MyDelegate();  
  
   public unsafe int * Test()  
   {  
      int j = 0;  
      MyDelegate d = delegate { return &j; };   // CS1686  
      return &j;   // OK  
   }  
}  
```
