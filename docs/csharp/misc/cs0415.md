<span data-ttu-id="fe2d3-104">El atributo 'IndexerName' solo es válido en un indexador que no sea una declaración de miembro de interfaz explícita</span><span class="sxs-lookup"><span data-stu-id="fe2d3-104">The 'IndexerName' attribute is valid only on an indexer that is not an explicit interface member declaration</span></span>  
  
 <span data-ttu-id="fe2d3-105">Este error se produce si se utiliza un atributo IndexerName en un indexador que es una implementación explícita de una interfaz.</span><span class="sxs-lookup"><span data-stu-id="fe2d3-105">This error occurs if you use an IndexerName attribute on an indexer that was an explicit implementation of an interface.</span></span> <span data-ttu-id="fe2d3-106">Este error se puede evitar quitando el nombre de interfaz de la declaración del indexador, si es posible.</span><span class="sxs-lookup"><span data-stu-id="fe2d3-106">This error may be avoided by removing the interface name from the declaration of the indexer, if possible.</span></span> <span data-ttu-id="fe2d3-107">Para más información, consulte [Clase IndexerNameAttribute](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span><span class="sxs-lookup"><span data-stu-id="fe2d3-107">For more information, see the [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span></span>  
  
 <span data-ttu-id="fe2d3-108">El ejemplo siguiente genera la advertencia CS0415:</span><span class="sxs-lookup"><span data-stu-id="fe2d3-108">The following sample generates CS0415:</span></span>  
  
```csharp
// CS0415.cs  
using System;  
using System.Runtime.CompilerServices;  
  
public interface IA  
{  
    int this[int index]  
    {  
        get;  
        set;  
    }  
}  
  
public class A : IA  
{  
    [IndexerName("Item")]  // CS0415  
    int IA.this[int index]  
    // Try this line instead:  
    // public int this[int index]  
    {  
        get { return 0; }  
        set { }  
    }  
  
    static void Main()  
    {  
    }  
}  
```
