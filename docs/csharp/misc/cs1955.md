## <a name="example"></a><span data-ttu-id="c4437-109">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="c4437-109">Example</span></span>

<span data-ttu-id="c4437-110">El código siguiente genera CS1955 porque el código está intentando invocar un campo y una propiedad mediante la [ `()` expresión de invocación ](../language-reference/operators/member-access-operators.md#invocation-expression-).</span><span class="sxs-lookup"><span data-stu-id="c4437-110">The following code generates CS1955 because the code is trying to invoke a field and a property by using the [invocation expression `()`](../language-reference/operators/member-access-operators.md#invocation-expression-).</span></span> <span data-ttu-id="c4437-111">No se puede llamar a un campo ni a una propiedad.</span><span class="sxs-lookup"><span data-stu-id="c4437-111">You cannot call a field or a property.</span></span> <span data-ttu-id="c4437-112">Use la [expresión `.` de acceso a miembros](../language-reference/operators/member-access-operators.md#member-access-expression-) para obtener acceso al valor que almacena.</span><span class="sxs-lookup"><span data-stu-id="c4437-112">Use the [member access expression `.`](../language-reference/operators/member-access-operators.md#member-access-expression-) to access the value it stores.</span></span>
  
```csharp  
// cs1955.cs  
class A  
{  
    public int x = 0;  
    public int X  
    {  
        get { return x; }  
        set { x = value; }  
    }  
}  
  
class Test  
{  
    static int Main()  
    {  
        A a = new A();  
        a.x(); // CS1955  
        a.X(); // CS1955  
        // Try this line instead:  
        // int num = a.x;  
    }  
}  
```
