# <a name="compiler-error-cs1520"></a><span data-ttu-id="1c11f-102">Error del compilador CS1520</span><span class="sxs-lookup"><span data-stu-id="1c11f-102">Compiler Error CS1520</span></span>
<span data-ttu-id="1c11f-103">El método debe tener un tipo de valor devuelto</span><span class="sxs-lookup"><span data-stu-id="1c11f-103">Method must have a return type</span></span>  
  
 <span data-ttu-id="1c11f-104">Un método declarado en una clase, estructura o interfaz debe tener un tipo de valor devuelto explícito.</span><span class="sxs-lookup"><span data-stu-id="1c11f-104">A method that is declared in a class, struct, or interface must have an explicit return type.</span></span> <span data-ttu-id="1c11f-105">En el ejemplo siguiente, el método Square tiene un valor devuelto de [cadena](../../csharp/language-reference/keywords/string.md):</span><span class="sxs-lookup"><span data-stu-id="1c11f-105">In the following example, the Square method has a return value of [string](../../csharp/language-reference/keywords/string.md):</span></span>  
  
```csharp  
class Test  
{  
    string IntToString(int i)  
    {  
        return i.ToString();  
    }  
}  
```  
  
 <span data-ttu-id="1c11f-106">El ejemplo siguiente genera la advertencia CS1520:</span><span class="sxs-lookup"><span data-stu-id="1c11f-106">The following sample generates CS1520:</span></span>  
  
```csharp  
// CS1520a.cs  
public class x  
{  
   // Method declaration missing a return type  
   MyMethod()   // CS1520     
   {}  
   // Add the desired return type:  
   // void MyMethod2()  
  // { }  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="1c11f-107">Este error también puede producirse si el caso de nombre de un constructor difiere del de la declaración de clase o estructura, como en el ejemplo siguiente.</span><span class="sxs-lookup"><span data-stu-id="1c11f-107">Alternatively, this error might be encountered when the case of a constructor's name differs from that of the class or struct declaration, as in the following sample.</span></span> <span data-ttu-id="1c11f-108">Dado que el nombre no es exactamente el mismo que el nombre de clase, el compilador lo interpreta como un método normal, no como un constructor, y genera el error:</span><span class="sxs-lookup"><span data-stu-id="1c11f-108">Because the name is not exactly the same as the class name, the compiler interprets it as a regular method, not a constructor, and produces the error:</span></span>  
  
```csharp  
// CS1520b.cs  
public class Class1  
{  
   // Should be Class1, not class1  
   public class1()   // CS1520  
   {  
   }  
   static void Main()  
   {  
   }  
}  
```
