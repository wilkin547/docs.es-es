# <a name="compiler-error-cs0215"></a>Error del compilador CS0215
El tipo de valor devuelto del operador True o False debe ser bool  
  
 Definido por el usuario [true y false](../language-reference/operators/true-false-operators.md) operadores deben tener un tipo de valor devuelto de [bool](../language-reference/keywords/bool.md). Para obtener más información, vea el artículo sobre la palabra clave [operator](../programming-guide/statements-expressions-operators/overloadable-operators.md) y [Operadores sobrecargables](../language-reference/keywords/operator.md).  
  
 El ejemplo siguiente genera la advertencia CS0215:  
  
```csharp  
// CS0215.cs  
class MyClass  
{  
   public static int operator true (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator true (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static int operator false (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
