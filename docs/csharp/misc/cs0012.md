 <span data-ttu-id="23f04-109">La siguiente secuencia de compilaciones dará como resultado CS0012:</span><span class="sxs-lookup"><span data-stu-id="23f04-109">The following sequence of compilations will result in CS0012:</span></span>  
  
```csharp

public class A {}  
```  
  
 <span data-ttu-id="23f04-110">A continuación:</span><span class="sxs-lookup"><span data-stu-id="23f04-110">Then:</span></span>  
  
```csharp

// compile with: /target:library /reference:cs0012a.dll  
public class B  
{  
   public static A f()  
   {  
      return new A();  
   }  
}  
```  
  
 <span data-ttu-id="23f04-111">A continuación:</span><span class="sxs-lookup"><span data-stu-id="23f04-111">Then:</span></span>  
  
```csharp

{  
   public static void Main()  
   {  
      object o = B.f();   // CS0012  
   }  
}  
```  
  
 <span data-ttu-id="23f04-112">Se puede resolver el error CS0012 compilando con `/reference:cs0012b.dll;cs0012a.dll`, o en Visual Studio mediante el [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) para agregar una referencia a cs0012a.dll además de cs0012b.dll.</span><span class="sxs-lookup"><span data-stu-id="23f04-112">You could resolve this CS0012 by compiling with `/reference:cs0012b.dll;cs0012a.dll`, or in Visual Studio by using the [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) to add a reference to cs0012a.dll in addition to cs0012b.dll.</span></span>
