# <a name="compiler-error-cs0216"></a><span data-ttu-id="48019-103">Error del compilador CS0216</span><span class="sxs-lookup"><span data-stu-id="48019-103">Compiler Error CS0216</span></span>

<span data-ttu-id="48019-104">El operador 'operator' requiere que también se defina un operador coincidente 'missing_operator'.</span><span class="sxs-lookup"><span data-stu-id="48019-104">The operator 'operator' requires a matching operator 'missing_operator' to also be defined</span></span>  
  
 <span data-ttu-id="48019-105">Un operador definido por el usuario [==](../language-reference/operators/equality-operators.md#equality-operator-) requiere un operador [! =](../language-reference/operators/equality-operators.md#inequality-operator-) definido por el usuario y viceversa.</span><span class="sxs-lookup"><span data-stu-id="48019-105">A user-defined [==](../language-reference/operators/equality-operators.md#equality-operator-) operator requires a user-defined [!=](../language-reference/operators/equality-operators.md#inequality-operator-) operator, and vice versa.</span></span>  
 <span data-ttu-id="48019-106">Lo mismo se aplica también a un operador [true](../language-reference/operators/true-false-operators.md) definido por el usuario y un operador [false](../language-reference/operators/true-false-operators.md) definido por el usuario.</span><span class="sxs-lookup"><span data-stu-id="48019-106">The same applies also to a user-defined [true](../language-reference/operators/true-false-operators.md) operator and a user-defined [false](../language-reference/operators/true-false-operators.md) operator.</span></span>  
  
 <span data-ttu-id="48019-107">El ejemplo siguiente genera la advertencia CS0216:</span><span class="sxs-lookup"><span data-stu-id="48019-107">The following sample generates CS0216:</span></span>  
  
```csharp  
// CS0216.cs  
class MyClass  
{  
   public static bool operator == (MyClass MyIntLeft, MyClass MyIntRight)   // CS0216  
   {  
      return MyIntLeft == MyIntRight;  
   }  
  
   // to resolve, uncomment the following operator definition  
   /*  
   public static bool operator != (MyClass MyIntLeft, MyClass MyIntRight)  
   {  
      return MyIntLeft != MyIntRight;  
   }  
   */  
  
   public override bool Equals (object obj)  
   {  
      return base.Equals (obj);  
   }  
  
   public override int GetHashCode()  
   {  
      return base.GetHashCode();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
