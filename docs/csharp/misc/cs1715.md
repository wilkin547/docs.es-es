# <a name="compiler-error-cs1715"></a><span data-ttu-id="33094-103">Error del compilador CS1715</span><span class="sxs-lookup"><span data-stu-id="33094-103">Compiler Error CS1715</span></span>

<span data-ttu-id="33094-104">'Type1': el tipo debe ser 'Type2' para que coincida con el miembro reemplazado 'MemberName'.</span><span class="sxs-lookup"><span data-stu-id="33094-104">'Type1': type must be 'Type2' to match overridden member 'MemberName'</span></span>  
  
 <span data-ttu-id="33094-105">Este error es el mismo que [Compiler Error CS0508](./cs0508.md), salvo que CS0508 ahora solo se aplica a los métodos que tienen tipos de valor devuelto, mientras que CS1715 se aplica a propiedades e indizadores que solo tienen 'tipos' en lugar de 'tipos de devolución'.</span><span class="sxs-lookup"><span data-stu-id="33094-105">This error is the same as [Compiler Error CS0508](./cs0508.md), except that CS0508 now only applies to methods that have return types, while CS1715 applies to properties and indexers that only have 'types' instead of 'return types'.</span></span>  
  
## <a name="example"></a><span data-ttu-id="33094-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="33094-106">Example</span></span>  

 <span data-ttu-id="33094-107">El código siguiente genera el error CS1715.</span><span class="sxs-lookup"><span data-stu-id="33094-107">The following code generates CS1715.</span></span>  
  
```csharp  
// CS1715.cs  
abstract public class Base  
{  
    abstract public int myProperty  
    {  
        get;  
        set;  
    }  
}  
  
public class Derived : Base  
{  
    int myField;  
    public override double myProperty  // CS1715  
    // try the following line instead  
    // public override int myProperty  
    {  
        get { return myField; }  
        set { myField;= value; }  
    }  
  
    public static void Main()  
    {  
        Derived d = new Derived();  
        d.myProperty = 5;  
    }  
}  
```
