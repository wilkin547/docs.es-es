# <a name="compiler-error-cs1909"></a>Error del compilador CS1909

 CS1909 se genera cuando se usa un atributo `DefaultValue` que no es aplicable en el tipo de par치metro.  
  
## <a name="example"></a>Ejemplo  

 El ejemplo siguiente genera la advertencia CS1909.  
  
```csharp  
// CS1909.cs  
// compile with: /target:library
```
