## <a name="example"></a><span data-ttu-id="18e32-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="18e32-108">Example</span></span>  

 <span data-ttu-id="18e32-109">El código siguiente produce el error CS1952:</span><span class="sxs-lookup"><span data-stu-id="18e32-109">The following code produces CS1952:</span></span>

```csharp  
using System.Linq.Expressions;  
  
class Test  
{  
    public static int M(__arglist)  
    {  
        return 1;  
    }  
  
    static int Main()  
    {  
        Expression<Func<int, int>> f = x => Test.M(__arglist(x)); // CS1952  
        return 1;  
    }  
}  
```
