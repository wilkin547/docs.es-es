# Error del compilador CS0726
'format specifier' no es un especificador de formato v√°lido  
  
 Este error se produce en el depurador. Cuando escriba un nombre de variable en una de las ventanas del depurador, puede agregarle una coma y luego un especificador de formato. Algunos ejemplos de ello son `myInt, h` y `myString,nq`. Este error se produce cuando el compilador no reconoce el elemento [Especificadores de formato en C\#](/visual-studio/debugger/format-specifiers-in-csharp).
