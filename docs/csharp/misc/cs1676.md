# <a name="compiler-error-cs1676"></a><span data-ttu-id="d370f-103">Error del compilador CS1676</span><span class="sxs-lookup"><span data-stu-id="d370f-103">Compiler Error CS1676</span></span>

<span data-ttu-id="d370f-104">Se debe declarar el parámetro 'número' con la palabra clave 'palabra clave'</span><span class="sxs-lookup"><span data-stu-id="d370f-104">Parameter 'number' must be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="d370f-105">Este error se produce cuando el modificador de tipo de parámetro de un método anónimo no coincide con el usado en la declaración del delegado al que se va a convertir el método.</span><span class="sxs-lookup"><span data-stu-id="d370f-105">This error occurs when the parameter type modifier in an anonymous method is different from that used in the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="d370f-106">El ejemplo siguiente genera la advertencia CS1676:</span><span class="sxs-lookup"><span data-stu-id="d370f-106">The following sample generates CS1676:</span></span>  
  
```csharp  
// CS1676.cs  
delegate void E(ref int i);  
class Errors
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```
