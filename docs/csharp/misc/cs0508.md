<span data-ttu-id="1d730-103">'Tipo 1': el tipo de valor devuelto debe ser 'Tipo 2' para que coincida con el miembro reemplazado 'Nombre de miembro'</span><span class="sxs-lookup"><span data-stu-id="1d730-103">'Type 1': return type must be 'Type 2' to match overridden member 'Member Name'</span></span>  
  
 <span data-ttu-id="1d730-104">Se ha intentado cambiar el tipo de valor devuelto en un reemplazo de método.</span><span class="sxs-lookup"><span data-stu-id="1d730-104">An attempt was made to change the return type in a method override.</span></span> <span data-ttu-id="1d730-105">Para resolver este error, asegúrese de que ambos métodos declaren el mismo tipo de valor devuelto.</span><span class="sxs-lookup"><span data-stu-id="1d730-105">To resolve this error, make sure both methods declare the same return type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1d730-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="1d730-106">Example</span></span>  
 <span data-ttu-id="1d730-107">El ejemplo siguiente genera la advertencia CS0508.</span><span class="sxs-lookup"><span data-stu-id="1d730-107">The following sample generates CS0508.</span></span>  
  
```csharp  
// CS0508.cs  
// compile with: /target:library  
abstract public class Clx  
{  
   public int i = 0;  
   // Return type is int.  
   abstract public int F();  
}  
  
public class Cly : Clx  
{  
   public override double F()  
   {  
      return 0.0;   // CS0508  
   }  
}  
```
