# Error del compilador CS0262
Las declaraciones parciales de 'tipo' tienen modificadores de accesibilidad conflictivos.  
  
 Este error se produce si un tipo parcial tiene modificadores incoherentes como public, private, protected, internal o abstract. Estos modificadores deben ser coherentes en todas las declaraciones parciales para ese tipo. Para obtener más información, consulta [Clases y métodos parciales](../../csharp/programming-guide/classes-and-structs/partial-classes-and-methods.md).  
  
## Ejemplo  
 El ejemplo siguiente genera la advertencia CS0262:  
  
```  
// CS0262.cs class A { public partial class C  // CS0262 { } private partial class C { } }  
```
