# <a name="compiler-error-cs1677"></a><span data-ttu-id="24f66-103">Error del compilador CS1677</span><span class="sxs-lookup"><span data-stu-id="24f66-103">Compiler Error CS1677</span></span>

<span data-ttu-id="24f66-104">No se debe declarar el parámetro 'número' con la palabra clave 'palabra clave'.</span><span class="sxs-lookup"><span data-stu-id="24f66-104">Parameter 'number' should not be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="24f66-105">Este error se produce cuando el modificador de tipo de parámetro de un método anónimo no coincide con el usado en la declaración del delegado, al que va a convertir el método.</span><span class="sxs-lookup"><span data-stu-id="24f66-105">This error occurs when the parameter type modifier in an anonymous method does not match that used in the declaration of the delegate, to which you are casting the method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="24f66-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="24f66-106">Example</span></span>  

 <span data-ttu-id="24f66-107">El ejemplo siguiente genera la advertencia CS1677:</span><span class="sxs-lookup"><span data-stu-id="24f66-107">The following sample generates CS1677:</span></span>  
  
```csharp  
// CS1677.cs  
delegate void D(int i);  
class Errors  
{  
    static void Main()
    {  
        D d = delegate(out int i) { };   // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int i) { };  
  
        D d = delegate(ref int j){}; // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int j){};  
    }  
}  
```
