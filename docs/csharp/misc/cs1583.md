# <a name="compiler-error-cs1583"></a>Error del compilador CS1583

'archivo' no es un archivo de recursos Win32 válido.  
  
 Este error se produce cuando se especifica un nombre de archivo con la opción del compilador [**Win32Resource**](../language-reference/compiler-options/resources.md#win32resource) que no es un archivo de recursos Win32 válido o con el formato correcto. En Visual Studio, el nombre de archivo se especifica en el panel Aplicación del Diseñador de proyectos.
