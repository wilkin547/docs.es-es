## <a name="example"></a><span data-ttu-id="90143-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="90143-108">Example</span></span>

```csharp  
// CS0403.cs  
// compile with: /target:library

{  
   public void f()  
   {  
      T t = null;  // CS0403  
      T t2 = default(T);   // OK  
    }  
}  
  
class D<T> where T : class
{  
   public void f()  
   {  
      T t = null;  // OK  
    }  
}  
```
