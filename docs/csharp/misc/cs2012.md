# <a name="compiler-error-cs2012"></a>Error del compilador CS2012

No se puede abrir 'label' para escritura  
  
Al usar la `-bugreport:` `file` opción del compilador, no se pudo abrir el archivo para escribir en él. Asegúrese de que ha especificado un nombre de archivo válido y de que el archivo no es de solo lectura.

> [!IMPORTANT]
> Ya `bugreport` no se admite la opción.
