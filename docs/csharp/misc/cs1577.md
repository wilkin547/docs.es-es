# <a name="compiler-error-cs1577"></a>Error del compilador CS1577
Error al generar el ensamblado — motivo  
  
 Error en la parte de la generación del ensamblado de la compilación después de que el compilador de C# compilara correctamente el código fuente. Consulte la documentación de errores de la herramienta ALink ([Al.exe Tool Errors and Warnings](http://msdn.microsoft.com/library/7f125d49-0a03-47a6-9ba9-d61a679a7d4b)) para obtener más información.
