## <a name="example"></a><span data-ttu-id="2ee0f-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="2ee0f-106">Example</span></span>  
 <span data-ttu-id="2ee0f-107">El ejemplo siguiente genera la advertencia C3015.</span><span class="sxs-lookup"><span data-stu-id="2ee0f-107">The following sample generates C3015.</span></span>

```csharp  
[assembly:CLSCompliant(true)]

   public MyAttribute(int[] ai) {}   // CS3015

   // public MyAttribute(int ai) {}  
}  
```
