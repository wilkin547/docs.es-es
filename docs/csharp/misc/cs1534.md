# <a name="compiler-error-cs1534"></a>Error del compilador CS1534
El operador binario sobrecargado 'operator' toma dos parámetros.  
  
 La definición de un archivo binario [operador](../../csharp/language-reference/operators/operator-overloading.md) debe tomar dos parámetros.  
  
 El ejemplo siguiente genera la advertencia CS1534:  
  
```csharp  
// CS1534.cs  
class MyClass  
{  
   public static MyClass operator - (MyClass MC1, MyClass MC2, MyClass MC3)   // CS1534  
   // try the following line instead  
   // public static MyClass operator - (MyClass MC1, MyClass MC2)  
   {  
      return new MyClass();  
   }  
  
   public static int Main()  
   {  
      return 1;  
   }  
}  
```
