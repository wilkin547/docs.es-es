# <a name="compiler-error-cs1593"></a><span data-ttu-id="473e2-102">Error del compilador CS1593</span><span class="sxs-lookup"><span data-stu-id="473e2-102">Compiler Error CS1593</span></span>
<span data-ttu-id="473e2-103">El delegado 'del' no toma argumentos 'number'</span><span class="sxs-lookup"><span data-stu-id="473e2-103">Delegate 'del' does not take 'number' arguments</span></span>  
  
 <span data-ttu-id="473e2-104">El número de argumentos pasados a una invocación [delegate](../../csharp/language-reference/keywords/delegate.md) no coincide con el número de parámetros de la declaración de delegado.</span><span class="sxs-lookup"><span data-stu-id="473e2-104">The number of arguments passed to a [delegate](../../csharp/language-reference/keywords/delegate.md) invocation does not agree with the number of parameters in the delegate declaration.</span></span>  
  
 <span data-ttu-id="473e2-105">El ejemplo siguiente genera la advertencia CS1593:</span><span class="sxs-lookup"><span data-stu-id="473e2-105">The following sample generates CS1593:</span></span>  
  
```csharp  
// CS1593.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello(8, 9);   // CS1593  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
