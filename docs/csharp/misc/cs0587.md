# <a name="compiler-error-cs0587"></a>Error del compilador CS0587
Error interno del compilador: fase 'stage'  
  
 Intente determinar si el compilador está fallando porque no puede analizar una sintaxis inesperada. Si no es así, intente [hable con nosotros](https://docs.microsoft.com/visualstudio/ide/talk-to-us).
