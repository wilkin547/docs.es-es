# <a name="compiler-error-cs0663"></a><span data-ttu-id="5e407-103">Error del compilador CS0663</span><span class="sxs-lookup"><span data-stu-id="5e407-103">Compiler Error CS0663</span></span>
<span data-ttu-id="5e407-104">No se pueden definir métodos sobrecargados que difieran solo en ref y out.</span><span class="sxs-lookup"><span data-stu-id="5e407-104">Cannot define overloaded methods that differ only on ref and out.</span></span>  
  
 <span data-ttu-id="5e407-105">No se permiten los métodos que difieren solo en el uso de [in](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) y [out](../language-reference/keywords/out-parameter-modifier.md) en un parámetro.</span><span class="sxs-lookup"><span data-stu-id="5e407-105">Methods that differ only on their use of [in](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) and [out](../language-reference/keywords/out-parameter-modifier.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="5e407-106">El ejemplo siguiente genera la advertencia CS0663:</span><span class="sxs-lookup"><span data-stu-id="5e407-106">The following sample generates CS0663:</span></span>  
  
```csharp  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
