 <span data-ttu-id="27bd1-104">Este error se produce si intenta pasar un miembro de una variable [foreach](../../csharp/language-reference/keywords/foreach-in.md) , una variable [using](../../csharp/language-reference/keywords/using-statement.md) o una variable [fixed](../../csharp/language-reference/keywords/fixed-statement.md) a una función como un parámetro ref o out.</span><span class="sxs-lookup"><span data-stu-id="27bd1-104">This error occurs if you are attempting to pass a member of a [foreach](../../csharp/language-reference/keywords/foreach-in.md) variable, a [using](../../csharp/language-reference/keywords/using-statement.md) variable, or a [fixed](../../csharp/language-reference/keywords/fixed-statement.md) variable to a function as a ref or out argument.</span></span> <span data-ttu-id="27bd1-105">Como estas variables se consideran de solo lectura en estos contextos, esto no está permitido.</span><span class="sxs-lookup"><span data-stu-id="27bd1-105">Because these variables are considered read-only in these contexts, this is not allowed.</span></span>

```csharp  
// CS1655.cs  
struct S   
{  
   public int i;  
}  
  
class CMain  
{  
  static void f(ref int iref)  
  {  
  }  
  
  public static void Main()  
  {  
     S[] sa = new S[10];

     {  
        CMain.f(ref s.i);  // CS1655  
     }  
  }  
}  
```
