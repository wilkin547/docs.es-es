# <a name="compiler-error-cs2011"></a>Error del compilador CS2011

No se puede abrir el archivo de respuesta 'file'  
  
 Se especificó un archivo de respuesta en una compilación, pero el compilador no pudo localizar y abrir el archivo.  
  
 Para obtener más información sobre los archivos de respuesta, vea [ **ResponseFiles** (especificar archivo de respuesta)](../language-reference/compiler-options/miscellaneous.md#responsefiles).
