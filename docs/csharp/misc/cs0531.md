# <a name="compiler-error-cs0531"></a><span data-ttu-id="4432a-102">Error del compilador CS0531</span><span class="sxs-lookup"><span data-stu-id="4432a-102">Compiler Error CS0531</span></span>
<span data-ttu-id="4432a-103">'miembro': los miembros de la interfaz no pueden tener una definición</span><span class="sxs-lookup"><span data-stu-id="4432a-103">'member' : interface members cannot have a definition</span></span>  
  
 <span data-ttu-id="4432a-104">Los métodos que se declaran en una [interfaz](../../csharp/language-reference/keywords/interface.md) debe implementarse en una clase que hereda de ella y no en la propia interfaz.</span><span class="sxs-lookup"><span data-stu-id="4432a-104">Methods that are declared in an [interface](../../csharp/language-reference/keywords/interface.md) must be implemented in a class that inherits from it and not in the interface itself.</span></span>  
  
 <span data-ttu-id="4432a-105">El ejemplo siguiente genera la advertencia CS0531:</span><span class="sxs-lookup"><span data-stu-id="4432a-105">The following sample generates CS0531:</span></span>  
  
```csharp  
// CS0531.cs  
namespace x  
{  
   public interface clx  
   {  
      int xclx()   // CS0531, cannot define xclx  
      // Try the following declaration instead:  
      // int xclx();  
      {  
         return 0;  
      }  
   }  
  
   public class cly  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
