# <a name="compiler-error-cs0531"></a>Error del compilador CS0531
'miembro': los miembros de la interfaz no pueden tener una definición  
  
 Los métodos que se declaran en una [interfaz](../../csharp/language-reference/keywords/interface.md) debe implementarse en una clase que hereda de ella y no en la propia interfaz.  
  
 El ejemplo siguiente genera la advertencia CS0531:  
  
```csharp  
// CS0531.cs  
namespace x  
{  
   public interface clx  
   {  
      int xclx()   // CS0531, cannot define xclx  
      // Try the following declaration instead:  
      // int xclx();  
      {  
         return 0;  
      }  
   }  
  
   public class cly  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
