# <a name="compiler-error-cs0065"></a><span data-ttu-id="95f5f-102">Error del compilador CS0065</span><span class="sxs-lookup"><span data-stu-id="95f5f-102">Compiler Error CS0065</span></span>
<span data-ttu-id="95f5f-103">'event': la propiedad del evento debe tener los descriptores de acceso add y remove</span><span class="sxs-lookup"><span data-stu-id="95f5f-103">'event' : event property must have both add and remove accessors</span></span>  
  
 <span data-ttu-id="95f5f-104">Un evento que no es un campo debe tener ambos métodos de acceso.</span><span class="sxs-lookup"><span data-stu-id="95f5f-104">An event that is not a field must have both access methods.</span></span>  
  
 <span data-ttu-id="95f5f-105">El ejemplo siguiente genera la advertencia CS0065:</span><span class="sxs-lookup"><span data-stu-id="95f5f-105">The following sample generates CS0065:</span></span>  
  
```csharp  
// CS0065.cs  
using System;  
public delegate void Eventhandler(object sender, int e);  
public class MyClass  
{  
   public event EventHandler Click   // CS0065,  
   {  
      // to fix, uncomment the add and remove definitions  
      /*  
      add  
      {  
         Click += value;  
      }  
      remove  
      {  
         Click -= value;  
      }  
      */  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
