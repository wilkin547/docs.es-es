# <a name="compiler-error-cs0065"></a><span data-ttu-id="2d044-103">Error del compilador CS0065</span><span class="sxs-lookup"><span data-stu-id="2d044-103">Compiler Error CS0065</span></span>

<span data-ttu-id="2d044-104">'event': la propiedad del evento debe tener los descriptores de acceso add y remove</span><span class="sxs-lookup"><span data-stu-id="2d044-104">'event' : event property must have both add and remove accessors</span></span>

<span data-ttu-id="2d044-105">Un evento que no es un campo debe tener ambos métodos de acceso.</span><span class="sxs-lookup"><span data-stu-id="2d044-105">An event that is not a field must have both access methods.</span></span>

<span data-ttu-id="2d044-106">El ejemplo siguiente genera la advertencia CS0065:</span><span class="sxs-lookup"><span data-stu-id="2d044-106">The following sample generates CS0065:</span></span>

```csharp
// CS0065.cs
using System;
public delegate void EventHandler(object sender, int e);
public class MyClass
{
   public event EventHandler Click   // CS0065,
   {
      // to fix, uncomment the add and remove definitions
      /*
      add
      {
         Click += value;
      }
      remove
      {
         Click -= value;
      }
      */
   }

   public static void Main()
   {
   }
}
```
