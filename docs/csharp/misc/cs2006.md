# <a name="compiler-error-cs2006"></a>Error del compilador CS2006
Error de sintaxis de línea de comandos: Falta 'texto' para ' opción '  
  
 La sintaxis de *option* requiere texto adicional. Para obtener información, consulte [Opciones del compilador](../../csharp/language-reference/compiler-options/index.md).
