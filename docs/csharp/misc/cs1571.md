# <a name="compiler-warning-level-2-cs1571"></a>Advertencia del compilador (nivel 2) CS1571

El comentario XML en 'construcción' tiene una etiqueta de parámetro duplicada para 'parámetro'.  
  
 Al usar la opción del compilador [**DocumentationFile**](../language-reference/compiler-options/output.md#documentationfile) , se encontraron varios comentarios para el mismo parámetro de método. Quite una de las líneas duplicadas.  
  
 El ejemplo siguiente genera la advertencia CS1571:  
  
```csharp  
// CS1571.cs  
// compile with: /W:2 /doc:x.xml  
  
/// <summary>help text</summary>  
public class MyClass  
{  
   /// <param name='Int1'>Used to indicate status.</param>  
   /// <param name='Char1'>An initial.</param>  
   /// <param name='Int1'>Used to indicate status.</param> // CS1571  
   public static void MyMethod(int Int1, char Char1)  
   {  
   }  
  
   /// <summary>help text</summary>  
   public static void Main ()  
   {  
   }  
}  
```
