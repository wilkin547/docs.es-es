# <a name="compiler-error-cs1630"></a>Error del compilador CS1630
Opción no válida 'option' para /errorreport; debe ser prompt, send, queue o none  
  
 La opción de línea de comandos [/errorreport](../language-reference/compiler-options/errorreport-compiler-option.md) debe ir seguida de **prompt**, **send**, **queue**o **none**para especificar la acción que desea realizar cuando se produzca un error interno del compilador.
