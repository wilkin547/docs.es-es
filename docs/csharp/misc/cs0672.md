# <a name="compiler-warning-level-1-cs0672"></a><span data-ttu-id="98eaa-102">Advertencia del compilador (nivel 1) CS0672</span><span class="sxs-lookup"><span data-stu-id="98eaa-102">Compiler Warning (level 1) CS0672</span></span>
<span data-ttu-id="98eaa-103">El miembro 'member1' reemplaza al miembro obsoleto 'member2'.</span><span class="sxs-lookup"><span data-stu-id="98eaa-103">Member 'member1' overrides obsolete member 'member2.</span></span> <span data-ttu-id="98eaa-104">Agregue el atributo Obsolete a 'member1'.</span><span class="sxs-lookup"><span data-stu-id="98eaa-104">Add the Obsolete attribute to 'member1'</span></span>  
  
 <span data-ttu-id="98eaa-105">El compilador encontró un `override` a un método marcado como `obsolete`.</span><span class="sxs-lookup"><span data-stu-id="98eaa-105">The compiler found an `override` to a method marked as `obsolete`.</span></span> <span data-ttu-id="98eaa-106">Sin embargo, el método de reemplazo no estaba marcado como obsoleto.</span><span class="sxs-lookup"><span data-stu-id="98eaa-106">However, the overriding method was not itself marked as obsolete.</span></span> <span data-ttu-id="98eaa-107">El método de reemplazo seguirá generando [CS0612](../../csharp/misc/cs0612.md), si se le llama.</span><span class="sxs-lookup"><span data-stu-id="98eaa-107">The overriding method will still generate [CS0612](../../csharp/misc/cs0612.md), if called.</span></span>  
  
 <span data-ttu-id="98eaa-108">Revise sus declaraciones de método e indique explícitamente si debe marcarse un método (y todos sus reemplazos) `obsolete`.</span><span class="sxs-lookup"><span data-stu-id="98eaa-108">Review your method declarations and explicitly indicate whether a method (and all of its overrides) should be marked `obsolete`.</span></span>  
  
 <span data-ttu-id="98eaa-109">El ejemplo siguiente genera la advertencia CS0672:</span><span class="sxs-lookup"><span data-stu-id="98eaa-109">The following sample generates CS0672:</span></span>  
  
```  
// CS0672.cs  
// compile with: /W:1  
class MyClass  
{  
   [System.Obsolete]  
   public virtual void ObsoleteMethod()  
   {  
   }  
}  
  
class MyClass2 : MyClass  
{  
   public override void ObsoleteMethod()   // CS0672  
   {  
   }  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
   }  
}  
```
