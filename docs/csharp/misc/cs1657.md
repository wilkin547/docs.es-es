## <a name="example"></a><span data-ttu-id="f6a05-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="f6a05-108">Example</span></span>

```csharp  
    public int i;  
    public void Dispose() {}

    static void f(ref C c)  
    {  
    }  
    static void Main()

    }  
}  
```
