# <a name="compiler-error-cs1556"></a>Error del compilador CS1556

La construcción 'construcción' que se especificó para el método Main debe ser una clase o una estructura válida  
  
 Se pasó a la opción del compilador [**MainEntryPoint**](../language-reference/compiler-options/advanced.md#mainentrypoint-or-startupobject) un identificador que no era un nombre de clase.
