# <a name="compiler-error-cs1555"></a>Error del compilador CS1555
No se encontró la 'class' especificada para el método Main  
  
 Se especificó una clase para la opción del compilador [-Main](../language-reference/compiler-options/main-compiler-option.md) , pero no se encontró el nombre de clase en el código fuente.
