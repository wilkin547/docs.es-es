# <a name="compiler-error-cs1555"></a>Error del compilador CS1555

No se encontró la 'class' especificada para el método Main  
  
 Se especificó una clase para la opción del compilador [**MainEntryPoint**](../language-reference/compiler-options/advanced.md#mainentrypoint-or-startupobject) , pero no se encontró el nombre de clase en el código fuente.
