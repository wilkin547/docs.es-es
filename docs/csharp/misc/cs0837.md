# <a name="compiler-error-cs0837"></a><span data-ttu-id="07a93-102">Error del compilador CS0837</span><span class="sxs-lookup"><span data-stu-id="07a93-102">Compiler Error CS0837</span></span>
<span data-ttu-id="07a93-103">El primer operando de un operador "is" o "as" no puede ser una expresión lambda ni un método anónimo.</span><span class="sxs-lookup"><span data-stu-id="07a93-103">The first operand of an "is" or "as" operator may not be a lambda expression or anonymous method.</span></span>  
  
 <span data-ttu-id="07a93-104">Los métodos anónimos y las expresiones lambda no pueden usarse en el lado izquierdo de [is](../../csharp/language-reference/keywords/is.md) o [as](../../csharp/language-reference/keywords/as.md).</span><span class="sxs-lookup"><span data-stu-id="07a93-104">Lambda expressions and anonymous methods may not be used on the left side of [is](../../csharp/language-reference/keywords/is.md) or [as](../../csharp/language-reference/keywords/as.md).</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="07a93-105">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="07a93-105">To correct this error</span></span>  
  
-   <span data-ttu-id="07a93-106">Si el error está relacionado con el operador `is` , recuerde que `is` toma un valor y un tipo y le indica si se puede convertir el valor en ese tipo mediante una conversión boxing, unboxing o de referencia.</span><span class="sxs-lookup"><span data-stu-id="07a93-106">If the error involves the `is` operator, remember that `is` takes a value and a type and tells you whether the value can be made into that type by a reference, boxing, or unboxing conversion.</span></span> <span data-ttu-id="07a93-107">Como las expresiones lambda no son valores y no tienen conversiones boxing, unboxing o de referencia, las expresiones lambda no son candidatas para `is`.</span><span class="sxs-lookup"><span data-stu-id="07a93-107">Because lambdas are not values and have no reference, boxing, or unboxing conversions, lambdas are not candidates for `is`.</span></span>  
  
-   <span data-ttu-id="07a93-108">Si el código hace un uso inadecuado de `as`, probablemente la corrección lo cambiará a una conversión.</span><span class="sxs-lookup"><span data-stu-id="07a93-108">If the code misuses `as`, the correction is probably to change it to a cast.</span></span>  
  
## <a name="example"></a><span data-ttu-id="07a93-109">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="07a93-109">Example</span></span>  
 <span data-ttu-id="07a93-110">El ejemplo siguiente genera el error CS0837:</span><span class="sxs-lookup"><span data-stu-id="07a93-110">The following example generates CS0837:</span></span>  
  
```csharp  
// cs0837.cs  
namespace TestNamespace  
{  
    public delegate void Del();  
  
    class Test  
    {  
        static int Main()  
        {  
            bool b1 = (() => { }) is Del;   // CS0837  
            bool b2 = delegate() { } is Del;// CS0837  
            Del d1 = () => { } as Del;      // CS0837  
            Del d2 = delegate() { } as Del; // CS0837  
            return 1;  
        }  
    }  
}  
```
