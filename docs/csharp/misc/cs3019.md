## <a name="example"></a><span data-ttu-id="ff3be-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="ff3be-108">Example</span></span>  

 <span data-ttu-id="ff3be-109">El ejemplo siguiente genera la advertencia CS3019:</span><span class="sxs-lookup"><span data-stu-id="ff3be-109">The following sample generates CS3019:</span></span>

```csharp  
    }

    }  
}  
```
