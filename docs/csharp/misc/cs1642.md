## <a name="example"></a><span data-ttu-id="c00e8-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="c00e8-106">Example</span></span>  
 <span data-ttu-id="c00e8-107">El ejemplo siguiente genera la advertencia CS1642.</span><span class="sxs-lookup"><span data-stu-id="c00e8-107">The following sample generates CS1642.</span></span>

```csharp  
unsafe class C  
{  
   fixed int a[10];   // CS1642

{  
    fixed int a[10];  
}

{  
   public int[] a = null;  
}  
```
