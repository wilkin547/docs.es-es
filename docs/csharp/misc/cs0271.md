 <span data-ttu-id="412e2-104">Este error se produce cuando se intenta acceder a un descriptor de acceso `get` inaccesible.</span><span class="sxs-lookup"><span data-stu-id="412e2-104">This error occurs when you try to access an inaccessible `get` accessor.</span></span> <span data-ttu-id="412e2-105">Para resolver este error, aumente la accesibilidad del descriptor de acceso o cambie la ubicación de la llamada.</span><span class="sxs-lookup"><span data-stu-id="412e2-105">To resolve this error, increase the accessibility of the accessor, or change the calling location.</span></span> <span data-ttu-id="412e2-106">Para obtener más información, consulte [accesibilidad del descriptor de acceso](../../csharp/programming-guide/classes-and-structs/restricting-accessor-accessibility.md) y [propiedades](../../csharp/programming-guide/classes-and-structs/properties.md).</span><span class="sxs-lookup"><span data-stu-id="412e2-106">For more information, see [Accessor Accessibility](../../csharp/programming-guide/classes-and-structs/restricting-accessor-accessibility.md) and [Properties](../../csharp/programming-guide/classes-and-structs/properties.md).</span></span>  
  
 <span data-ttu-id="412e2-107">El ejemplo siguiente genera el error CS0271:</span><span class="sxs-lookup"><span data-stu-id="412e2-107">The following example generates CS0271:</span></span>  
  
```csharp  
// CS0271.cs  
public class MyClass  
{  
   public int Property  
   {  
      private get { return 0; }  
      set { }  
   }  
  
   public int Property2  
   {  
      get { return 0; }  
      set { }  
   }  
}  
  
public class Test  
{  
   public static void Main(string[] args)   
   {  
      MyClass c = new MyClass();  
      int a = c.Property;   // CS0271  
      int b = c.Property2;   // OK  
   }  
}  
```
