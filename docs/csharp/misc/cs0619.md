## <a name="example"></a><span data-ttu-id="09b26-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="09b26-106">Example</span></span>

<span data-ttu-id="09b26-107">El ejemplo siguiente genera la advertencia CS0619:</span><span class="sxs-lookup"><span data-stu-id="09b26-107">The following sample generates CS0619:</span></span>  
  
```csharp
using System;

public class C
{
    [Obsolete("Use NewMethod instead", true)] // generates an error on use
    public static void OldMethod()
    {
    }

    // this is the method you should be using
    public static void NewMethod()
    {
    }  
}

class MyClass
{
   public static void Main()
   {
      C.OldMethod();   // CS0619
   }
}
```
