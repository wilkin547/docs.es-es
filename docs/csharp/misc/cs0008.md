# <a name="compiler-error-cs0008"></a>Error del compilador CS0008

Error inesperado al leer metadatos del archivo 'archivo': 'descripción'.  
  
 Un archivo DLL se abrió correctamente para recuperar metadatos, pero está dañado, por lo que no se pudieron leer los datos. Para obtener más información, vea [ **referencias** (opciones del compilador de C#)](../language-reference/compiler-options/inputs.md#references).
