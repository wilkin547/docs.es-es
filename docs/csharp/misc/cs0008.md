# Error del compilador CS0008
Error inesperado al leer metadatos del archivo 'archivo': 'descripción'.  
  
 Un archivo DLL se abrió correctamente para recuperar metadatos, pero está dañado, por lo que no se pudieron leer los datos. Para obtener más información, consulta [\/reference \(Import Metadata\)](../../csharp/language-reference/compiler-options/reference-compiler-option.md).
