# <a name="compiler-error-cs0244"></a>Error del compilador CS0244
Ni 'is' ni 'as' son válidos como tipos de puntero  
  
 El [es](../../csharp/language-reference/operators/type-testing-and-conversion-operators.md#is-operator) y [como](../../csharp/language-reference/operators/type-testing-and-conversion-operators.md#as-operator) operadores no son válidos para su uso en tipos de puntero. Para obtener más información, vea [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md) (Código no seguro y punteros [Guía de programación de C#]).  
  
 El ejemplo siguiente genera la advertencia CS0244:  
  
```csharp  
// CS0244.cs  
// compile with: /unsafe  
  
class UnsafeTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      bool b = p is object;   // CS0244 p is pointer  
   }  
  
   unsafe public static void Main()  
   {  
      int i = 5;  
      SquarePtrParam (&i);  
   }  
}  
```
