# <a name="compiler-error-cs0009"></a>Error del compilador CS0009

No se pudo abrir el archivo de metadatos 'file' — 'description'  
  
 El archivo especificado con la opción del compilador [**References**](../language-reference/compiler-options/inputs.md#references) no contiene metadatos válidos.
