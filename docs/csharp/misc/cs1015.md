# <a name="compiler-error-cs1015"></a>Error del compilador CS1015
Un objeto, una cadena o un tipo de clase esperados  
  
 Se intentó pasar un tipo de datos predefinido en un bloque [catch](../../csharp/language-reference/keywords/try-catch.md) . Solo los tipos de datos que se derivan de <xref:System.Exception?displayProperty=nameWithType> se pueden pasar en un bloque `catch` . Para más información sobre excepciones, vea [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md).  
  
## <a name="example"></a>Ejemplo  
 El ejemplo siguiente genera la advertencia CS1015:  
  
```csharp  
// CS1015.cs  
class Sample  
{  
    static void Main()  
    {  
        try   
        {  
        }  
        catch(int)   // CS1015, int is not derived from System.Exception  
        {  
        }  
    }  
}  
```
