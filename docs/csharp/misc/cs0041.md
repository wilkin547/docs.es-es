# Error del compilador CS0041
El nombre completo de 'type' es demasiado largo para la información de depuración. Realice la compilación sin la opción '\/debug'.  
  
 Este error puede producirse al usar la opción [\/debug](../../csharp/language-reference/compiler-options/debug-compiler-option.md) del compilador. Si se produce este error, intente eliminar los archivos PDB en el directorio bin y vuelva a realizar la compilación. Si este error persiste, es posible que deba reparar o reinstalar [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].
