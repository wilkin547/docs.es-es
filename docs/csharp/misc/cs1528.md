 <span data-ttu-id="7f9b7-104">Se formó una referencia a una clase como si se estuviera creando un objeto para la clase.</span><span class="sxs-lookup"><span data-stu-id="7f9b7-104">A reference to a class was formed as if an object to the class was being created.</span></span> <span data-ttu-id="7f9b7-105">Por ejemplo, se intentó pasar una variable a un constructor.</span><span class="sxs-lookup"><span data-stu-id="7f9b7-105">For example, there was an attempt to pass a variable to a constructor.</span></span> <span data-ttu-id="7f9b7-106">Utilice el operador [new](../../csharp/language-reference/keywords/new.md) para crear un objeto de una clase.</span><span class="sxs-lookup"><span data-stu-id="7f9b7-106">Use the [new](../../csharp/language-reference/keywords/new.md) operator to create an object of a class.</span></span>  
  
 <span data-ttu-id="7f9b7-107">El ejemplo siguiente genera la advertencia CS1528:</span><span class="sxs-lookup"><span data-stu-id="7f9b7-107">The following sample generates CS1528:</span></span>  
  
```csharp  
// CS1528.cs  
using System;  
  
public class B  
{  
   public B(int i)  
   {  
      _i = i;  
   }  
  
   public void PrintB()  
   {  
      Console.WriteLine(_i);  
   }  
  
   private int _i;  
}  
  
public class mine  
{  
   public static void Main()  
   {  
      B b(3);   // CS1528, reference is not an object  
      // try one of the following  
      // B b;  
      // or  
      // B bb = new B(3);  
      // bb.PrintB();  
   }  
}  
```
