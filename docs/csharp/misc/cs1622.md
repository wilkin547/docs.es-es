<span data-ttu-id="e2c4a-103">No se puede devolver un valor a partir de un iterador.</span><span class="sxs-lookup"><span data-stu-id="e2c4a-103">Cannot return a value from an iterator.</span></span> <span data-ttu-id="e2c4a-104">Utilice la instrucción yield return para devolver un valor o yield break para terminar la iteración.</span><span class="sxs-lookup"><span data-stu-id="e2c4a-104">Use the yield return statement to return a value, or yield break to end the iteration.</span></span>

 <span data-ttu-id="e2c4a-107">El ejemplo siguiente genera la advertencia CS1622:</span><span class="sxs-lookup"><span data-stu-id="e2c4a-107">The following sample generates CS1622:</span></span>  
  
```csharp  
// CS1622.cs  
// compile with: /target:library  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      return (IEnumerator) this;  // CS1622  
      yield return this;   // OK  
   }  
}  
```
