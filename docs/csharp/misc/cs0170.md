# <a name="compiler-error-cs0170"></a>Error del compilador CS0170
Uso del campo 'field' posiblemente sin asignar  
  
 Se usó un campo de una estructura sin inicializarlo primero. Para solucionar este problema, determine primero de qué campo se canceló la inicialización e inicialícelo antes de intentar acceder a este. Para obtener más información acerca de cómo inicializar estructuras, vea [Structs](../../csharp/programming-guide/classes-and-structs/structs.md) y [usar estructuras](../../csharp/programming-guide/classes-and-structs/using-structs.md).  
  
 El ejemplo siguiente genera la advertencia CS0170:  
  
```csharp  
// CS0170.cs  
public struct error  
{  
   public int i;  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      error e;  
      // uncomment the next line to resolve this error  
      // e.i = 0;  
      System.Console.WriteLine( e.i );   // CS0170 because   
                                         //e.i was never assigned  
   }  
}  
```
