# <a name="compiler-error-cs0006"></a>Error del compilador CS0006

No se encontró el archivo de metadatos 'dll_name'
  
 El programa se ha compilado y ha pasado explícitamente el nombre de un archivo que contenía metadatos; en cambio, el archivo .dll no se ha encontrado. Para obtener más información, vea [**References** (opciones del compilador de C#)](../compiler-options/inputs.md#references).
