# <a name="compiler-warning-level-1-cs1762"></a><span data-ttu-id="d55a5-103">Advertencia del compilador (nivel 1) CS1762</span><span class="sxs-lookup"><span data-stu-id="d55a5-103">Compiler Warning (level 1) CS1762</span></span>

<span data-ttu-id="d55a5-104">Se ha creado una referencia al ensamblado de interoperabilidad "\<assembly1>" incrustado debido a una referencia indirecta a dicho ensamblado desde el ensamblado "\<assembly2>".</span><span class="sxs-lookup"><span data-stu-id="d55a5-104">A reference was created to embedded interop assembly '\<assembly1>' because of an indirect reference to that assembly from assembly '\<assembly2>'.</span></span> <span data-ttu-id="d55a5-105">Considere cambiar la propiedad "Incrustar tipos de interoperabilidad" en uno de los ensamblados.</span><span class="sxs-lookup"><span data-stu-id="d55a5-105">Consider changing the 'Embed Interop Types' property on either assembly.</span></span>  
  
 <span data-ttu-id="d55a5-106">Ha agregado una referencia a un ensamblado (ensamblado1) cuya propiedad `Embed Interop Types` está establecida en `True`.</span><span class="sxs-lookup"><span data-stu-id="d55a5-106">You have added a reference to an assembly (assembly1) that has the `Embed Interop Types` property set to `True`.</span></span> <span data-ttu-id="d55a5-107">Esto indica al compilador que incruste la información de tipo de interoperabilidad desde este ensamblado.</span><span class="sxs-lookup"><span data-stu-id="d55a5-107">This instructs the compiler to embed interop type information from that assembly.</span></span> <span data-ttu-id="d55a5-108">Sin embargo, el compilador no puede incrustar información de tipo de interoperabilidad desde este ese ensamblado porque otro ensamblado al que ha hecho referencia (ensamblado2) también hace referencia a este ensamblado (ensamblado1) y tiene la propiedad `Embed Interop Types` establecida en `False`.</span><span class="sxs-lookup"><span data-stu-id="d55a5-108">However, the compiler cannot embed interop type information from that assembly because another assembly that you have referenced (assembly2) also references that assembly (assembly1) and has the `Embed Interop Types` property set to `False`.</span></span>  
  
> [!NOTE]
> <span data-ttu-id="d55a5-109">Para el compilador de línea de comandos, el establecimiento de la propiedad `Embed Interop Types` de una referencia a ensamblado en `True` equivale a hacer referencia al ensamblado utilizando la opción `/link`.</span><span class="sxs-lookup"><span data-stu-id="d55a5-109">Setting the `Embed Interop Types` property on an assembly reference to `True` is equivalent to referencing the assembly by using the `/link` option for the command-line compiler.</span></span>

- <span data-ttu-id="d55a5-111">Para incrustar información de tipo de interoperabilidad para ambos ensamblados, establezca la propiedad `Embed Interop Types` de todas las referencias a ensamblado1 en `True`.</span><span class="sxs-lookup"><span data-stu-id="d55a5-111">To embed interop type information for both assemblies, set the `Embed Interop Types` property on all references to assembly1 to `True`.</span></span> <span data-ttu-id="d55a5-112">Para obtener más información sobre cómo establecer esa propiedad, vea [Tutorial: Insertar los tipos de los ensamblados administrados](../../../standard/assembly/embed-types-visual-studio.md).</span><span class="sxs-lookup"><span data-stu-id="d55a5-112">For more information about how to set that property, see [Walkthrough: Embedding Types from Managed Assemblies](../../../standard/assembly/embed-types-visual-studio.md).</span></span>
