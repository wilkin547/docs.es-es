# <a name="compiler-error-cs1009"></a><span data-ttu-id="b7fb8-102">Error del compilador CS1009</span><span class="sxs-lookup"><span data-stu-id="b7fb8-102">Compiler Error CS1009</span></span>
<span data-ttu-id="b7fb8-103">Secuencia de escape no reconocida</span><span class="sxs-lookup"><span data-stu-id="b7fb8-103">Unrecognized escape sequence</span></span>  
  
 <span data-ttu-id="b7fb8-104">Un carácter inesperado después de una barra diagonal inversa (\\) en una [cadena](../../../csharp/language-reference/keywords/string.md).</span><span class="sxs-lookup"><span data-stu-id="b7fb8-104">An unexpected character follows a backslash (\\) in a [string](../../../csharp/language-reference/keywords/string.md).</span></span> <span data-ttu-id="b7fb8-105">El compilador espera uno de los caracteres de escape válidos.</span><span class="sxs-lookup"><span data-stu-id="b7fb8-105">The compiler expects one of the valid escape characters.</span></span> <span data-ttu-id="b7fb8-106">Para más información, consulte [Escapes de carácter](../../../standard/base-types/character-escapes-in-regular-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="b7fb8-106">For more information, see [Character Escapes](../../../standard/base-types/character-escapes-in-regular-expressions.md).</span></span>  
  
 <span data-ttu-id="b7fb8-107">En el ejemplo siguiente se genera el error CS1009.</span><span class="sxs-lookup"><span data-stu-id="b7fb8-107">The following sample generates CS1009.</span></span>  
  
```csharp  
// CS1009-a.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string a = "\m";     
      // Try the following line instead.  
      // string a = "\t";  
   }  
}  
```  
  
 <span data-ttu-id="b7fb8-108">Una causa común de este error es el uso del carácter de barra diagonal inversa en un nombre de archivo, como se muestra en el ejemplo siguiente.</span><span class="sxs-lookup"><span data-stu-id="b7fb8-108">A common cause of this error is using the backslash character in a file name, as the following example shows.</span></span>  
  
```csharp  
string filename = "c:\myFolder\myFile.txt";  
```  
  
 <span data-ttu-id="b7fb8-109">Para resolver este error, use "\\\\" o el literal de cadena @-quoted como se muestra en el ejemplo siguiente.</span><span class="sxs-lookup"><span data-stu-id="b7fb8-109">To resolve this error, use "\\\\" or the @-quoted string literal, as the following example shows.</span></span>  
  
```csharp  
// CS1009-b.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string filename = "c:\myFolder\myFile.txt";     
      // Try one of the following lines instead.  
      // string filename = "c:\\myFolder\\myFile.txt";  
      // string filename = @"c:\myFolder\myFile.txt";  
   }  
}  
```
