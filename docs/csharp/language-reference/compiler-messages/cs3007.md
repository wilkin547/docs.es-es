# <a name="compiler-warning-level-1-cs3007"></a><span data-ttu-id="94ba7-102">Advertencia del compilador (nivel 1) CS3007</span><span class="sxs-lookup"><span data-stu-id="94ba7-102">Compiler Warning (level 1) CS3007</span></span>
<span data-ttu-id="94ba7-103">El método sobrecargado 'método' que sólo se diferencia por tipos de matriz sin nombre no es conforme a CLS</span><span class="sxs-lookup"><span data-stu-id="94ba7-103">Overloaded method 'method' differing only by unnamed array types is not CLS-compliant</span></span>  
  
 <span data-ttu-id="94ba7-104">Este error aparece si tiene un método sobrecargado que toma una matriz escalonada (jagged array) y la única diferencia entre las firmas de método es el tipo de elemento de la matriz.</span><span class="sxs-lookup"><span data-stu-id="94ba7-104">This error occurs if you have an overloaded method that takes a jagged array and the only difference between the method signatures is the element type of the array.</span></span> <span data-ttu-id="94ba7-105">Para evitar este error, piense en utilizar una matriz rectangular en lugar de una matriz escalonada; utilice un parámetro adicional para eliminar la ambigüedad de la llamada a la función; cambie el nombre de uno o varios métodos sobrecargados; o bien, si no es necesaria la conformidad con CLS, quite el atributo <xref:System.CLSCompliantAttribute>.</span><span class="sxs-lookup"><span data-stu-id="94ba7-105">To avoid this error, consider using a rectangular array rather than a jagged array; use an additional parameter to disambiguate the function call; rename one or more of the overloaded methods; or, if CLS Compliance is not needed, remove the <xref:System.CLSCompliantAttribute> attribute.</span></span> <span data-ttu-id="94ba7-106">Para obtener más información sobre la conformidad con CLS, vea [Independencia del lenguaje y componentes independientes del lenguaje](https://msdn.microsoft.com/library/12a7a7h3).</span><span class="sxs-lookup"><span data-stu-id="94ba7-106">For more information on CLS Compliance, see [Language Independence and Language-Independent Components](https://msdn.microsoft.com/library/12a7a7h3).</span></span>  
  
## <a name="example"></a><span data-ttu-id="94ba7-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="94ba7-107">Example</span></span>  
 <span data-ttu-id="94ba7-108">El ejemplo siguiente genera el error CS3007:</span><span class="sxs-lookup"><span data-stu-id="94ba7-108">The following example generates CS3007:</span></span>  
  
```csharp  
// CS3007.cs  
[assembly: System.CLSCompliant(true)]  
public struct S  
{  
    public void F(int[][] array) { }  
    public void F(byte[][] array) { }  // CS3007  
    // Try this instead:  
    // public void F1(int[][] array) {}  
    // public void F2(byte[][] array) {}  
    // or   
    // public void F(int[,] array) {}  
    // public void F(byte[,] array) {}  
}  
```
