# <a name="compiler-error-cs8411"></a><span data-ttu-id="f6833-103">Error del compilador CS8411</span><span class="sxs-lookup"><span data-stu-id="f6833-103">Compiler Error CS8411</span></span>

<span data-ttu-id="f6833-104">Una instrucción foreach asincrónica no puede funcionar en variables de tipo "type", porque "type" no contiene una definición de instancia pública adecuada para "GetAsyncEnumerator".</span><span class="sxs-lookup"><span data-stu-id="f6833-104">Asynchronous foreach statement cannot operate on variables of type 'type' because 'type' does not contain a suitable public instance definition for 'GetAsyncEnumerator'</span></span>

<span data-ttu-id="f6833-105">La instrucción `await foreach` solo funciona en tipos que tienen una definición de `GetAsyncEnumerator`, como `IAsyncEnumerable<T>`.</span><span class="sxs-lookup"><span data-stu-id="f6833-105">`await foreach` statement operates only on types having a definition of `GetAsyncEnumerator`, such as `IAsyncEnumerable<T>`.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="f6833-106">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="f6833-106">To correct this error</span></span>

<span data-ttu-id="f6833-107">Reemplace `await foreach` por `foreach`.</span><span class="sxs-lookup"><span data-stu-id="f6833-107">Replace `await foreach` with `foreach`.</span></span>

## <a name="example"></a><span data-ttu-id="f6833-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="f6833-108">Example</span></span>

```csharp
using System.Collections.Generic;
using System.Threading.Tasks;

class Program
{
    async Task Example(IAsyncEnumerator<int> enumerator)
    {
        // error CS8411: Asynchronous foreach statement cannot operate on variables
        // of type 'IAsyncEnumerator<T>' because 'IAsyncEnumerator<T>' does not
        // contain a suitable public instance definition for 'GetAsyncEnumerator'
        await foreach (int i in enumerator)
        {
        }
    }
}
```
