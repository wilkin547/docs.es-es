# <a name="compiler-warning-level-1-cs3003"></a>Advertencia del compilador (nivel 1) CS3003
El tipo de 'variable' no es conforme a CLS  
  
 Una variable [pública](../../../csharp/language-reference/keywords/public.md), [protegida](../../../csharp/language-reference/keywords/protected.md) o `protected internal` debe ser de un tipo conforme a CLS (Common Language Specification). Para obtener más información sobre la conformidad con CLS, vea [Independencia del lenguaje y componentes independientes del lenguaje](https://msdn.microsoft.com/library/12a7a7h3).  
  
## <a name="example"></a>Ejemplo  
 El ejemplo siguiente genera el error CS3003:  
  
```csharp  
// CS3003.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public ushort a1;   // CS3003, public variable  
    public static void Main()  
    {  
    }  
}  
```
