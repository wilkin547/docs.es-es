# <a name="compiler-error-cs8403"></a>Error del compilador CS8403

El método "method" con un bloque de iteradores debe ser "asincrónico" para devolver "{IAsyncEnumerable\<T>|IAsyncEnumerator\<T>}".

## <a name="to-correct-this-error"></a>Para corregir este error

Marque el método con el modificador `async`.
