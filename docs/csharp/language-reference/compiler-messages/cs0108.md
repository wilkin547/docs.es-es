# <a name="compiler-warning-level-2-cs0108"></a><span data-ttu-id="d3326-103">Advertencia del compilador (nivel 2) CS0108</span><span class="sxs-lookup"><span data-stu-id="d3326-103">Compiler Warning (level 2) CS0108</span></span>

<span data-ttu-id="d3326-104">'member1' oculta el miembro heredado 'member2'.</span><span class="sxs-lookup"><span data-stu-id="d3326-104">'member1' hides inherited member 'member2'.</span></span> <span data-ttu-id="d3326-105">Use la palabra clave New si la ocultación estaba prevista.</span><span class="sxs-lookup"><span data-stu-id="d3326-105">Use the new keyword if hiding was intended.</span></span>

 <span data-ttu-id="d3326-106">Se ha declarado una variable con el mismo nombre como una variable en una clase base.</span><span class="sxs-lookup"><span data-stu-id="d3326-106">A variable was declared with the same name as a variable in a base class.</span></span> <span data-ttu-id="d3326-107">En cambio, la palabra clave [new](../keywords/new-modifier.md) no se ha usado.</span><span class="sxs-lookup"><span data-stu-id="d3326-107">However, the [new](../keywords/new-modifier.md) keyword was not used.</span></span> <span data-ttu-id="d3326-108">Esta advertencia le informa de que debería usar **new**; la variable se declara como si **new** se hubiera usado en la declaración.</span><span class="sxs-lookup"><span data-stu-id="d3326-108">This warning informs you that you should use **new**; the variable is declared as if **new** had been used in the declaration.</span></span>

 <span data-ttu-id="d3326-109">En el ejemplo siguiente se genera el error CS0108:</span><span class="sxs-lookup"><span data-stu-id="d3326-109">The following sample generates CS0108:</span></span>

```csharp
// CS0108.cs
// compile with: /W:2
using System;

namespace x
{
    public class clx
    {
        public int i = 1;
    }

    public class cly : clx
    {
        public static int i = 2;   // CS0108, use the new keyword
        // Use the following line instead:
        // public static new int i = 2;

      public static void Main()
      {
         Console.WriteLine(i);
      }
   }
}
```
