## <a name="examples"></a><span data-ttu-id="c9bef-111">Ejemplos</span><span class="sxs-lookup"><span data-stu-id="c9bef-111">Examples</span></span>

 <span data-ttu-id="c9bef-113">Este código crea el archivo DLL con la segunda copia del tipo ambiguo.</span><span class="sxs-lookup"><span data-stu-id="c9bef-113">This code creates the DLL with the second copy of the ambiguous type.</span></span>

```csharp  
namespace TypeBindConflicts
{  
   public class AggPubImpAggPubImp {}
```  

 <span data-ttu-id="c9bef-114">En el ejemplo siguiente se genera el error CS0433.</span><span class="sxs-lookup"><span data-stu-id="c9bef-114">The following example generates CS0433.</span></span>

```csharp  
      AggPubImpAggPubImp n6 = new AggPubImpAggPubImp();   // CS0433  
   }

```
