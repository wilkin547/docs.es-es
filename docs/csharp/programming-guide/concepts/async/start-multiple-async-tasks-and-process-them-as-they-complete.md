## <a name="create-the-asynchronous-sum-page-sizes-method"></a><span data-ttu-id="9692d-125">Creación de un método SumPageSizes asincrónico</span><span class="sxs-lookup"><span data-stu-id="9692d-125">Create the asynchronous sum page sizes method</span></span>

<span data-ttu-id="9692d-126">Agregue el método `SumPageSizesAsync` después del método `Main`:</span><span class="sxs-lookup"><span data-stu-id="9692d-126">Below the `Main` method, add the `SumPageSizesAsync` method:</span></span>

```csharp
static async Task SumPageSizesAsync()
{
    var stopwatch = Stopwatch.StartNew();

    IEnumerable<Task<int>> downloadTasksQuery =
        from url in s_urlList
        select ProcessUrlAsync(url, s_client);

    List<Task<int>> downloadTasks = downloadTasksQuery.ToList();

    int total = 0;
    while (downloadTasks.Any())
    {
        Task<int> finishedTask = await Task.WhenAny(downloadTasks);
        downloadTasks.Remove(finishedTask);
        total += await finishedTask;
    }

    stopwatch.Stop();

    Console.WriteLine($"\nTotal bytes returned:  {total:#,#}");
    Console.WriteLine($"Elapsed time:          {stopwatch.Elapsed}\n");
}
```

<span data-ttu-id="9692d-127">El método comienza creando una instancia e iniciando una clase <xref:System.Diagnostics.Stopwatch>.</span><span class="sxs-lookup"><span data-stu-id="9692d-127">The method starts by instantiating and starting a <xref:System.Diagnostics.Stopwatch>.</span></span> <span data-ttu-id="9692d-128">Después, incluye una consulta que, cuando se ejecuta, crea una colección de tareas.</span><span class="sxs-lookup"><span data-stu-id="9692d-128">It then includes a query that, when executed, creates a collection of tasks.</span></span> <span data-ttu-id="9692d-129">Cada llamada a `ProcessUrlAsync` en el siguiente código devuelve un objeto <xref:System.Threading.Tasks.Task%601>, donde `TResult` es un entero:</span><span class="sxs-lookup"><span data-stu-id="9692d-129">Each call to `ProcessUrlAsync` in the following code returns a <xref:System.Threading.Tasks.Task%601>, where `TResult` is an integer:</span></span>
