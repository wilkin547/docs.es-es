# <a name="security-in-net"></a>Seguridad en .NET
Common language runtime y .NET proporcionan muchas clases y servicios útiles que permiten a los desarrolladores escribir código seguro fácilmente y permiten a los administradores del sistema personalizar los permisos concedidos al código para que puede tener acceso a los recursos protegidos. Además, el runtime y .NET proporcionan clases y servicios que facilitan el uso de criptografía y la seguridad basada en funciones útiles.
