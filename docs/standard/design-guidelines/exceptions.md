# <a name="design-guidelines-for-exceptions"></a>Instrucciones de diseño de excepciones

El control de excepciones tiene muchas ventajas en comparación con los informes de errores basados en valores devueltos. Un buen diseño del marco ayuda al desarrollador de aplicaciones a obtener las ventajas de las excepciones. En esta sección se describen las ventajas de las excepciones y se presentan las instrucciones para usarlas de forma eficaz.
