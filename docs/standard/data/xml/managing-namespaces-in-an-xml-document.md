# <a name="managing-namespaces-in-an-xml-document"></a><span data-ttu-id="3e1cd-102">Administrar espacios de nombres en un documento XML</span><span class="sxs-lookup"><span data-stu-id="3e1cd-102">Managing Namespaces in an XML Document</span></span>
<span data-ttu-id="3e1cd-103">Los espacios de nombres XML asocian nombres de elementos y de atributos de un documento XML con identificadores URI personalizados y predefinidos.</span><span class="sxs-lookup"><span data-stu-id="3e1cd-103">XML namespaces associate element and attribute names in an XML document with custom and predefined URIs.</span></span> <span data-ttu-id="3e1cd-104">Para crear estas asociaciones, puede definir los prefijos de los URI del espacio de nombres y usar dichos prefijos para calificar los nombres de los elementos y de los atributos en los datos XML.</span><span class="sxs-lookup"><span data-stu-id="3e1cd-104">To create these associations, you define prefixes for namespace URIs, and use those prefixes to qualify element and attribute names in XML data.</span></span> <span data-ttu-id="3e1cd-105">Los espacios de nombres evitan conflictos de nombres de elementos y atributos y permiten que los elementos y atributos con el mismo nombre se traten y se validen de forma diferente.</span><span class="sxs-lookup"><span data-stu-id="3e1cd-105">Namespaces prevent element and attribute name collisions, and enable elements and attributes of the same name to be handled and validated differently.</span></span>

## <a name="declaring-namespaces"></a><span data-ttu-id="3e1cd-106">Declarar espacios de nombres</span><span class="sxs-lookup"><span data-stu-id="3e1cd-106">Declaring namespaces</span></span>  
 <span data-ttu-id="3e1cd-107">Para declarar un espacio de nombres en un elemento, puede usar el atributo `xmlns:`:</span><span class="sxs-lookup"><span data-stu-id="3e1cd-107">To declare a namespace on an element, you use the `xmlns:` attribute:</span></span>  
  
 `xmlns:<name>=<"uri">`  
  
 <span data-ttu-id="3e1cd-108">Donde `<name>` es el prefijo de espacio de nombres y `<"uri">` es el URI que identifica el espacio de nombres.</span><span class="sxs-lookup"><span data-stu-id="3e1cd-108">where `<name>` is the namespace prefix and `<"uri">` is the URI that identifies the namespace.</span></span> <span data-ttu-id="3e1cd-109">Después de declarar el prefijo, puede usarlo para calificar elementos y atributos de un documento XML y asociarlos al identificador URI del espacio de nombres.</span><span class="sxs-lookup"><span data-stu-id="3e1cd-109">After you declare the prefix, you can use it to qualify elements and attributes in an XML document and associate them with the namespace URI.</span></span> <span data-ttu-id="3e1cd-110">Como el prefijo de espacio de nombres se utiliza en todo el documento, debe ser corto.</span><span class="sxs-lookup"><span data-stu-id="3e1cd-110">Because the namespace prefix is used throughout a document, it should be short in length.</span></span>  
  
 <span data-ttu-id="3e1cd-111">Este ejemplo define dos elementos `BOOK`.</span><span class="sxs-lookup"><span data-stu-id="3e1cd-111">This example defines two `BOOK` elements.</span></span> <span data-ttu-id="3e1cd-112">El primer elemento se califica mediante el prefijo del espacio de nombres `mybook` y el segundo elemento se califica con el prefijo `bb`.</span><span class="sxs-lookup"><span data-stu-id="3e1cd-112">The first element is qualified by the prefix, `mybook`, and the second element is qualified by the prefix, `bb`.</span></span> <span data-ttu-id="3e1cd-113">Cada prefijo se asocia a un URI del espacio de nombres distinto:</span><span class="sxs-lookup"><span data-stu-id="3e1cd-113">Each prefix is associated with a different namespace URI:</span></span>  
  
```xml  
<mybook:BOOK xmlns:mybook="http://www.contoso.com/books.dtd">  
<bb:BOOK xmlns:bb="urn:blueyonderairlines">  
```  
  
 <span data-ttu-id="3e1cd-114">Para indicar que un elemento es parte de un espacio de nombres en particular, agréguele el prefijo de espacio de nombres.</span><span class="sxs-lookup"><span data-stu-id="3e1cd-114">To signify that an element is a part of a particular namespace, add the namespace prefix to it.</span></span> <span data-ttu-id="3e1cd-115">Por ejemplo, si el elemento `Author` pertenece al espacio de nombres `mybook`, se declara como `<mybook:Author>`.</span><span class="sxs-lookup"><span data-stu-id="3e1cd-115">For example, if a `Author` element belongs to the `mybook` namespace, it is declared as `<mybook:Author>`.</span></span>

## <a name="declaration-scope"></a><span data-ttu-id="3e1cd-116">Ámbito de la declaración</span><span class="sxs-lookup"><span data-stu-id="3e1cd-116">Declaration scope</span></span>  
 <span data-ttu-id="3e1cd-117">Un espacio de nombres es efectivo desde el momento de su declaración hasta el fin del elemento en el que se ha declarado.</span><span class="sxs-lookup"><span data-stu-id="3e1cd-117">A namespace is effective from its point of declaration until the end of the element it was declared in.</span></span> <span data-ttu-id="3e1cd-118">En este ejemplo, el espacio de nombres definido en el elemento `BOOK` no se aplica a elementos que estén fuera del elemento `BOOK`, tales como el elemento `Publisher`:</span><span class="sxs-lookup"><span data-stu-id="3e1cd-118">In this example, the namespace defined in the `BOOK` element doesn't apply to elements outside the `BOOK` element, such as the `Publisher` element:</span></span>  
  
```xml  
<Author>Joe Smith</Author>  
<BOOK xmlns:book="http://www.contoso.com">  
    <title>My Wonderful Day</title>  
      <price>$3.95</price>  
</BOOK>  
<Publisher>  
    <Name>MSPress</Name>  
</Publisher>  
```  
  
 <span data-ttu-id="3e1cd-119">Para usar un espacio de nombres, antes éste se debe declarar, pero no tiene que aparecer al principio del documento XML.</span><span class="sxs-lookup"><span data-stu-id="3e1cd-119">A namespace must be declared before it can be used, but it doesn't have to appear at the top of the XML document.</span></span>  
  
 <span data-ttu-id="3e1cd-120">Si se usan varios espacios de nombres en un documento XML, puede definir un espacio de nombres como predeterminado para crear un documento en el que se mejora la claridad.</span><span class="sxs-lookup"><span data-stu-id="3e1cd-120">When you use multiple namespaces in an XML document, you can define one namespace as the default namespace to create a cleaner looking document.</span></span> <span data-ttu-id="3e1cd-121">El espacio de nombres predeterminado se declara en el elemento raíz y se aplica a todos los elementos sin calificar del documento.</span><span class="sxs-lookup"><span data-stu-id="3e1cd-121">The default namespace is declared in the root element and applies to all unqualified elements in the document.</span></span> <span data-ttu-id="3e1cd-122">Los espacios de nombres predeterminados solo se aplican a los elementos, no a los atributos.</span><span class="sxs-lookup"><span data-stu-id="3e1cd-122">Default namespaces apply to elements only, not to attributes.</span></span>  
  
 <span data-ttu-id="3e1cd-123">Para usar el espacio de nombres predeterminado, omita el prefijo y los dos puntos de la declaración en el elemento:</span><span class="sxs-lookup"><span data-stu-id="3e1cd-123">To use the default namespace, omit the prefix and the colon from the declaration on the element:</span></span>  
  
```xml  
<BOOK xmlns="http://www.contoso.com/books.dtd">  
```  
  
## <a name="managing-namespaces"></a><span data-ttu-id="3e1cd-124">Administrar los espacios de nombres</span><span class="sxs-lookup"><span data-stu-id="3e1cd-124">Managing namespaces</span></span>
