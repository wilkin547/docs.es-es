# <a name="mybase-cannot-be-used-with-method-ltmethodnamegt-because-it-is-declared-mustoverride"></a>'MyBase' no se puede usar con el método '&lt;methodname&gt;' porque está declarado como 'MustOverride'
Ha intentado usar `MyBase` con un método que se ha declarado `MustOverride`.  
  
 **Identificador de error:** BC30399  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite la declaración `MustOverride` .
