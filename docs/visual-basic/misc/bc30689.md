# La instrucci&#243;n no puede aparecer fuera de un cuerpo de m&#233;todo o una expresi&#243;n lambda de varias l&#237;neas
Las instrucciones ejecutables deben usarse en los procedimientos.  
  
 **Identificador de error:** BC30689  
  
### Para corregir este error  
  
-   Mueva la instrucción a un procedimiento o subrutina.
