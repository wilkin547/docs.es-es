# La asignaci&#243;n enlazada en tiempo de ejecuci&#243;n a un campo de tipo de valor &#39;&lt;typename&gt;&#39; no es v&#225;lida si &#39;&lt;name&gt;&#39; es el resultado de una expresi&#243;n enlazada en tiempo de ejecuci&#243;n
Intentó realizar una asignación enlazada en tiempo de ejecución que no es válida.  
  
### Para corregir este error  
  
-   Realice la asignación en tiempo de compilación.
