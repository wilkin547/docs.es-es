# <a name="instance-members-and-me-cannot-be-used-within-a-lambda-expression-in-structures"></a><span data-ttu-id="beb0d-103">No se pueden usar miembros de instancia ni 'Me' dentro de una expresión lambda en estructuras</span><span class="sxs-lookup"><span data-stu-id="beb0d-103">Instance members and 'Me' cannot be used within a lambda expression in structures</span></span>

<span data-ttu-id="beb0d-104">Desde dentro de una estructura, se ha definido una expresión lambda que hace referencia a un miembro de instancia de la estructura o usa `Me`.</span><span class="sxs-lookup"><span data-stu-id="beb0d-104">From within a structure, you have defined a lambda expression that refers to an instance member of the structure or uses `Me`.</span></span> <span data-ttu-id="beb0d-105">El código siguiente muestra estas dos referencias no válidas.</span><span class="sxs-lookup"><span data-stu-id="beb0d-105">The following code illustrates these two invalid references.</span></span>  
  
```vb  
Structure Structure1  
  
    Public InstanceMember As Integer  
  
    Public Function ExampleFun() As Integer  
        '' The error is caused by use of InstanceMember.  
        'Dim fun1 = Function() InstanceMember  
        '' The error is caused by use of Me.  
        'Dim fun2 = Function() Me.InstanceMember  
        'Return fun1()  
    End Function  
  
End Structure  
```  
  
 <span data-ttu-id="beb0d-106">**Identificador de error:** BC36638</span><span class="sxs-lookup"><span data-stu-id="beb0d-106">**Error ID:** BC36638</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="beb0d-107">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="beb0d-107">To correct this error</span></span>  
  
- <span data-ttu-id="beb0d-108">Asigne el miembro de instancia a una variable local y use la variable local en la instrucción.</span><span class="sxs-lookup"><span data-stu-id="beb0d-108">Assign the instance member to a local variable, and use the local variable in your statement.</span></span>  
  
```vb  
Public Function ExampleFunFix() As Integer  
    Dim temp = InstanceMember  
    Dim fun1 = Function() temp  
    Return fun1()  
End Function  
```
