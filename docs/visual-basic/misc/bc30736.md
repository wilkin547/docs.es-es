# <a name="expression-cannot-be-evaluated-at-this-time"></a>No se puede evaluar la expresión en este momento

No se pudo evaluar la expresión especificada en el contexto actual.  
  
 **Identificador de error:** BC30736  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Cambie la sintaxis de la expresión.
