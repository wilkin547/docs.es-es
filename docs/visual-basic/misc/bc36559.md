# <a name="anonymous-type-member-property-propertyname-cannot-be-used-to-infer-the-type-of-another-member-property-because-the-type-of-propertyname-is-not-yet-established"></a><span data-ttu-id="0a38e-103">La propiedad '\<propertyname>' del miembro de tipo anónimo no se puede usar para inferir el tipo de otra propiedad de miembro porque el tipo de '\<propertyname>' no se ha establecido aún</span><span class="sxs-lookup"><span data-stu-id="0a38e-103">Anonymous type member property '\<propertyname>' cannot be used to infer the type of another member property because the type of '\<propertyname>' is not yet established</span></span>

<span data-ttu-id="0a38e-104">Hasta que se establezca el tipo de una la propiedad de tipo anónimo, no se lo puede usar para establecer el tipo de otra propiedad.</span><span class="sxs-lookup"><span data-stu-id="0a38e-104">Until the type of an anonymous type property is established, it cannot be used to establish the type of another property.</span></span> <span data-ttu-id="0a38e-105">Por ejemplo, en la siguiente declaración `.IDName = .LastName` no es válido porque `.LastName` todavía no se ha inicializado.</span><span class="sxs-lookup"><span data-stu-id="0a38e-105">For example, in the following declaration `.IDName = .LastName` is not valid because `.LastName` has not yet been initialized.</span></span>

```vb
' Not valid.
Dim anon1 = New With {Key .IDName = .LastName, Key .LastName = "Jones"}
```

<span data-ttu-id="0a38e-106">**Identificador de error:** BC36559</span><span class="sxs-lookup"><span data-stu-id="0a38e-106">**Error ID:** BC36559</span></span>
