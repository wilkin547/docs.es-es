# <a name="error-setting-assembly-manifest-option-error-message"></a>Error al establecer la opción de manifiesto del ensamblado: \<mensaje de error >
El compilador de Visual Basic llama a la herramienta Assembly Linker (Al.exe, también denominado Alink) para generar un ensamblado con un manifiesto. El vinculador notificó un error al buscar un archivo de clave o un nombre de clave con el que firmar el ensamblado.  
  
 **Identificador de error:** BC30139  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Examine el mensaje de error citado y consulte los errores enumerados en [Al.exe (Assembly Linker)](../../framework/tools/al-exe-assembly-linker.md) para obtener más información y consejos.  
  
2.  Si el error persiste, reúna información sobre las circunstancias y notifíquelo a los Servicios de soporte técnico de Microsoft.
