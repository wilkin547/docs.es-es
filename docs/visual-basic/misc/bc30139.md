# <a name="error-setting-assembly-manifest-option-error-message"></a>Error al configurar la opción de manifiesto del ensamblado: \<error message>

El compilador Visual Basic llama a Assembly Linker (Al.exe, también conocido como ALink) para generar un ensamblado con un manifiesto. El vinculador notificó un error al buscar un archivo de clave o un nombre de clave con el que firmar el ensamblado.  
  
 **Identificador de error:** BC30139  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Examine el mensaje de error citado y consulte los errores enumerados en [Al.exe (Assembly Linker)](../../framework/tools/al-exe-assembly-linker.md) para obtener una explicación más detallada y consejos.  
  
2. Si el error persiste, reúna información sobre las circunstancias y notifíquelo a los Servicios de soporte técnico de Microsoft.
