# &#39;End Get&#39; debe ser la primera instrucci&#243;n de una l&#237;nea
Una instrucción `End Get` sigue a un separador de instrucciones de dos puntos \(:\).`End Get` debe ser la única instrucción en su línea de código fuente.  
  
 **Identificador de error:** BC32033  
  
### Para corregir este error  
  
-   Separe las distintas instrucciones en líneas diferentes.
