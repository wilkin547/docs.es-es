# <a name="expression-expected"></a>Se esperaba una expresión

Un operador como `+` o `=` necesita una expresión para su operando derecho, pero no hay ninguna expresión reconocible.  
  
 **Identificador de error:** BC30201  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Proporcione una expresión válida para el operando derecho del operador.
