# <a name="type-argument-typeargumentname-is-declared-mustinherit-and-does-not-satisfy-the-new-constraint-for-type-parameter-typeparametername"></a><span data-ttu-id="f1a13-102">Argumento de tipo '\<nombredeargumentodetipo >' está declarado como 'MustInherit' y no satisface la restricción 'New' para el parámetro de tipo '\<typeparametername >'</span><span class="sxs-lookup"><span data-stu-id="f1a13-102">Type argument '\<typeargumentname>' is declared 'MustInherit' and does not satisfy the 'New' constraint for type parameter '\<typeparametername>'</span></span>
<span data-ttu-id="f1a13-103">Se invoca un tipo genérico con una clase `MustInherit` como argumento de tipo, pero el parámetro de tipo correspondiente está declarado con la restricción `New` .</span><span class="sxs-lookup"><span data-stu-id="f1a13-103">A generic type is invoked with a `MustInherit` class as a type argument, while the corresponding type parameter is declared with the `New` constraint.</span></span>  
  
 <span data-ttu-id="f1a13-104">La restricción `New` establece que el tipo que se pasa en el argumento de tipo correspondiente debe admitir la creación de objetos.</span><span class="sxs-lookup"><span data-stu-id="f1a13-104">The `New` constraint requires that the type passed in the corresponding type argument must support the creation of objects.</span></span> <span data-ttu-id="f1a13-105">Sin embargo, una clase *abstract* , es decir, una clase declarada como `MustInherit`, no expone ningún constructor porque no es posible crear objetos a partir de ella.</span><span class="sxs-lookup"><span data-stu-id="f1a13-105">However, an *abstract* class, that is, a class declared as `MustInherit`, does not expose any constructors because you cannot create any objects from it.</span></span>  
  
 <span data-ttu-id="f1a13-106">**Identificador de error:** BC32082</span><span class="sxs-lookup"><span data-stu-id="f1a13-106">**Error ID:** BC32082</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="f1a13-107">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="f1a13-107">To correct this error</span></span>

3.  <span data-ttu-id="f1a13-110">Si el parámetro de tipo correspondiente no necesita crear objetos a partir del tipo que se le pasa, quite la restricción `New` de su declaración.</span><span class="sxs-lookup"><span data-stu-id="f1a13-110">If the corresponding type parameter does not need to create any objects from the type passed to it, then remove the `New` constraint from its declaration.</span></span>
