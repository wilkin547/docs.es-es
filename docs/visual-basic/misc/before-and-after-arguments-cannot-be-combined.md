# <a name="before-and-after-arguments-cannot-be-combined"></a>Los argumentos 'Before' y 'After' no se pueden combinar

Una lista de argumentos incluye ambos argumentos `Before` y `After` , que son mutuamente excluyentes.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Quite el argumento `Before` o `After` .
