# No se encuentra el campo o la propiedad &#39;&lt;membername&gt;&#39;
Se hace referencia a una propiedad o un campo no definidos del atributo.  
  
 **Id. de error:** BC30661  
  
### Para corregir este error  
  
1.  Compruebe que el campo o la propiedad estén bien escritos.  
  
2.  Compruebe que el proyecto puede acceder a la clase de atributo correcta y agréguela a las referencias del proyecto, si es necesario.
