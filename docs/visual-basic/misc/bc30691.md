# <a name="lttypenamegt-is-a-type-in-ltcontainernamegt-and-cannot-be-used-as-an-expression"></a>'&lt;typename&gt;'es un tipo en'&lt;containername&gt;' y no se puede usar como una expresión.
Uno de los nombres que se va a usar ya se ha declarado como el nombre de un tipo y no se puede usar en una expresión.  
  
 **Identificador de error:** BC30691  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Nombre único del miembro.  
  
-   Usar corchetes alrededor del nombre del miembro.
