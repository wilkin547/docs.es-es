# Los miembros de una estructura no se pueden declarar como &#39;&lt;specifier&gt;&#39;
Se ha usado un especificador que no es válido en un miembro dentro de una instrucción `Structure`.  
  
 **Id. de error:** BC30435  
  
### Para corregir este error  
  
-   Quite el especificador.
