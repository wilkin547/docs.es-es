# <a name="members-in-a-structure-cannot-be-declared-39ltspecifiergt39"></a>No se pueden declarar miembros de una estructura &#39; &lt;especificador&gt;&#39;
Se ha usado un especificador que no es válido en un miembro dentro de una instrucción `Structure` .  
  
 **Id. de error:** BC30435  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el especificador.
