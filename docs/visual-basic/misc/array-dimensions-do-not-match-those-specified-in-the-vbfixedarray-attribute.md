# <a name="array-dimensions-do-not-match-those-specified-in-the-vbfixedarray-attribute"></a>Las dimensiones de la matriz no coinciden con las especificadas en el atributo 'VBFixedArray'.

Las dimensiones de la matriz no coinciden con las dimensiones que ha especificado en el atributo `VBFixedArray` .  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Haga que las dimensiones de la matriz de la instrucción `Dim` de la matriz coincida con las dimensiones especificadas por el atributo `VBFixedArray` .
