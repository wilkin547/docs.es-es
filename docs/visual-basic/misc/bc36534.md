# <a name="expression-cannot-be-converted-into-an-expression-tree"></a>La expresión no se puede convertir en un árbol de expresión

La mayoría de expresiones se pueden convertir en árboles de expresión, pero algunas no. Este error indica que se intentó convertir una expresión que no se puede convertir.  
  
 **Id. de error:** BC36534
