# &#39;&lt;modificador&gt;&#39; no es v&#225;lido en una declaraci&#243;n de m&#233;todo
Una instrucción `Function` o `Sub` contiene una palabra clave no válida.  
  
 **Identificador de error:** BC30242  
  
### Para corregir este error  
  
-   Quite la palabra clave no válida de la instrucción `Function` o `Sub`.
