# <a name="inheriting-from-39systemltclassnamegt39-is-not-valid"></a>Heredar &#39;sistema. &lt;classname&gt; &#39; no es válido
Una clase no puede heredar de `System.Array`, `System.Delegate`, `System.Enum`o `System.ValueType`.  
  
 **Identificador de error:** BC30015  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite la instrucción `Inherits` o cámbiela para heredar de una clase base válida.
