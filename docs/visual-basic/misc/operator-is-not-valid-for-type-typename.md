# <a name="operator-is-not-valid-for-type-typename"></a>El operador no es válido para el tipo ' \<typename> '

Un operador no es adecuado para los tipos.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Asegúrese de que el operador es adecuado para los tipos en los que opera.
