# <a name="cannot-inherit-interface-interfacename1-because-the-interface-interfacename2-from-which-it-inherits-could-be-identical-to-interface-interfacename3-from-which-the-interface-interfacename4-inherits-for-some-type-arguments"></a><span data-ttu-id="50239-103">No se puede heredar la interfaz '\<interfacename1>' porque la interfaz '\<interfacename2>' de la que hereda podría ser idéntica a la interfaz '\<interfacename3>' de la que la interfaz '\<interfacename4>' hereda para algunos argumentos de tipo</span><span class="sxs-lookup"><span data-stu-id="50239-103">Cannot inherit interface '\<interfacename1>' because the interface '\<interfacename2>' from which it inherits could be identical to interface '\<interfacename3>' from which the interface '\<interfacename4>' inherits for some type arguments</span></span>

<span data-ttu-id="50239-104">Una interfaz genérica hereda de dos o más interfaces genéricas, y dos de las herencias podrían entrar en conflicto para ciertos valores de argumentos de tipo.</span><span class="sxs-lookup"><span data-stu-id="50239-104">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>

 <span data-ttu-id="50239-105">Las instrucciones siguientes pueden generar este error.</span><span class="sxs-lookup"><span data-stu-id="50239-105">The following statements can generate this error.</span></span>

```vb
Public Interface interfaceA(Of u)
End Interface
Public Interface interfaceX(Of v)
    Inherits interfaceA(Of v)
End Interface
Public Interface interfaceY(Of w)
    Inherits interfaceA(Of w)
End Interface
Public Interface derivedInterface(Of t1, t2)
    Inherits interfaceX(Of t1), interfaceY(Of t2)
End Interface
```

<span data-ttu-id="50239-106">Si `derivedInterface` se construye o se implementa al proporcionar el mismo tipo tanto a `t1` como a `t2`, debe heredar dos versiones de `interfaceA` con argumentos de tipo idénticos.</span><span class="sxs-lookup"><span data-stu-id="50239-106">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="50239-107">Esto produciría ambigüedad respecto a la versión a la que se debe acceder.</span><span class="sxs-lookup"><span data-stu-id="50239-107">Doing so would produce an ambiguity about which version to access.</span></span>

<span data-ttu-id="50239-108">**Id. de error:** BC32122</span><span class="sxs-lookup"><span data-stu-id="50239-108">**Error ID:** BC32122</span></span>
