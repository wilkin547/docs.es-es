# La palabra clave no denomina un tipo.
Aparece una palabra clave donde se requiere un nombre de tipo. El contexto llama a un tipo de datos básico o a un nombre de elemento declarado que represente un tipo.  
  
 **Identificador de error:** BC30180  
  
### Para corregir este error  
  
-   Reemplace la palabra clave con un nombre de tipo válido.
