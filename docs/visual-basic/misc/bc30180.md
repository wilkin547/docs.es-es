# <a name="keyword-does-not-name-a-type"></a>La palabra clave no denomina un tipo.

Aparece una palabra clave donde se requiere un nombre de tipo. El contexto llama a un tipo de datos básico o a un nombre de elemento declarado que representa un tipo.  
  
 **Identificador de error:** BC30180  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Reemplace la palabra clave con un nombre de tipo válido.
