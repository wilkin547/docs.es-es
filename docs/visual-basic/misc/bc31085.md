# <a name="date-constant-is-not-valid"></a>La constante de fecha no es válida

Se declaró una constante de fecha con una fecha no válida.  
  
 **Id. de error:** BC31085  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Corrija la fecha utilizada en la constante.
