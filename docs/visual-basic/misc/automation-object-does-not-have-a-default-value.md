# El objeto de automatizaci&#243;n no tiene un valor predeterminado
[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] no puede determinar el miembro predeterminado del objeto especificado.  
  
### Para corregir el error  
  
-   Compruebe la documentación del objeto e incluya una especificación explícita para la propiedad o el método.
