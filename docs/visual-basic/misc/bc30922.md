# <a name="derivedtypename-cannot-inherit-from-type-constructedbasetypename-because-it-expands-the-access-of-type-internaltypename-outside-the-assembly"></a><span data-ttu-id="e077f-103">' \<derivedtypename> ' no puede heredar de \<type> ' \<constructedbasetypename> ' porque expande el acceso del tipo ' \<internaltypename> ' fuera del ensamblado</span><span class="sxs-lookup"><span data-stu-id="e077f-103">'\<derivedtypename>' cannot inherit from \<type> '\<constructedbasetypename>' because it expands the access of type '\<internaltypename>' outside the assembly</span></span>

<span data-ttu-id="e077f-104">Una clase o interfaz derivada intenta ampliar el nivel de acceso de un tipo restringido. Para ello, lo usa como un argumento de tipo para una clase base o una interfaz.</span><span class="sxs-lookup"><span data-stu-id="e077f-104">A derived class or interface attempts to expand the access level of a restricted type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="e077f-105">El código siguiente puede generar este error.</span><span class="sxs-lookup"><span data-stu-id="e077f-105">The following code can generate this error.</span></span>  
  
```vb  
Public Class baseClass(Of t)  
End Class  
Public Class derivedClass  
    Inherits baseClass(Of restrictedStructure)  
End Class  
Friend Structure restrictedStructure  
    Dim firstMember As Integer  
End Structure  
```  
  
 <span data-ttu-id="e077f-106">El código situado fuera del ensamblado no puede acceder a `restrictedStructure`.</span><span class="sxs-lookup"><span data-stu-id="e077f-106">Code outside the assembly is not allowed to access `restrictedStructure`.</span></span> <span data-ttu-id="e077f-107">Sin embargo, se puede acceder a `derivedClass` desde cualquier código que pueda hacer referencia a este.</span><span class="sxs-lookup"><span data-stu-id="e077f-107">However, `derivedClass` can be accessed from any code that can reference it.</span></span> <span data-ttu-id="e077f-108">Por lo tanto, `derivedClass` no puede heredar `baseClass` si usa `restrictedStructure` como un argumento de tipo, porque ello podría exponer `restrictedStructure` al código de cualquier ensamblado.</span><span class="sxs-lookup"><span data-stu-id="e077f-108">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `restrictedStructure` as a type argument, because that could expose `restrictedStructure` to code in any assembly.</span></span>  
  
 <span data-ttu-id="e077f-109">**Identificador de error:** BC30922</span><span class="sxs-lookup"><span data-stu-id="e077f-109">**Error ID:** BC30922</span></span>

- <span data-ttu-id="e077f-111">Ajuste los niveles de acceso de las clases o las interfaces para que el tipo derivado no amplíe el nivel de acceso del tipo restringido.</span><span class="sxs-lookup"><span data-stu-id="e077f-111">Adjust the access levels of the classes or interfaces so the derived type does not expand the access level of the restricted type.</span></span>  
  
     <span data-ttu-id="e077f-112">o bien</span><span class="sxs-lookup"><span data-stu-id="e077f-112">-or-</span></span>  
  
- <span data-ttu-id="e077f-113">Si no puede ajustar los niveles de acceso, no use el tipo restringido como un argumento de tipo al construir la clase base o la interfaz.</span><span class="sxs-lookup"><span data-stu-id="e077f-113">If you cannot adjust the access levels, do not use the restricted type as a type argument when constructing the base class or interface.</span></span>
