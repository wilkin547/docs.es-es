# <a name="xml-literal-cannot-appear-here-unless-it-is-enclosed-in-parentheses"></a><span data-ttu-id="9718e-102">Aquí no puede aparecer un literal XML a menos que se incluya entre paréntesis</span><span class="sxs-lookup"><span data-stu-id="9718e-102">XML literal cannot appear here unless it is enclosed in parentheses</span></span>
<span data-ttu-id="9718e-103">Una declaración de literal de XML se usa en una expresión de una ubicación que es ambigua para el compilador de [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] .</span><span class="sxs-lookup"><span data-stu-id="9718e-103">An XML literal declaration is used in an expression in a location that is ambiguous to the [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] compiler.</span></span> <span data-ttu-id="9718e-104">Es decir, el [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] compilador no puede determinar si la menor-que carácter (<) está pensado como operador comparativo o para el inicio de un literal XML.</span><span class="sxs-lookup"><span data-stu-id="9718e-104">That is, the [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] compiler cannot determine whether the less-than character (<) is intended as a comparative operator or the start of an XML literal.</span></span> <span data-ttu-id="9718e-105">El siguiente fragmento de código muestra un ejemplo:</span><span class="sxs-lookup"><span data-stu-id="9718e-105">The following code provides an example:</span></span>  

```vb  
' Generates an error.  
Dim queryResult = From element In elements _  
                  Where <sample>Value</sample> = "Value" _  
                  Select element  
```  
  
 <span data-ttu-id="9718e-106">**Identificador de error:** BC31198</span><span class="sxs-lookup"><span data-stu-id="9718e-106">**Error ID:** BC31198</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="9718e-107">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="9718e-107">To correct this error</span></span>  
  
-   <span data-ttu-id="9718e-108">Incluya la declaración de literal XML entre paréntesis, tal como se muestra en el ejemplo siguiente:</span><span class="sxs-lookup"><span data-stu-id="9718e-108">Enclose the XML literal declaration in parentheses, as shown in the following example:</span></span>  
  
```vb

                  Where (<sample> Value</sample>) = "Value" _  
                  Select element  
```  
  
-   <span data-ttu-id="9718e-109">Modifique el código para hacer referencia a un literal XML existente, tal como se muestra en el ejemplo siguiente:</span><span class="sxs-lookup"><span data-stu-id="9718e-109">Modify your code to refer to an existing XML literal, as shown in the following example:</span></span>

```vb  
                  Where e.<sample>.Value = "Value" _  
                  Select element  
```
