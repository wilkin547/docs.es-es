# <a name="readonly-variable-cannot-be-the-target-of-an-assignment-in-a-lambda-expression-inside-a-constructor"></a><span data-ttu-id="ec035-103">La variable 'ReadOnly' no puede ser el destino de una asignación en una expresión lambda dentro de un constructor</span><span class="sxs-lookup"><span data-stu-id="ec035-103">'ReadOnly' variable cannot be the target of an assignment in a lambda expression inside a constructor</span></span>

<span data-ttu-id="ec035-104">Hizo referencia a una variable `ReadOnly` desde una expresión lambda, lo que no está permitido.</span><span class="sxs-lookup"><span data-stu-id="ec035-104">You have referred to a `ReadOnly` variable from within a lambda expression, which is not permitted.</span></span> <span data-ttu-id="ec035-105">El código siguiente provoca este error al enviar la variable de `ReadOnly``m` como el argumento de un parámetro `ByRef` .</span><span class="sxs-lookup"><span data-stu-id="ec035-105">The following code causes this error by sending the `ReadOnly` variable `m` as the argument to a `ByRef` parameter.</span></span>  
  
```vb  
Class Class1  
  
    ReadOnly m As Integer  
  
    Sub New()  
        ' The use of m triggers the error.  
        Dim f = Function() Test(m)  
    End Sub  
  
    Function Test(ByRef n As Integer) As String  
    End Function  
  
End Class  
```  
  
 <span data-ttu-id="ec035-106">**Identificador de error:** BC36602</span><span class="sxs-lookup"><span data-stu-id="ec035-106">**Error ID:** BC36602</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="ec035-107">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="ec035-107">To correct this error</span></span>  
  
- <span data-ttu-id="ec035-108">Si es posible, cambie el parámetro `n` de la función `Test` a un parámetro `ByVal` , como se muestra en el código siguiente.</span><span class="sxs-lookup"><span data-stu-id="ec035-108">If possible, change parameter `n` in function `Test` to a `ByVal` parameter, as shown in the following code.</span></span>  
  
    ```vb  
    Class Class1Fix1  
  
        ReadOnly m As Integer  
  
        Sub New()  
            Dim f = Function() Test(m)  
        End Sub  
  
        Function Test(ByVal n As Integer) As String  
        End Function  
  
    End Class  
    ```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="ec035-109">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="ec035-109">To correct this error</span></span>  
  
- <span data-ttu-id="ec035-110">Asigne la variable `ReadOnly``m` a una variable local en el constructor y use la variable local en lugar de `m`, como se muestra en el código siguiente.</span><span class="sxs-lookup"><span data-stu-id="ec035-110">Assign the `ReadOnly` variable `m` to a local variable in the constructor, and use the local variable in place of `m`, as shown in the following code.</span></span>  
  
    ```vb  
    Class Class1Fix2  
         ReadOnly m As Integer  
  
        Sub New()  
            Dim temp = m  
            Dim f = Function() Test(temp)  
        End Sub  
  
        Function Test(ByRef n As Integer) As String  
        End Function  
  
    End Class  
    ```
