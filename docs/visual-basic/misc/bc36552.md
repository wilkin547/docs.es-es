## <a name="example"></a><span data-ttu-id="5349b-110">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="5349b-110">Example</span></span>  
 <span data-ttu-id="5349b-111">El primer parámetro del ejemplo siguiente indica que el método `Print` extiende el tipo de datos `String` .</span><span class="sxs-lookup"><span data-stu-id="5349b-111">The first parameter in the following example indicates that the `Print` method extends the `String` data type.</span></span>

```  
    Console.WriteLine(str)  
End Sub  
```
