# &#39;Enum&#39; debe terminar con la instrucci&#243;n &#39;End Enum&#39; correspondiente.
Una instrucción `Enum` aparece sin una instrucción `End Enum` correspondiente. Una instrucción `End Enum` debe usarse para finalizar la enumeración.  
  
 **Identificador de error:** BC30185  
  
### Para corregir este error  
  
1.  Compruebe que los miembros `Enum` tengan el formato correcto.  
  
2.  Agregue una instrucción `End Enum` al final de la enumeración.
