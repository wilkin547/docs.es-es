# <a name="raiseevent-statements-are-not-valid-in-the-immediate-window"></a>Las instrucciones 'RaiseEvent' no son válidas en la ventana Inmediato.

`RaiseEvent` las instrucciones no se pueden usar en la ventana **inmediato** .  
  
 **Identificador de error:** BC30739  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Pruebe las instrucciones `RaiseEvent` desde el código del **Editor de código**.
