# La clase no est&#225; registrada en el equipo local.
Normalmente se encuentra este error como resultado de un archivo dañado. Puede que necesite volver a instalar [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
### Para corregir este error  
  
-   Vuelva a instalar [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].
