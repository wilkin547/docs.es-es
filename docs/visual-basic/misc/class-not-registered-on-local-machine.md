# <a name="class-not-registered-on-local-machine"></a>La clase no está registrada en el equipo local.

Normalmente se encuentra este error como resultado de un archivo dañado. Es posible que tenga que volver a instalar Visual Studio.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Vuelva a instalar Visual Studio.
