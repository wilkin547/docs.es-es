# El par&#225;metro &#39;&lt;parametername&gt;&#39; correspondiente del argumento se reduce de &#39;&lt;type1&gt;&#39; a &#39;&lt;type2&gt;&#39;
Ha realizado una llamada a un método sobrecargado, pero el compilador no encuentra un método al que se pueda llamar sin una conversión de restricción. Una conversión de restricción cambia un valor a un tipo de datos que podría no mantener con precisión algunos de los valores posibles.  
  
 **Id. de error:** BC30520  
  
### Para corregir este error  
  
-   Especifique `Option Strict Off`.
