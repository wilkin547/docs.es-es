# <a name="option-optionname-requires-argument"></a>Opción \<optionname > requiere \<argumento >
Se ha omitido un argumento requerido.  
  
 **Identificador de error:** BC2006  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Facilite el argumento que falta.
