# Hay pendiente una solicitud de suspensi&#243;n
En el depurador de Visual Studio, una expresión especifica una llamada a un procedimiento, pero hay una solicitud de detención del subproceso.  
  
 El depurador no intenta llamar a un procedimiento de un subproceso que no está activo.  
  
 **Identificador de error:** BC30947  
  
### Para corregir este error  
  
-   Si es posible, determine el origen de la solicitud para detener el subproceso para poder evitar que vuelva a producirse.  
  
-   Finalice y reinicie la depuración.
