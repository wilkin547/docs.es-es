# &#39;&lt;argumentname&gt;&#39; no es un par&#225;metro de &#39;&lt;methoddeclaration&gt;&#39;
Una llamada a procedimiento facilita un argumento adicional que no lo define el procedimiento.  
  
 **Id. de error:** BC30272  
  
### Para corregir este error  
  
-   Quite el argumento superfluo de la lista de argumentos.
