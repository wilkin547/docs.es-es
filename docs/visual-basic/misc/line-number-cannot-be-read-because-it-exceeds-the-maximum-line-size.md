# <a name="line-number-cannot-be-read-because-it-exceeds-the-maximum-line-size"></a>La línea \<number> no se puede leer porque supera el tamaño de línea máximo

La línea no se puede leer porque supera el tamaño de línea máximo. Normalmente, esto significa que en el archivo hay comillas de apertura, pero no hay comillas de cierre.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Inserte una comilla de cierre en el archivo.
