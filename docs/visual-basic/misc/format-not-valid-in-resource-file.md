# El formato del archivo de recursos no es v&#225;lido.
Un archivo de recursos en el proyecto no sigue el formato estándar para un archivo de recursos de Windows o bien el archivo de recursos y el proyecto [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] no coinciden.  
  
### Para corregir este error  
  
1.  Vuelva a crear el origen de recursos y los archivos de recursos de forma que se ajusten a la sintaxis documentada en el Kit de desarrollo de software de Windows.  
  
2.  Elimine el archivo de recursos no válido del proyecto.
