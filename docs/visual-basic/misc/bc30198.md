# <a name="-expected"></a>Se esperaba ')'
Una lista, como una lista de argumentos en una definición de procedimiento o una llamada de procedimiento, no termina con un carácter de paréntesis de cierre (`)`).  
  
 **Identificador de error:** BC30198  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Agregue un paréntesis de cierre al final de la lista.
