# <a name="compare-expected"></a>Se esperaba 'Compare'.

Una instrucción `Option` especifica `Text` o `Binary`, pero no `Compare`.  
  
 **Identificador de error:** BC30208  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Agregue la palabra clave `Compare` a la instrucción `Option` ; por ejemplo, `Option Compare Binary`.
