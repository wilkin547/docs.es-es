# El car&#225;cter de tipo no se puede usar en una declaraci&#243;n de par&#225;metro de tipo.
Una declaración de parámetro de tipo contiene al menos un carácter de tipo identificador.  
  
 Un parámetro de tipo tipo genérico debe ser un nombre válido de Visual Basic. Los caracteres permitidos no incluyen ninguno de los caracteres de tipo identificador \(`%`, `&`, `@`, `!`, `#` y `$`\). Vea [Nombres de elementos declarados](../../visual-basic/programming-guide/language-features/declared-elements/declared-element-names.md).  
  
 **Identificador de error:** BC32041  
  
### Para corregir este error  
  
-   Quite el carácter o los caracteres de tipo identificador de la declaración de parámetro de tipo.
