# <a name="type-character-cannot-be-used-in-a-type-parameter-declaration"></a>El carácter de tipo no se puede usar en una declaración de parámetro de tipo.

Una declaración de parámetro de tipo contiene al menos un carácter de tipo identificador.  
  
 Un parámetro de tipo tipo genérico debe ser un nombre válido de Visual Basic. Los caracteres permitidos no incluyen ninguno de los caracteres de tipo de identificador (`%`, `&`, `@`, `!`, `#`y `$`). Vea [Declared Element Names](../programming-guide/language-features/declared-elements/declared-element-names.md).  
  
 **Identificador de error:** BC32041  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite el carácter o los caracteres de tipo identificador de la declaración de parámetro de tipo.
