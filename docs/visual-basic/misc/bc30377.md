# <a name="method-ltmethodnamegt-is-already-declared-in-interface-ltinterfacenamegt"></a>Método '&lt;methodname&gt;'ya está declarado en la interfaz'&lt;interfacename&gt;'

 **Identificador de error:** BC30377  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite la declaración duplicada.
