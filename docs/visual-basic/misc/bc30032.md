# Los eventos no pueden tener un tipo de valor devuelto
Los eventos pueden aceptar argumentos, pero no pueden devolver valores directamente.  
  
 **Identificador de error:** BC30032  
  
### Para corregir este error  
  
-   Quite el tipo de valor devuelto de la instrucción `Event`.
