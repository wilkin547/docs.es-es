# <a name="class-classname-cannot-be-indexed-because-it-has-no-default-property"></a>No se puede indizar la clase '\<classname>' porque no tiene ninguna propiedad predeterminada

No suministró una propiedad predeterminada para una clase.  
  
 **Identificador de error:** BC30367  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Especifique una propiedad como propiedad predeterminada comenzando su declaración con la palabra clave `Default` .
