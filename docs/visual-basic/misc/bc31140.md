# <a name="custom-modifier-can-only-be-used-immediately-before-an-event-declaration"></a>El modificador 'Custom' solo se puede usar inmediatamente antes de una declaración 'Event'.

La palabra clave `Custom` debe preceder solo a la palabra clave `Event` en una declaración de evento personalizado.  
  
 **Identificador de error:** BC31140  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Mueva la palabra clave `Custom` para que preceda a la palabra clave `Event` .  
  
     -O bien-  
  
- Quite la palabra clave `Custom` .
