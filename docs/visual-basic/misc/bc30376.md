# <a name="39new39-cannot-be-used-on-class-39ltclassnamegt39-because-it-contains-a-39mustoverride39-member-that-has-not-been-overridden"></a>&#39; Nuevo &#39; no se puede usar en la clase &#39; &lt;classname&gt;&#39; porque contiene un &#39; MustOverride &#39; miembro que no se ha reemplazado.
Intentó usar `New` en una clase que contiene un miembro `MustOverride` que no se reemplazó.  
  
 **Identificador de error:** BC30376  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite la instrucción `New` .
