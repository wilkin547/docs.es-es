# <a name="new-cannot-be-used-on-class-classname-because-it-contains-a-mustoverride-member-that-has-not-been-overridden"></a>'New' no se puede usar en la clase '\<classname>' porque contiene un miembro 'MustOverride' que no se ha invalidado

Intentó usar `New` en una clase que contiene un miembro `MustOverride` que no se reemplazó.  
  
 **Identificador de error:** BC30376  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite la instrucción `New` .
