# <a name="expanded-properties-cannot-be-initialized"></a><span data-ttu-id="4eb50-102">Las propiedades expandidas no se pueden inicializar</span><span class="sxs-lookup"><span data-stu-id="4eb50-102">Expanded properties cannot be initialized</span></span>
<span data-ttu-id="4eb50-103">Se puede inicializar una propiedad implementada automáticamente como parte de su declaración, pero no una propiedad expandida.</span><span class="sxs-lookup"><span data-stu-id="4eb50-103">An auto-implemented property can be initialized as part of its declaration, but an expanded property cannot be.</span></span>  
  
 <span data-ttu-id="4eb50-104">**Identificador de error:** BC36714</span><span class="sxs-lookup"><span data-stu-id="4eb50-104">**Error ID:** BC36714</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="4eb50-105">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="4eb50-105">To correct this error</span></span>  
  
-   <span data-ttu-id="4eb50-106">Use una propiedad implementada automáticamente o quite la inicialización de la declaración de propiedad.</span><span class="sxs-lookup"><span data-stu-id="4eb50-106">Either use an auto-implemented property or remove the initialization from the property declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="4eb50-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="4eb50-107">Example</span></span>  
 <span data-ttu-id="4eb50-108">Los ejemplos siguientes muestran propiedades implementadas automáticamente y propiedades expandidas.</span><span class="sxs-lookup"><span data-stu-id="4eb50-108">The following examples show both auto-implemented and expanded properties.</span></span> <span data-ttu-id="4eb50-109">Se pueden inicializar propiedades implementadas automáticamente mediante la asignación o una cláusula `New` , pero no propiedades expandidas.</span><span class="sxs-lookup"><span data-stu-id="4eb50-109">Auto-implemented properties can be initialized by using assignment or a `New` clause, but expanded properties cannot be.</span></span>  
  
```vb  
Class AutoImplementedExample  
    ' An auto-implemented property can be assigned an initial value.  
    Property IDNum As Integer = 33542  
    ' An auto-implemented property can be initialized with New.  
    Property Name As New String("Don Hall")  
End Class  
  
Class ExpandedExample  
    ' Attempting to expand an initialized auto-implemented property  
    ' causes this error.  
    'Property IDNum As Integer = 33542  
    '    Get  
    '    End Get  
    '    Set(ByVal value As Integer)  
    '    End Set  
    'End Property  
  
    ' Instead, you can assign the initial value to the backing field.  
    Private _IDNum As Integer = 33542  
    Property IDNum As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
End Class  
```
