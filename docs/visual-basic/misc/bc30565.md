# <a name="array-initializer-has-too-few-dimensions"></a>El inicializador de matriz no tiene suficientes dimensiones.

No ha proporcionado suficientes dimensiones en el inicializador de la matriz.  
  
 **Identificador de error:** BC30565  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Compruebe el inicializador de matriz para determinar cuántas dimensiones son necesarias.  
  
- Proporcione el argumento o argumentos que faltan.
