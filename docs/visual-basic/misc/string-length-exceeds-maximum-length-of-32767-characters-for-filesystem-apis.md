# La longitud de cadena supera la longitud m&#225;xima de 32767 caracteres para las API de &#39;FileSystem&#39;.
La longitud de cadena supera la longitud máxima de 32767 caracteres.  
  
### Para corregir este error  
  
1.  Acorte la cadena.
