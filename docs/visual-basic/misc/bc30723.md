# <a name="method-call-did-not-return-a-value"></a>Llamada a método no devolvió un valor
El método llamado no devolvió un valor como se esperaba.  
  
 **Identificador de error:** BC30723  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Compruebe que el método llamado es una función que devuelve un valor.
