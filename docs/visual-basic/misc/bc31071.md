# Una estructura de una interfaz no se puede declarar como &#39;&lt;specifier&gt;&#39;
Se especificó un modificador no válido con una estructura declarada en una interfaz.  
  
 **Identificador de error:** BC31071  
  
### Para corregir este error  
  
-   Quite los modificadores como, por ejemplo, `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend` o `Overridable`.
