# <a name="methods-declared-overrides-cannot-be-declared-overridable-because-they-are-implicitly-overridable"></a>Los métodos declarados como 'Overrides' no se pueden declarar como 'Overridable' porque se pueden reemplazar implícitamente
A diferencia de la mayoría de los métodos, los métodos marcados con el modificador `Overrides` se pueden reemplazar de forma predeterminada.  
  
 **Identificador de error:** BC30730  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Ignore el modificador `Overridable` de los métodos marcados con el modificador `Overrides` .
