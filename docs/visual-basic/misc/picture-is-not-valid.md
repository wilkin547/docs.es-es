# La imagen no es v&#225;lida
Intentó asignar un formato gráfico distinto de un mapa de bits, un icono o un metarchivo de Windows a la propiedad `Picture` de un formulario o un control.  
  
### Para corregir este error  
  
1.  Asegúrese de que el archivo que está intentando cargar en la propiedad `Picture` es un archivo gráfico válido compatible con [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)].
