# Insuficientes argumentos de tipo para &#39;&lt;nombreM&#233;todoGen&#233;rico&gt;&#39;
Se ha invocado un método genérico con menos argumentos de tipo que los parámetros de tipo que hay.  
  
 Al invocar un método genérico, debe proporcionar un argumento de tipo para cada parámetro de tipo definido por el método.  
  
 **Identificador de error:** BC32042  
  
### Para corregir este error  
  
-   Agregue argumentos de tipo a la lista de argumentos de tipo para que haya uno para cada parámetro de tipo del método genérico que se está invocando.
