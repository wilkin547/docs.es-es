# El argumento &#39;&lt;argumentname&gt;&#39; debe estar en el intervalo entre -32768 y 65535
El valor de un argumento está fuera del intervalo permitido entre \-32768 y 65535.  
  
### Para corregir este error  
  
1.  Asegúrese de que el valor del argumento esté dentro del intervalo válido.  
  
2.  Si se facilita el argumento mediante un cálculo, compruebe los valores de los procesos de cálculo.
