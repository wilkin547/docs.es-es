# <a name="argument-argumentname-must-be-in-the-range-of--32768-to-65535"></a>Argumento '\<argumentname >' debe estar en el intervalo entre-32768 y 65535
El valor de un argumento está fuera del intervalo permitido entre -32768 y 65535.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Asegúrese de que el valor del argumento está dentro del intervalo válido.  
  
2.  Si se proporciona el argumento mediante un cálculo, compruebe los valores de los procesos de cálculo.
