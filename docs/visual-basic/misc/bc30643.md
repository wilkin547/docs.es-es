# La propiedad &#39;&lt;nombreDePropiedad&gt;&#39; es de un tipo no compatible
Una propiedad de un ensamblado de referencia contiene un tipo que no se puede representar.  
  
 **Identificador de error:** BC30643  
  
### Para corregir este error  
  
-   Especifique un tipo que se pueda usar con la aplicación.
