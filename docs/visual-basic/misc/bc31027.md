# <a name="unable-to-open-file-ltfilenamegt-lterrorgt"></a>No se puede abrir el archivo '&lt;filename&gt;': &lt;error&gt;
Se produjo un error al abrir un archivo en este proyecto.  
  
 **Identificador de error:** BC31027  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Compruebe que el archivo existe y que no está dañado o bloqueado por otra aplicación.
