# <a name="expression-is-not-an-array-or-a-method-and-cannot-have-an-argument-list"></a>La expresión no es una matriz ni un método y no puede tener una lista de argumentos

Ha facilitado una lista de argumentos para una expresión que no es una matriz ni un método. Solo las matrices y los métodos pueden tener listas de argumentos.  
  
 **Id. de error:** BC30471  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite la lista de argumentos.
