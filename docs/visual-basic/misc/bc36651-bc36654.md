<span data-ttu-id="08d4c-104">Los tipos de datos de los parámetros de tipo del método ' \<methodname> ' no se pueden inferir de estos argumentos porque hay más de un tipo posible.</span><span class="sxs-lookup"><span data-stu-id="08d4c-104">Data type(s) of the type parameter(s) in method '\<methodname>' cannot be inferred from these arguments because more than one type is possible.</span></span> <span data-ttu-id="08d4c-105">Especificar los tipos de datos explícitamente puede corregir este error.</span><span class="sxs-lookup"><span data-stu-id="08d4c-105">Specifying the data type(s) explicitly might correct this error.</span></span>

<span data-ttu-id="08d4c-106">Se ha intentado usar la inferencia de tipos para determinar el tipo o los tipos de datos del parámetro o los parámetros de tipo en una llamada a un procedimiento genérico.</span><span class="sxs-lookup"><span data-stu-id="08d4c-106">An attempt has been made to use type inference to determine the type or types of the type parameter or parameters in a call to a generic procedure.</span></span> <span data-ttu-id="08d4c-107">El compilador encuentra más de un tipo de datos para uno o varios de los parámetros de tipo e informa de este error.</span><span class="sxs-lookup"><span data-stu-id="08d4c-107">The compiler finds more than one possible data type for one or more of the type parameters, and it reports this error.</span></span>

> [!NOTE]
> <span data-ttu-id="08d4c-108">Al especificar argumentos no es una opción (por ejemplo, para operadores de consulta en expresiones de consulta), el mensaje de error aparece sin la segunda oración.</span><span class="sxs-lookup"><span data-stu-id="08d4c-108">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>

<span data-ttu-id="08d4c-109">El código siguiente muestra el error.</span><span class="sxs-lookup"><span data-stu-id="08d4c-109">The following code demonstrates the error.</span></span>

```vb

    Sub Main()
        '' Not valid.
        'targetMethod(1, "2")
    End Sub

    Sub targetMethod(Of T)(ByVal p1 As T, ByVal p2 As T)
    End Sub

End Module
```
