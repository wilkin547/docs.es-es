# La declaraci&#243;n &#39;RemoveHandler&#39; debe terminar con la declaraci&#243;n &#39;End RemoveHandler&#39; correspondiente
Una declaración `RemoveHandler` debe terminar con una instrucción `End RemoveHandler`.  
  
 **Identificador de error:** BC31116  
  
### Para corregir este error  
  
-   Compruebe que la declaración `RemoveHandler` finaliza con una instrucción `End RemoveHandler`.
