<span data-ttu-id="e02bf-103">En un método parcial que incluye uno o varios parámetros de tipo, los nombres de los parámetros de tipo deben ser iguales en la declaración del método y en la implementación del método.</span><span class="sxs-lookup"><span data-stu-id="e02bf-103">In a partial method that includes one or more type parameters, the names of the type parameters must be the same in the declaration of the method and in the implementation of the method.</span></span>  
  
 <span data-ttu-id="e02bf-104">Por ejemplo, la declaración e implementación siguientes provocan este error.</span><span class="sxs-lookup"><span data-stu-id="e02bf-104">For example, the following declaration and implementation cause this error.</span></span>  
  
