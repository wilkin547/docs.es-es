# <a name="variant-is-no-longer-a-supported-type-use-the-object-type-instead"></a>Ya no se admite el tipo 'Variant'; use el tipo 'Object' en su lugar.

El tipo de datos `Object` reemplaza a `Variant`.  
  
 **Identificador de error:** BC30804  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Declare el elemento como `Object`.
