# El par&#225;metro &lt;par&#225;metro&gt; de tipo de comentario XML no coincide con un par&#225;metro de tipo de la instrucci&#243;n &lt;palabra clave&gt; correspondiente.
El parámetro \<parámetro\> de comentario XML no coincide con un parámetro de tipo de la instrucción \<palabra clave\> correspondiente. Se omitirá el comentario XML.  
  
 Se especificó una etiqueta de comentario XML para algunos, pero no para todos los parámetros de tipo en un método.  
  
 **Identificador de error:** BC42317  
  
### Para corregir este error  
  
-   Proporcione etiquetas de comentario XML para todos los parámetros de tipo en el método.
