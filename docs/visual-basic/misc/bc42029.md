# <a name="39catch39-block-never-reached-because-39ltname1gt39-inherits-from-39ltname2gt39"></a>&#39;Detectar&#39; nunca se alcanzó el bloque, porque &#39; &lt;nombre1&gt; &#39; hereda &#39; &lt;nombre2&gt;&#39;
Un bloque `Catch` dentro del código nunca se alcanza.  
  
 De forma predeterminada, este mensaje es una advertencia. Para obtener más información sobre cómo ocultar las advertencias o cómo tratarlas como errores, vea [Configuring Warnings in Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **Identificador de error:** BC42029  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Elimine el bloque `Catch` superfluo.
