# <a name="namespace-or-type-name-has-already-been-imported"></a>Namespace o tipo '\<nombre >' ya se ha importado.
Se intentó importar un espacio de nombres más de una vez.  
  
 **Identificador de error:** BC31051  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite las instrucciones `Imports` redundantes.
