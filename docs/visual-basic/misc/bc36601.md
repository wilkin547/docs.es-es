# <a name="type-characters-cannot-be-used-in-range-variable-declarations"></a>No se pueden usar caracteres de tipo en declaraciones de variable de rango

Un carácter de tipo se incluyó al final del nombre de una declaración de variable de rango en una consulta LINQ.  
  
 **Identificador de error:** BC36601  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Quite el carácter de tipo de la declaración de variable de rango.
