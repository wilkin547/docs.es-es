# <a name="alias-namespacename-is-already-declared"></a>El alias '\<namespacename>' ya se ha declarado

Ha intentado usar `Imports` para importar un espacio de nombres que ya se ha declarado.  
  
 **Identificador de error:** BC30572  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Compruebe el espacio de nombres que está intentando importar.
