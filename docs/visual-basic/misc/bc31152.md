# <a name="xml-namespace-prefix-39prefix39-is-reserved-for-use-by-xml-and-the-namespace-uri-cannot-be-changed"></a>Prefijo de espacio de nombres XML &#39; prefijo &#39; está reservado para su uso por XML y el espacio de nombres que no se puede cambiar el URI
Un prefijo de espacio de nombres XML reservado, como `xmlns`, se usó en un literal XML.  
  
 **Identificador de error:** BC31152  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Cambie el prefijo del espacio de nombres XML a un valor que no esté reservado.
