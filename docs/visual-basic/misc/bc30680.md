# <a name="end-region-must-be-preceded-by-a-matching-region"></a>'#End Region' debe ir precedida de la instrucción '#Region' correspondiente
Con `#Region` puede especificar un bloque de código para expandir o contraer cuando se utiliza la característica de esquematización del Editor de código de Visual Studio. El inicio y fin de las instrucciones `#Region` deben estar en el mismo bloque de código.  
  
 **Identificador de error:** BC30680  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Inserte `#Region` en el lugar adecuado antes de la instrucción `#End Region` correspondiente.
