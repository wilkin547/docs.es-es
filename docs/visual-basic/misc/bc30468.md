# <a name="type-declaration-characters-are-not-valid-in-this-context"></a>Los caracteres de declaración de tipo no son válidos en este contexto
Intentó usar un carácter que no es válido en este contexto, como $.  
  
 **Identificador de error:** BC30468  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Quite el carácter.
