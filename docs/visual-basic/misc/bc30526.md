# <a name="property-propertyname-is-readonly"></a>La propiedad '\<propertyname>' es 'ReadOnly'

Ha intentado asignar un valor a una propiedad declarada como `ReadOnly`.  
  
 **Identificador de error:** BC30526  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite el especificador `ReadOnly` de declaración de propiedad.
