# <a name="data-types-of-the-type-parameters-in-extension-method-methodname-defined-in-typename-cannot-be-inferred-from-these-arguments-because-they-do-not-convert-to-the-same-type"></a><span data-ttu-id="6261f-103">Los tipos de datos de los parámetros de tipo del método de extensión '\<methodname>' definido en 'typename' no se pueden inferir de estos argumentos porque no se convierten en el mismo tipo</span><span class="sxs-lookup"><span data-stu-id="6261f-103">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in 'typename' cannot be inferred from these arguments because they do not convert to the same type</span></span>

<span data-ttu-id="6261f-104">Los tipos de datos de los parámetros de tipo del método de extensión ' \<methodname> ' definido en ' TypeName ' no se pueden inferir de estos argumentos porque no se convierten al mismo tipo.</span><span class="sxs-lookup"><span data-stu-id="6261f-104">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in 'typename' cannot be inferred from these arguments because they do not convert to the same type.</span></span> <span data-ttu-id="6261f-105">Especificar los tipos de datos explícitamente puede corregir este error.</span><span class="sxs-lookup"><span data-stu-id="6261f-105">Specifying the data type(s) explicitly might correct this error.</span></span>

<span data-ttu-id="6261f-106">Se ha intentado usar la inferencia de tipo para determinar el tipo (o tipos) de datos del parámetro (o parámetros) de tipo al evaluar una llamada a un método de extensión genérico.</span><span class="sxs-lookup"><span data-stu-id="6261f-106">An attempt has been made to use type inference to determine the data type (or types) of the type parameter (or parameters) when evaluating a call to a generic extension method.</span></span> <span data-ttu-id="6261f-107">El compilador no pudo encontrar un tipo de datos que cumpliera las restricciones de todos los argumentos.</span><span class="sxs-lookup"><span data-stu-id="6261f-107">The compiler could not find a data type that meets the constraints of all the arguments.</span></span> <span data-ttu-id="6261f-108">Por lo tanto, informa de este error.</span><span class="sxs-lookup"><span data-stu-id="6261f-108">Therefore, it reported this error.</span></span>

> [!NOTE]
> <span data-ttu-id="6261f-109">Al especificar argumentos no es una opción (por ejemplo, para operadores de consulta en expresiones de consulta), el mensaje de error aparece sin la segunda oración.</span><span class="sxs-lookup"><span data-stu-id="6261f-109">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>

<span data-ttu-id="6261f-110">El código siguiente muestra el error.</span><span class="sxs-lookup"><span data-stu-id="6261f-110">The following code demonstrates the error.</span></span>

```vb
Option Strict Off
Module Module3
    Sub Main()

        Dim c1 As New Class1

        '' Not valid. Integer and Date do not convert to the same type.
        'c1.targetMethod(19, #3/4/2007#)

    End Sub

    <System.Runtime.CompilerServices.Extension()> _
    Sub targetMethod(Of T)(ByVal p0 As Class1, ByVal p1 As T, ByVal p2 As T)
    End Sub

    Class Class1
    End Class

End Module
```

<span data-ttu-id="6261f-111">**Identificador de error:** BC36661 y BC36658</span><span class="sxs-lookup"><span data-stu-id="6261f-111">**Error ID:** BC36661 and BC36658</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="6261f-112">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="6261f-112">To correct this error</span></span>

