# <a name="xml-literals-and-xml-axis-properties-are-not-available-during-this-debugging-session-because-they-are-not-used-in-this-assembly"></a>Los literales XML y las propiedades de eje XML no están disponibles durante esta sesión de depuración porque no se utilizan en este ensamblado

Se ha hecho referencia a un literal XML o a una propiedad de eje XML en la ventana **inspección** o **inmediato** durante una sesión de depuración en la que no están disponibles las características de XML en Visual Basic. Este es el caso de un ensamblado que no utiliza el XML en Visual Basic características o es una compilación de versión.  
  
 **Id. de error:** BC31196  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Inicie una nueva sesión de depuración con la compilación de depuración del ensamblado.
