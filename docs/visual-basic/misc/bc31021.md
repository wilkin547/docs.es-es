# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-error"></a>El compilador de Visual Basic no se puede recuperar del error siguiente: \<error>

El compilador de Visual Basic no puede recuperarse del siguiente error: <`error`>.  
  
 Se han deshabilitado algunas características del IDE.  
  
 Se ha producido un error interno en el compilador de Visual Basic.  
  
 **Identificador de error:** BC31021  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Guarde el trabajo y reinicie Visual Studio.  
  
2. Si el error se repite, vuelva a instalar Visual Studio.  
  
3. Si el error persiste después de la reinstalación, informe al respecto a los Servicios de soporte técnico de Microsoft.
