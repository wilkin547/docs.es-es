# El compilador de Visual Basic no se puede recuperar del error siguiente: &lt;error&gt;.
El compilador [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] de Visual Basic no se puede recuperar del error siguiente: \<`error`\>.  
  
 Se han deshabilitado algunas características del IDE.  
  
 Se ha producido un error interno en el compilador [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)].  
  
 **Identificador de error:** BC31021  
  
### Para corregir este error  
  
1.  Guarde el trabajo y reinicie [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
2.  Si el error se repite, reinstale [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
3.  Si el error persiste después de la reinstalación, informe al respecto a los Servicios de soporte técnico de Microsoft.
