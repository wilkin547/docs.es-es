<span data-ttu-id="3dc28-104">No se puede inferir un tipo común para los operandos primero y segundo del operador 'If' binario.</span><span class="sxs-lookup"><span data-stu-id="3dc28-104">Cannot infer a common type for the first and second operands of the binary 'If' operator.</span></span> <span data-ttu-id="3dc28-105">Uno debe tener una conversión de ampliación al otro.</span><span class="sxs-lookup"><span data-stu-id="3dc28-105">One must have a widening conversion to the other.</span></span>

<span data-ttu-id="3dc28-106">El operador `If` binario requiere que exista una conversión de ampliación entre uno de los argumentos y el otro.</span><span class="sxs-lookup"><span data-stu-id="3dc28-106">The binary `If` operator requires that there be a widening conversion between one of the arguments and the other argument.</span></span> <span data-ttu-id="3dc28-107">Por ejemplo, como no existe una conversión de ampliación en ninguna dirección entre `Integer` y `String`, el código siguiente provoca este error.</span><span class="sxs-lookup"><span data-stu-id="3dc28-107">For example, because there is not a widening conversion in either direction between `Integer` and `String`, the following code causes this error.</span></span>

```vb
Dim second As String = "First is Nothing"
'' Not valid.
' Console.WriteLine(If(first, second))
```


  ```vb
  End If
  ```
