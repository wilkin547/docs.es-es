# <a name="source-file-ltfilenamegt-specified-multiple-times"></a>Archivo de código fuente &lt;nombre de archivo&gt; especificó varias veces
Un archivo se ha especificado varias veces.  
  
 **Identificador de error:** BC2002  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite los especificadores de archivos redundantes.
