# <a name="source-file-filename-specified-multiple-times"></a>El archivo de código fuente '\<filename>' se especificó varias veces

Un archivo se ha especificado varias veces.  
  
 **Identificador de error:** BC2002  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite los especificadores de archivos redundantes.
