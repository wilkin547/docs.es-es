# <a name="fully-qualified-names-for-types-in-namespaces-cannot-be-longer-than-ltnumbergt-characters"></a>Nombres completos para los tipos de espacios de nombres no pueden tener más de &lt;número&gt; caracteres
El nombre completo del tipo no puede superar el número de caracteres especificado.  
  
 **Identificador de error:** BC30031  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Acorte el nombre del tipo.
