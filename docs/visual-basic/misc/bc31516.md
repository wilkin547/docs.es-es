# <a name="type-typename-cannot-be-used-as-an-attribute-because-it-is-not-declared-public"></a>Tipo de '\<typename >' no se puede usar como atributo porque no está declarado como 'Public'.
Los atributos deben ser públicos.  
  
 **Identificador de error:** BC31516  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Agregue el modificador `Public` a la clase de atributo.
