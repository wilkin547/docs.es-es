# <a name="user-interrupt-occurred"></a>Se produjo una interrupción por parte del usuario

Un usuario presiona CTRL+ENTRAR u otra tecla de interrupción.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Vuelva a intentar la operación.
