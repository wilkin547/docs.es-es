# <a name="option-strict-on-does-not-allow-narrowing-in-implicit-type-conversions-between-the-lambda-expression-and-delegate-delegatename"></a><span data-ttu-id="ceacb-103">Option Strict on no permite la restricción en conversiones de tipo implícitas entre la expresión lambda y el delegado ' \<delegatename> '</span><span class="sxs-lookup"><span data-stu-id="ceacb-103">Option Strict On does not allow narrowing in implicit type conversions between the lambda expression and delegate '\<delegatename>'</span></span>

<span data-ttu-id="ceacb-104">Con `Option Strict` On, no puede haber una conversión de restricción entre el tipo de datos de un parámetro de un delegado y el parámetro correspondiente de una expresión lambda asignada a una variable de ese tipo de delegado.</span><span class="sxs-lookup"><span data-stu-id="ceacb-104">With `Option Strict` on, you cannot have a narrowing conversion between the data type of a parameter in a delegate and the corresponding parameter of a lambda expression assigned to a variable of that delegate type.</span></span> <span data-ttu-id="ceacb-105">Por ejemplo, en el código siguiente, el delegado `Del` tiene un parámetro de tipo `Integer`.</span><span class="sxs-lookup"><span data-stu-id="ceacb-105">For example, in the following code, delegate `Del` has one parameter of type `Integer`.</span></span>  
  
```vb  
Delegate Function Del(ByVal p As Integer) As String  
```  
  
 <span data-ttu-id="ceacb-106">Por lo tanto, el parámetro correspondiente de cualquier expresión lambda que se asigna a una variable de tipo `Del` puede ser `Integer` o cualquier tipo de datos para el que hay una conversión de ampliación desde `Integer`.</span><span class="sxs-lookup"><span data-stu-id="ceacb-106">Therefore, the corresponding parameter of any lambda expression assigned to a variable of type `Del` can be an `Integer` or any data type for which there is a widening conversion from `Integer`.</span></span>  
  
```vb  
' Valid.  
Dim example1 As Del = Function(n As Integer) "Valid"  
Dim example2 As Del = Function(n As Long) "Valid"  
  
' Not valid.  
Dim example3 As Del = Function(n As Short) "Not Valid"  
```  
  
 <span data-ttu-id="ceacb-107">**Identificador de error:** BC36662</span><span class="sxs-lookup"><span data-stu-id="ceacb-107">**Error ID:** BC36662</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="ceacb-108">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="ceacb-108">To correct this error</span></span>  
  
- <span data-ttu-id="ceacb-109">Cambie el tipo de datos del parámetro en el delegado o la expresión lambda para que exista la relación de ampliación necesaria.</span><span class="sxs-lookup"><span data-stu-id="ceacb-109">Change the data type of the parameter in the delegate or the lambda expression so that the required widening relationship exists.</span></span>  
  
- <span data-ttu-id="ceacb-110">No especifique tipos de datos de parámetro en la expresión lambda.</span><span class="sxs-lookup"><span data-stu-id="ceacb-110">Do not specify parameter data types in the lambda expression.</span></span> <span data-ttu-id="ceacb-111">Los tipos se inferirán de los parámetros correspondientes en el delegado.</span><span class="sxs-lookup"><span data-stu-id="ceacb-111">Types will be inferred from the corresponding parameters in the delegate.</span></span>  
  
```vb  
Dim example4 As Del = Function(n) "Valid"  
```
