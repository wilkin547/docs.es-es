# <a name="member-is-already-declared-in-this-structure"></a>' \<member> ' ya se ha declarado en esta estructura

Un miembro de la enumeración se declaró más de una vez en esta estructura.  
  
 **Identificador de error:** BC31421  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Elimine la declaración superflua.
