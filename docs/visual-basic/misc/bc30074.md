# La constante no puede ser el destino de una asignaci&#243;n.
Aparece una constante en un contexto que le asigna un valor. Solo se pueden asignar valores durante la ejecución a variables, propiedades y elementos de matriz que se pueden escribir.  
  
 **Identificador de error:** BC30074  
  
### Para corregir este error  
  
-   Reemplace la constante por una variable, una propiedad o un elemento de matriz de escritura única.
