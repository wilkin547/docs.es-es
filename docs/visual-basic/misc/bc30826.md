# <a name="endif-statements-are-no-longer-supported-use-end-if-instead"></a>Ya no se admiten instrucciones 'EndIf'; utilice 'End If' en su lugar

Ya no se admite la instrucción `EndIf` .  
  
 **Identificador de error:** BC30826  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Use `End If` para terminar las instrucciones `If` .
