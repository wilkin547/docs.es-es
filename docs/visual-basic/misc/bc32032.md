# <a name="end-function-must-be-the-first-statement-on-a-line"></a>'End Function' debe ser la primera instrucción de una línea.
Una instrucción `End Function` sigue a un separador de instrucciones de dos puntos (:). `End Function` debe ser la única instrucción en su línea de código fuente.  
  
 **Identificador de error:** BC32032  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Separe las distintas instrucciones en líneas diferentes.
