# <a name="name-cannot-expose-the-underlying-delegate-type-delegatetype-of-the-event-it-is-implementing-to-specifier1-type1-through-specifier2-type2"></a>' \<name> ' no puede exponer el tipo delegado subyacente ' \<delegatetype> ' del evento que implementa en \<specifier1> ' \<type1> ' a través de \<specifier2> ' \<type2> '

El código expone un tipo privado fuera de la clase pública.  
  
 **Identificador de error:** BC30914  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Declare el tipo `Public` o use otro tipo.
