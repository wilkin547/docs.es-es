# No se declar&#243; &#39;&lt;lsetstmt&gt;&#39;
No se declaró '\<lsetstmt\>'. Ya no se admiten instrucciones LSet; use Microsoft.VisualBasic.LSet en su lugar.  
  
 Varias funciones que eran intrínsecas de [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] en versiones anteriores se movieron al espacio de nombres <xref:Microsoft.VisualBasic?displayProperty=fullName>. Esto hace que sus funciones estén disponibles de manera más general para todos los lenguajes de programación.  
  
 **Identificador de error:** BC30820  
  
### Para corregir este error  
  
-   Use la función `LSet` del espacio de nombres `Microsoft.VisualBasic` en su lugar.
