# <a name="39ltnamegt39-is-ambiguous-between-declarations-in-modules-39ltmodulename1gt39-and-39ltmodulename2gt39"></a>&#39;&lt;nombre&gt; &#39; es ambiguo entre las declaraciones de módulos &#39; &lt;nombredemódulo1&gt; &#39; y &#39; &lt;nombredemódulo2&gt;&#39;
Se proporcionó un nombre que es ambiguo y, por tanto, entra en conflicto con otro nombre. El compilador de Visual Basic no tiene reglas de resolución de conflictos; deben eliminar la ambigüedad de los nombres usted mismo.  
  
 **Identificador de error:** BC30562  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Use el nombre completo.
