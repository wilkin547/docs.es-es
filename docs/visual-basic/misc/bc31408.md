# <a name="ltspecifier1gt-and-ltspecifier2gt-cannot-be-combined"></a>'&lt;specifier1&gt;'y'&lt;specifier2&gt;' no se pueden combinar
Dos especificadores compatibles, como `Shadows` y `Overrides` se usan juntos.  
  
 **Identificador de error:** BC31408  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite un especificador u otro.
