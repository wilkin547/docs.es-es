# <a name="argument-not-specified-for-parameter-39ltparameternamegt39"></a>Argumento no especificado para el parámetro &#39; &lt;parametername&gt;&#39;
No se proporcionó un argumento para un parámetro necesario.  
  
 **Identificador de error:** BC36569  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Proporcione un argumento para el parámetro especificado.
