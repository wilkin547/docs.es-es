# El valor &#39;&lt;nombreDeValor1&gt;&#39; no se puede convertir en &#39;&lt;nombreDeValor2&gt;&#39;
Se intentó una conversión de tipo entre dos tipos incompatibles.  
  
 **Identificador de error:** BC30742  
  
### Para corregir este error  
  
-   Compruebe los tipos que se están convirtiendo.
