# <a name="value-ltvaluename1gt-cannot-be-converted-to-ltvaluename2gt"></a>Valor '&lt;nombredevalor1&gt;'no se puede convertir a'&lt;nombredevalor2&gt;'
Se intentó una conversión de tipo entre dos tipos incompatibles.  
  
 **Identificador de error:** BC30742  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Compruebe los tipos que se van a convertir.
