# <a name="value-valuename1-cannot-be-converted-to-valuename2"></a>El valor ' \<valuename1> ' no se puede convertir en ' \<valuename2> '

Se intentó una conversión de tipo entre dos tipos incompatibles.  
  
 **Identificador de error:** BC30742  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Compruebe los tipos que se van a convertir.
