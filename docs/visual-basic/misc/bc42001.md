# Ya no se admite el evento &#39;Class_Initialize&#39;
Ya no se admite el evento 'Class\_Initialize'. Use 'Sub New' para inicializar una clase.  
  
 El evento `Class_Initialize` de versiones anteriores de [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] se sustituyó por constructores de clase.  
  
 De forma predeterminada, este mensaje es una advertencia. Para obtener información sobre cómo ocultar las advertencias o cómo tratarlas como errores, vea [Configurar advertencias en Visual Basic](/visual-studio/ide/configuring-warnings-in-visual-basic).  
  
 **Identificador de error:** BC42001  
  
### Para corregir este error  
  
-   Declare uno o varios procedimientos `Sub` denominados `New` para inicializar una clase. Se llama a `Sub New` cuando se crea una instancia de clase nueva.
