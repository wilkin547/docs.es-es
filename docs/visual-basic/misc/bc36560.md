# <a name="type-characters-cannot-be-used-in-anonymous-type-declarations"></a><span data-ttu-id="0f248-103">No se pueden usar caracteres de tipo en declaraciones de tipo anónimo</span><span class="sxs-lookup"><span data-stu-id="0f248-103">Type characters cannot be used in anonymous type declarations</span></span>

<span data-ttu-id="0f248-104">No puede usar un carácter de tipo en un nombre de propiedad cuando se declara una instancia de un tipo anónimo.</span><span class="sxs-lookup"><span data-stu-id="0f248-104">You cannot use a type character on a property name when you declare an instance of an anonymous type.</span></span> <span data-ttu-id="0f248-105">El tipo de datos de la propiedad se deduce del valor asignado a él.</span><span class="sxs-lookup"><span data-stu-id="0f248-105">The data type of the property is inferred from the value assigned to it.</span></span> <span data-ttu-id="0f248-106">Por ejemplo, las siguientes declaraciones no son válidas:</span><span class="sxs-lookup"><span data-stu-id="0f248-106">For example, the following declarations are not valid.</span></span>  
  
```vb  
'' Not valid.

'Dim anon2 = New With {.ID$ = 42}  
```  
  
 <span data-ttu-id="0f248-107">**Identificador de error:** BC36560</span><span class="sxs-lookup"><span data-stu-id="0f248-107">**Error ID:** BC36560</span></span>
