# &#39;ReadOnly&#39; y &#39;WriteOnly&#39; no se pueden combinar
Los modificadores `ReadOnly` y `WriteOnly` se excluyen mutuamente.  
  
 **Id. de error:** BC30635  
  
### Para corregir este error  
  
-   Especifique `ReadOnly` o `WriteOnly`, pero no ambos.
