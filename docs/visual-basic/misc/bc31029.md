# <a name="method-39ltmethodnamegt39-cannot-handle-event-39lteventnamegt39-because-they-do-not-have-the-same-signature"></a>Método &#39; &lt;methodname&gt;&#39; no puede controlar el evento &#39;&lt; EventName&gt;&#39; porque no tienen la misma firma
Los parámetros o los tipos de parámetro del controlador de eventos especificado no coinciden con los definidos para el evento especificado.  
  
 **Id. de error:** BC31029  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Vuelva a definir el evento o el controlador de eventos para que las listas de parámetros coincidan.
