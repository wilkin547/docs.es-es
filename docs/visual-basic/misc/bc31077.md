# <a name="assemblyname-is-an-assembly-and-cannot-be-referenced-as-a-module"></a>' \<assemblyname> ' es un ensamblado y no se puede hacer referencia a él como módulo

Se intentó usar un ensamblado como un módulo.  
  
 **Identificador de error:** BC31077  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Cambie la referencia a un módulo.
