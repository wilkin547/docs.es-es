# <a name="no-input-sources-specified"></a>No se especificó ningún origen de entrada

 **Identificador de error:** BC2008  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Asegúrese de incluir un archivo de código fuente.
