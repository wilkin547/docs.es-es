# <a name="39myclass39-cannot-be-used-outside-of-a-class"></a>&#39; MyClass &#39; no se puede usar fuera de una clase
Ha usado la palabra clave `MyClass` fuera de una estructura de clase.  
  
 **Id. de error:** BC30470  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Mueva la declaración `MyClass` a la estructura de clase.
