# <a name="class-classname-has-no-accessible-sub-new-and-cannot-be-inherited"></a><span data-ttu-id="bada5-103">La clase '\<classname>' no tiene 'Sub New' accesible y no se puede heredar</span><span class="sxs-lookup"><span data-stu-id="bada5-103">Class '\<classname>' has no accessible 'Sub New' and cannot be inherited</span></span>

<span data-ttu-id="bada5-104">Una clase utiliza un elemento [Inherits Statement](../language-reference/statements/inherits-statement.md) para especificar una clase base, pero no puede acceder a ningún constructor de la clase base deseada.</span><span class="sxs-lookup"><span data-stu-id="bada5-104">A class uses an [Inherits Statement](../language-reference/statements/inherits-statement.md) to specify a base class, but it cannot access any constructor on the intended base class.</span></span>  
  
 <span data-ttu-id="bada5-105">Esto puede ocurrir si la clase base deseada no tiene constructores o si tiene constructores con niveles de acceso que impiden el acceso desde otra clase.</span><span class="sxs-lookup"><span data-stu-id="bada5-105">This can happen if the intended base class has no constructors or if it has constructors with access levels that prevent access from another class.</span></span>  
  
 <span data-ttu-id="bada5-106">Cuando se hereda una clase, el constructor debe llamar al constructor de clase base mediante [MyBase](../programming-guide/program-structure/me-my-mybase-and-myclass.md#mybase).</span><span class="sxs-lookup"><span data-stu-id="bada5-106">When you inherit a class, your constructor should call the base class constructor using [MyBase](../programming-guide/program-structure/me-my-mybase-and-myclass.md#mybase).</span></span> <span data-ttu-id="bada5-107">Si no realiza esta llamada o ni siquiera escribe un constructor explícito, Visual Basic genera un constructor implícito que llama a `MyBase.New()`.</span><span class="sxs-lookup"><span data-stu-id="bada5-107">If you do not make this call, or if you do not even write an explicit constructor, Visual Basic generates an implicit constructor that calls `MyBase.New()`.</span></span>  
  
 <span data-ttu-id="bada5-108">**Id. de error:** BC31399</span><span class="sxs-lookup"><span data-stu-id="bada5-108">**Error ID:** BC31399</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="bada5-109">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="bada5-109">To correct this error</span></span>  
  
1. <span data-ttu-id="bada5-110">Si tiene control de código fuente sobre la clase base deseada, cambie el nivel de acceso de al menos uno de sus constructores para que otra clase pueda acceder a ellos.</span><span class="sxs-lookup"><span data-stu-id="bada5-110">If you have source control over the intended base class, then change the access level of at least one of its constructors so that another class can access them.</span></span>
