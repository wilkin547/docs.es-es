# <a name="unable-to-create-assembly-linker-object-error-message"></a>No se puede crear el objeto Assembly Linker: \<error message>

El compilador Visual Basic llama a Assembly Linker (Al.exe, también conocido como ALink) para generar un ensamblado con un manifiesto. El enlazador informa de un error al crear el objeto de trabajo. Podría tratarse de un problema al generar una referencia a un archivo de metadatos o un error al llamar a un archivo de importación.  
  
 **Identificador de error:** BC30141  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Examine el mensaje de error citado y consulte el tema [Al.exe](../../framework/tools/al-exe-assembly-linker.md) para obtener una explicación más detallada y consejos.  
  
2. Si el error persiste, reúna información sobre las circunstancias y notifíquelo a los Servicios de soporte técnico de Microsoft.
