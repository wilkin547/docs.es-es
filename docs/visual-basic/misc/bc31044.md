# <a name="events-declared-with-an-39as39-clause-must-have-a-delegate-type"></a>Los eventos declarados con una cláusula 'As' deben tener un tipo de delegado
Se especificó un tipo que no es un delegado.  
  
 **Id. de error:** BC31044  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Quite la cláusula `As` .  
  
2.  Especifique un delegado con la cláusula `As` .
