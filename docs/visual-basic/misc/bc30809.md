# <a name="wend-statements-are-no-longer-supported-use-end-while-statements-instead"></a>Ya no se admiten instrucciones 'Wend'; use instrucciones 'End While' en su lugar.

La construcción `End While` reemplaza a la palabra clave `Wend` .  
  
 **Identificador de error:** BC30809  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Use `End While` para finalizar un bloque `While` .
