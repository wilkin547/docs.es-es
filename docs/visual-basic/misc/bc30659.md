# <a name="property-or-field-name-does-not-have-a-valid-attribute-type"></a>La propiedad o el campo '\<name> no tiene un tipo de atributo válido

Es posible que esté utilizando un constructor de atributos que no acepta un tipo de atributo válido, aunque el atributo pueda tener otros constructores adecuados.  
  
 **Identificador de error:** BC30659  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Compruebe el uso de este atributo.
