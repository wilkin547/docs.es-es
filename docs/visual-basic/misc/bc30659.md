# <a name="property-or-field-ltnamegt-does-not-have-a-valid-attribute-type"></a>Propiedad o campo '&lt;nombre&gt;' no tiene un tipo de atributo válido
Puede que esté usando un constructor de atributos que no acepta un tipo de atributo válido, aun cuando el atributo pueda tener otros constructores adecuados.  
  
 **Identificador de error:** BC30659  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Compruebe el uso de este atributo.
