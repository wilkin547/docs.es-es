# <a name="the-value-value-is-invalid-for-option-optionname"></a>el valor ' \<value> ' no es válido para la opción ' \<optionname> '

Se proporcionó un valor para una opción de línea de comandos que no era válida.  
  
 **Identificador de error:** BC2014  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Compruebe los valores para encontrar el que no es válido.
