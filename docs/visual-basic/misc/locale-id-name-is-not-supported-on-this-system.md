# Este sistema no admite el id. de configuraci&#243;n regional &#39;&lt;nombre&gt;&#39;
Está intentando usar `VbStrConv`, pero el sistema no admite la configuración regional especificada.  
  
### Para corregir este error  
  
1.  Actualice la máquina para que admita la configuración regional o ejecute la aplicación en una máquina que la admita.
