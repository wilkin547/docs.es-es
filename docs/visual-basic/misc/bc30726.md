# <a name="39with39-contexts-and-statements-are-not-valid-in-debug-windows"></a>'Los contextos e instrucciones with' no son válidas en ventanas de depuración
`With`instrucciones sólo pueden usarse en código fuente.  
  
 **Identificador de error:** BC30726  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite las instrucciones `With` del código de depuración.
