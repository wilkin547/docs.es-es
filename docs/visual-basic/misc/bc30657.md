# <a name="39ltprocedurenamegt39-has-a-return-type-that-is-not-supported-or-parameter-types-that-are-not-supported"></a>'&lt;nombreProcedimiento&gt;' tiene un tipo de valor devuelto no es compatible o tipos de parámetro que no son compatibles
Se declara un procedimiento con parámetros o un tipo de valor devuelto no se puede representar por [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
 **Identificador de error:** BC30657  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Defina los tipos de devolución o los parámetros como un tipo compatible.  
  
-   Desarrolle un nuevo tipo que describa los datos usados por este procedimiento.
