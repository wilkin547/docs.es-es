# <a name="39end-sub39-expected"></a>&#39;End Sub&#39; esperado
Una instrucción `End` debe coincidir con el bloque que termina. Una definición de procedimiento `Sub` no se terminó correctamente.  
  
 **Identificador de error:** BC30026  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Agregue una instrucción `End Sub` al final del procedimiento.
