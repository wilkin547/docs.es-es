# La instrucci&#243;n &#39;#Region&#39; debe terminar con una instrucci&#243;n &#39;#End Region&#39; correspondiente
La directiva `#Region` permite especificar un bloque de código que puede expandir o contraer cuando use la característica de esquematización del Editor de código de [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]. El inicio y fin de las instrucciones `#Region` deben estar en el mismo bloque de código.  
  
 **Id.de error:** BC30681  
  
### Para corregir este error  
  
1.  Inserte `#End Region` en el lugar adecuado del código después de la instrucción `#Region`.
