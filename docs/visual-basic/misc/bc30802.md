# Ya no se admiten las instrucciones &#39;Type&#39;; use las instrucciones &#39;Structure&#39; en su lugar
La palabra clave `Type` ya no puede usarse para definir los tipos definidos por el usuario.  
  
 **Identificador de error:** BC30802  
  
### Para corregir este error  
  
-   Use la instrucción `Structure` para definir un nuevo tipo.
