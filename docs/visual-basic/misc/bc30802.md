# <a name="type-statements-are-no-longer-supported-use-structure-statements-instead"></a>Ya no se admiten las instrucciones 'Type'; use las instrucciones 'Structure' en su lugar

La palabra clave `Type` ya no puede usarse para definir los tipos definidos por el usuario.  
  
 **Identificador de error:** BC30802  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Use la instrucción `Structure` para definir un nuevo tipo.
