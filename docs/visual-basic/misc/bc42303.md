# <a name="xml-comment-cannot-appear-within-a-method-or-a-property"></a>El comentario XML no puede aparecer dentro de un método o de una propiedad

Un comentario XML no puede aparecer dentro de un método o de una propiedad. Se omitirá el comentario XML. Los bloques de comentarios XML no pueden aparecer dentro de métodos o de propiedades.  
  
 **Identificador de error:** BC42303  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite el bloque de comentario o muévalo a una ubicación más adecuada.
