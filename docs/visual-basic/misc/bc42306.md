# No se permite la etiqueta de comentario &#39;&lt;tag&gt;&#39; en un elemento de idioma &#39;&lt;element&gt;&#39;
No se permite la etiqueta de comentario \<tag\> en un elemento de idioma '\<element\>'. Se omitirá el comentario XML.  
  
 Una etiqueta de comentario XML no es compatible con el elemento de lenguaje al que se aplica.  
  
 **Identificador de error:** BC42306  
  
### Para corregir este error  
  
-   Quite la etiqueta de comentario XML no compatible.
