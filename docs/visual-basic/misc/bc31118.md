# <a name="end-addhandler-must-be-the-first-statement-on-a-line"></a>'End AddHandler' debe ser la primera instrucción de una línea

Una instrucción `End AddHandler` sigue a un separador de instrucciones de dos puntos (:). `End AddHandler` debe ser la única instrucción en su línea de código fuente.  
  
 **Identificador de error:** BC31118  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Separe las distintas instrucciones en líneas diferentes.
