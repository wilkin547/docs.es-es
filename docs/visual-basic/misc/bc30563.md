# &#39;&lt;nombre&gt;&#39; es ambiguo en los objetos de aplicaci&#243;n &#39;&lt;lista&gt;&#39;
Se proporcionó un nombre que es ambiguo y, por tanto, entra en conflicto con otro nombre. El compilador de [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] no tiene reglas de resolución de conflictos. Es necesario eliminar la ambigüedad en los nombres de forma manual. La lista proporcionada en el mensaje de error detalla los objetos de aplicación en los que el nombre es ambiguo.  
  
 **Identificador de error:** BC30563  
  
### Para corregir este error  
  
-   Use el nombre completo.
