# <a name="39select39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Seleccione&#39; instrucciones no son válidas en la ventana Inmediato.
Las instrucciones`Select` solo pueden usarse en el código fuente.  
  
 **Identificador de error:** BC30718  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite las instrucciones `Select` del código de depuración.
