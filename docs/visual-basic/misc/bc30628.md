# <a name="structures-cannot-have-inherits-statements"></a>Las estructuras no pueden tener instrucciones 'Inherits'.

A diferencia de las clases, las estructuras no admiten la herencia.  
  
 **Identificador de error:** BC30628  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite la instrucción `Inherits` de la estructura.  
  
- Vuelva a diseñar la aplicación para que use clases en su lugar.
