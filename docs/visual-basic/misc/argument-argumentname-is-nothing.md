# El argumento &#39;&lt;argumentname&gt;&#39; no es nada
Una expresión contiene un argumento nulo.  
  
### Para corregir este error  
  
1.  Compruebe la ortografía de los argumentos de la expresión. Un nombre de variable mal escrito puede crear implícitamente una variable numérica que se inicializa a cero.  
  
2.  Compruebe las operaciones anteriores sobre variables de la expresión, especialmente las que se pasan al procedimiento como argumentos de otros procedimientos.
