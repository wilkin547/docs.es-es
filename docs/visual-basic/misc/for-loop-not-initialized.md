# <a name="for-loop-not-initialized-visual-basic"></a>Bucle for no inicializado (Visual Basic)

Saltó al medio de un bucle `For...Next` .  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Quite el salto en el bucle.
