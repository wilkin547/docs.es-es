# <a name="39custom-event39-must-end-with-a-matching-39end-event39"></a>&#39;Evento personalizado&#39; debe terminar con una coincidencia &#39;End Event&#39;
Una declaración `Custom Event` debe terminar con una instrucción `End Event` .  
  
 **Identificador de error:** BC31114  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Compruebe que la declaración `Custom Event` finaliza con una instrucción `End Event` .
