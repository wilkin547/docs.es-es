# &#39;Option Explicit&#39; solo puede ir seguida de &#39;On&#39; u &#39;Off&#39;
Las instrucciones `Option Explicit` deben especificar `On` o `Off`. Todos los demás valores de entrada no son válidos.  
  
 **Identificador de error:** BC30640  
  
### Para corregir este error  
  
-   Especifique `Option Explicit On` o `Option Explicit Off`.
