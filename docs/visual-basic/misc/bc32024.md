# Los valores predeterminados no se pueden proporcionar para par&#225;metros no declarados como &#39;Optional&#39;.
Los argumentos requeridos deben proporcionarse cuando se llama al procedimiento, y no toman valores predeterminados en la declaración.  
  
 **Identificador de error:** BC32024  
  
### Para corregir este error  
  
-   Declare el argumento para que sea `Optional` o elimine el valor predeterminado.
