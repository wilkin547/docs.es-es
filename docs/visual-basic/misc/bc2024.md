# <a name="the-default-assembly-reference-39ltreferencegt39-has-already-been-automatically-added-ignored"></a>La referencia de ensamblado predeterminado &#39; &lt;referencia&gt; &#39; ha ya se agregó automáticamente; ignorada
Se realizó un intento de agregar la referencia de ensamblado predeterminada, pero el ensamblado ya se agregó, por lo que se omitió el intento.  
  
 **Identificador de error:** BC2024  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el código superfluo.
