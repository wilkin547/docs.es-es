# <a name="the-default-assembly-reference-reference-has-already-been-automatically-added-ignored"></a>Ya se ha agregado automáticamente la referencia de ensamblado predeterminado '\<reference>'; se ha omitido

Se realizó un intento de agregar la referencia de ensamblado predeterminada, pero el ensamblado ya se agregó, por lo que se omitió el intento.  
  
 **Identificador de error:** BC2024  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite el código superfluo.
