# <a name="array-initializer-is-missing-ltnumbergt-elements"></a>Inicializador de matriz es falta &lt;número&gt; elementos
No ha facilitado suficientes elementos en el inicializador de la matriz.  
  
 **Id. de error:** BC30567  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Facilite el argumento o los argumentos que faltan.  
  
-   Use `ReDim` para cambiar el tamaño de la matriz.
