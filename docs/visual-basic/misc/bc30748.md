# <a name="cannot-convert-to-39lttypegt39"></a>No se puede convertir en &#39; &lt;tipo&gt;&#39;
Se intentó realizar una conversión no válida.  
  
 **Identificador de error:** BC30748  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Elija un tipo de conversión compatible.
