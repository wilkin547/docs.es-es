# <a name="cannot-convert-to-type"></a>No se puede convertir a '\<tipo >'
Se intentó realizar una conversión no válida.  
  
 **Identificador de error:** BC30748  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Elija un tipo de conversión compatible.
