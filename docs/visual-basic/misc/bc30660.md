# <a name="attributes-cannot-be-applied-to-local-variables"></a>No se pueden aplicar los atributos a las variables locales

No se pueden aplicar los atributos a las variables locales.  
  
 **Identificador de error:** BC30660  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Declare esta variable en el nivel de ámbito siguiente, como el nivel de clase o módulo.
