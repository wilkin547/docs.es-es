# <a name="withevents-variable-does-not-raise-any-instance-events-that-are-accessible-to-containername"></a>'Variable de WithEvents no genera eventos de instancia que se puede acceder a'\<containername >'
El objeto al que hace referencia no genera eventos accesibles.  
  
 **Identificador de error:** BC30600  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Modifique los modificadores de acceso para el evento o elija un evento que sea accesible en el contexto actual.
