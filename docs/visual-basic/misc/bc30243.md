# &#39;&lt;modificador&gt;&#39; no es v&#225;lido en una declaraci&#243;n de evento
Una instrucción `Event` contiene una palabra clave no válida, como `ReadOnly`.  
  
 **Identificador de error:** BC30243  
  
### Para corregir este error  
  
1.  Quite la palabra clave no válida de la instrucción `Event`.
