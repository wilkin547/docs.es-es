# Try debe tener al menos una instrucci&#243;n &#39;Catch&#39; o &#39;Finally&#39;
Un bloque `Try` debe incluir un bloque `Finally` o al menos un bloque `Catch` antes de la instrucción `End Try`.  
  
 **Identificador de error:** BC30030  
  
### Para corregir este error  
  
-   Agregue un bloque `Catch` o `Finally` en el bloque `Try`.
