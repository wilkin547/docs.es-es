# Los operadores de conversi&#243;n no pueden convertir de un tipo de interfaz
Un operador de conversión se declara con un tipo de interfaz para el parámetro.  
  
 En tiempo de compilación, [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] considera que existe una conversión predefinida desde cualquier interfaz a cualquier tipo de referencia. Este tipo de conversión puede producir un error en tiempo de ejecución, pero el compilador no puede predecir los resultados en tiempo de ejecución, por lo que permite que estas conversiones se compilen.  
  
 Dado que el compilador considera que esta conversión ya está definida, no permite la redefinición.  
  
 **Identificador de error:** BC33029  
  
### Para corregir este error  
  
-   Quite completamente esta definición de operador. Ya está predefinido.
