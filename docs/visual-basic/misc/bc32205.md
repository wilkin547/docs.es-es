# La longitud del nombre de la variable WithEvents no puede superar los 1019 caracteres
El nombre o el identificador de cada variable declarada mediante `WithEvents` se limita a 1019 caracteres.  
  
 **Identificador de error:** BC32205  
  
### Para corregir este error  
  
-   Reduzca la longitud del identificador.
