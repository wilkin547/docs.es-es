## <a name="different-lifetimes"></a><span data-ttu-id="28ab4-108">Diferentes períodos de duración</span><span class="sxs-lookup"><span data-stu-id="28ab4-108">Different Lifetimes</span></span>  
 <span data-ttu-id="28ab4-109">A *variable miembro* (declarado en el nivel de módulo, fuera de cualquier procedimiento) normalmente tiene la misma duración que el elemento en el que se declara.</span><span class="sxs-lookup"><span data-stu-id="28ab4-109">A *member variable* (declared at module level, outside any procedure) typically has the same lifetime as the element in which it is declared.</span></span> <span data-ttu-id="28ab4-110">Una variable no compartida declarada en una clase o estructura existe como una copia independiente para cada instancia de la clase o estructura en la que se ha declarado.</span><span class="sxs-lookup"><span data-stu-id="28ab4-110">A nonshared variable declared in a class or structure exists as a separate copy for each instance of the class or structure in which it is declared.</span></span> <span data-ttu-id="28ab4-111">Cada variable tiene la misma duración que su instancia.</span><span class="sxs-lookup"><span data-stu-id="28ab4-111">Each such variable has the same lifetime as its instance.</span></span> <span data-ttu-id="28ab4-112">Sin embargo, un `Shared` variable tiene solo una duración, que tiene una validez de todo el tiempo que se ejecuta la aplicación.</span><span class="sxs-lookup"><span data-stu-id="28ab4-112">However, a `Shared` variable has only a single lifetime, which lasts for the entire time your application is running.</span></span>  
  
 <span data-ttu-id="28ab4-113">A *variable local* (declarada dentro de un procedimiento) sólo existe mientras se ejecuta el procedimiento en el que se declara.</span><span class="sxs-lookup"><span data-stu-id="28ab4-113">A *local variable* (declared inside a procedure) exists only while the procedure in which it is declared is running.</span></span> <span data-ttu-id="28ab4-114">Esto también se aplica a los parámetros del procedimiento y a cualquier valor devuelto de función.</span><span class="sxs-lookup"><span data-stu-id="28ab4-114">This applies also to that procedure's parameters and to any function return.</span></span> <span data-ttu-id="28ab4-115">Sin embargo, si ese procedimiento llama a otros procedimientos, las variables locales conservan sus valores mientras se ejecutan los procedimientos llamados.</span><span class="sxs-lookup"><span data-stu-id="28ab4-115">However, if that procedure calls other procedures, the local variables retain their values while the called procedures are running.</span></span>  
  
## <a name="beginning-of-lifetime"></a><span data-ttu-id="28ab4-116">Principio de la duración</span><span class="sxs-lookup"><span data-stu-id="28ab4-116">Beginning of Lifetime</span></span>  
 <span data-ttu-id="28ab4-117">Duración de una variable local se inicia cuando el control entra en el procedimiento en el que se declara.</span><span class="sxs-lookup"><span data-stu-id="28ab4-117">A local variable's lifetime begins when control enters the procedure in which it is declared.</span></span> <span data-ttu-id="28ab4-118">Cada variable local se inicializa en el valor predeterminado para su tipo de datos tan pronto como comienza el procedimiento ejecuta.</span><span class="sxs-lookup"><span data-stu-id="28ab4-118">Every local variable is initialized to the default value for its data type as soon as the procedure begins running.</span></span> <span data-ttu-id="28ab4-119">Cuando se encuentra el procedimiento una `Dim` instrucción que especifica valores iniciales, Establece esas variables en esos valores, incluso si el código ya tenía asignado otros valores para ellas.</span><span class="sxs-lookup"><span data-stu-id="28ab4-119">When the procedure encounters a `Dim` statement that specifies initial values, it sets those variables to those values, even if your code had already assigned other values to them.</span></span>  
  
 <span data-ttu-id="28ab4-120">Cada miembro de una variable de estructura se inicializa como si fuera una variable independiente.</span><span class="sxs-lookup"><span data-stu-id="28ab4-120">Each member of a structure variable is initialized as if it were a separate variable.</span></span> <span data-ttu-id="28ab4-121">Del mismo modo, cada elemento de una variable de matriz se inicializa de manera individual.</span><span class="sxs-lookup"><span data-stu-id="28ab4-121">Similarly, each element of an array variable is initialized individually.</span></span>
