# <a name="constant-expression-not-representable-in-type-typename"></a>Expresión constante no se puede representar en el tipo '\<typename >'
Está intentando evaluar una constante que no encaja en el tipo de destino, normalmente porque desborda el intervalo.  
  
 **Identificador de error:** BC30439  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Cambiar el tipo de destino para que pueda controlar la constante.
