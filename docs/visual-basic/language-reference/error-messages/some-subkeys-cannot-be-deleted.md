# <a name="some-subkeys-cannot-be-deleted"></a>Algunas subclaves no se pueden eliminar

Se intentó eliminar una clave del registro, pero se produjo un error en la operación porque algunas subclaves no se pueden eliminar. Normalmente esto se debe a una falta de permisos.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Asegúrese de que tiene permisos suficientes para eliminar las subclaves especificadas.
