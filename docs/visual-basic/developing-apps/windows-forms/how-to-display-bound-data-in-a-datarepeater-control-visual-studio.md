### <a name="to-create-a-data-bound-datarepeater"></a><span data-ttu-id="1761f-109">Para crear un DataRepeater enlazado a datos</span><span class="sxs-lookup"><span data-stu-id="1761f-109">To create a data-bound DataRepeater</span></span>  
  
1.  <span data-ttu-id="1761f-110">Arrastre un <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> controlar desde la **Visual Basic PowerPacks** pestaña en el **cuadro de herramientas** a un control de formulario o contenedor.</span><span class="sxs-lookup"><span data-stu-id="1761f-110">Drag a <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control from the **Visual Basic PowerPacks** tab in the **Toolbox** to a form or container control.</span></span>  
  
2.  <span data-ttu-id="1761f-111">Arrastre las asas de ajuste de tamaño y posición a tamaño y posición del control.</span><span class="sxs-lookup"><span data-stu-id="1761f-111">Drag the sizing and position handles to size and position the control.</span></span>  
  
     <span data-ttu-id="1761f-112">Tenga en cuenta que el control tiene dos regiones rectangulares.</span><span class="sxs-lookup"><span data-stu-id="1761f-112">Note that the control has two rectangular regions.</span></span> <span data-ttu-id="1761f-113">La región superior es el *plantilla de elemento*; los controles agregados a la plantilla se repetirá en cada elemento en el <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control en tiempo de ejecución.</span><span class="sxs-lookup"><span data-stu-id="1761f-113">The upper region is the *item template*; controls added to the template will be repeated in each item in the <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control at run time.</span></span> <span data-ttu-id="1761f-114">La región inferior está la *ventanilla*, donde se mostrarán los elementos.</span><span class="sxs-lookup"><span data-stu-id="1761f-114">The lower region is the *viewport*, where the items will be displayed.</span></span>  
  
     <span data-ttu-id="1761f-115">También puede cambiar el tamaño y colocarla en el control o la plantilla de elemento cambiando el **tamaño** y **posición** propiedades en la ventana Propiedades.</span><span class="sxs-lookup"><span data-stu-id="1761f-115">You can also size and position the control or the item template by changing the **Size** and **Position** properties in the Properties window.</span></span>  
  
3.  <span data-ttu-id="1761f-116">En el menú **Datos** , haga clic en **Mostrar orígenes de datos**.</span><span class="sxs-lookup"><span data-stu-id="1761f-116">On the **Data** menu, click **Show Data Sources**.</span></span>  
  
    > [!NOTE]
    >  <span data-ttu-id="1761f-117">Si el **orígenes de datos** ventana está vacía, agregue un origen de datos a él.</span><span class="sxs-lookup"><span data-stu-id="1761f-117">If the **Data Sources** window is empty, add a data source to it.</span></span> <span data-ttu-id="1761f-118">Para obtener más información, vea [Agregar nuevos orígenes de datos](/visualstudio/data-tools/add-new-data-sources).</span><span class="sxs-lookup"><span data-stu-id="1761f-118">For more information, see [Add new data sources](/visualstudio/data-tools/add-new-data-sources).</span></span>  
  
4.  <span data-ttu-id="1761f-119">En el **orígenes de datos** ventana, seleccione el nodo de nivel superior de la tabla que contiene los datos que desea enlazar.</span><span class="sxs-lookup"><span data-stu-id="1761f-119">In the **Data Sources** window, select the top-level node for the table that contains the data that you want to bind.</span></span>  
  
5.  <span data-ttu-id="1761f-120">Cambiar el tipo de colocación de la tabla a `Details` haciendo `Details` en la lista desplegable en el nodo de tabla.</span><span class="sxs-lookup"><span data-stu-id="1761f-120">Change the drop type of the table to `Details` by clicking `Details` in the drop-down list on the table node.</span></span>  
  
6.  <span data-ttu-id="1761f-121">Seleccione el nodo de tabla y arrástrelo a la región de la plantilla de elemento de la <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control.</span><span class="sxs-lookup"><span data-stu-id="1761f-121">Select the table node and drag it onto the item template region of the <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control.</span></span>  
  
     <span data-ttu-id="1761f-122">Puede especificar qué tipos de controles se muestran para cada campo.</span><span class="sxs-lookup"><span data-stu-id="1761f-122">You can specify which types of controls are displayed for each field.</span></span> <span data-ttu-id="1761f-123">Para obtener más información, consulte [establecer el control que se creará al arrastrar desde la ventana Orígenes de datos](/visualstudio/data-tools/set-the-control-to-be-created-when-dragging-from-the-data-sources-window).</span><span class="sxs-lookup"><span data-stu-id="1761f-123">For more information, see [Set the control to be created when dragging from the Data Sources window](/visualstudio/data-tools/set-the-control-to-be-created-when-dragging-from-the-data-sources-window).</span></span>
