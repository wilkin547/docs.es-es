<span data-ttu-id="29757-104">El almacén de instancias de flujo de trabajo de SQL ejecuta una tarea interna que se inicia periódicamente y que detecta instancias de flujo de trabajo ejecutables o activables en la base de datos de persistencia.</span><span class="sxs-lookup"><span data-stu-id="29757-104">The SQL Workflow Instance Store runs an internal task that periodically wakes up and detects runnable or activatable workflow instances in the persistence database.</span></span> <span data-ttu-id="29757-105">Si encuentra una instancia de flujo de trabajo ejecutable, notifica al host de flujo de trabajo que es capaz de activar la instancia.</span><span class="sxs-lookup"><span data-stu-id="29757-105">If it finds a runnable workflow instance, it notifies the workflow host that is capable of activating the instance.</span></span> <span data-ttu-id="29757-106">Si el almacén de instancias encuentra una instancia de flujo de trabajo activable, notifica a un host genérico que activa un host del flujo de trabajo, que a su vez ejecuta la instancia de flujo de trabajo.</span><span class="sxs-lookup"><span data-stu-id="29757-106">If the instance store finds an activatable workflow instance, it notifies a generic host that activates a workflow host, which in turn runs the workflow instance.</span></span> <span data-ttu-id="29757-107">Las siguientes secciones en este tema explican el proceso de activación de la instancia en detalle.</span><span class="sxs-lookup"><span data-stu-id="29757-107">The following sections in this topic explain the instance activation process in detail.</span></span>
