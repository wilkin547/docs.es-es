# <a name="1140---flowchartstart"></a>1140 - FlowchartStart

## <a name="properties"></a>Propiedades  
  
|||  
|-|-|  
|ID|1140|  
|Palabras clave|WFActivities|  
|Nivel|Información|  
|Canal|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Descripción  

 Indica que se ha programado el inicio de un diagrama de flujo.  
  
## <a name="message"></a>Message  

 Flowchart '%1': se ha programado el inicio.  
  
## <a name="details"></a>Detalles  
  
|Nombre del elemento de datos|Tipo del elemento de datos|Descripción|  
|--------------------|--------------------|-----------------|  
|Diagrama de flujo|xs:string|El nombre para mostrar del diagrama de flujo.|  
|AppDomain|xs:string|La cadena devuelta por AppDomain.CurrentDomain.FriendlyName.|
