|||  
|-|-|  
|ID|2028|  
|Palabras clave|WFRuntime|  
|Nivel|Verbose|  
|Canal|Microsoft-Windows-Application Server-Applications/Debug|

 CacheRootMetadata se detiene en la actividad %1.
