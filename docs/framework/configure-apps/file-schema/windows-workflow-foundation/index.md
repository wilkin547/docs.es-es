# <a name="windows-workflow-foundation-configuration-schema"></a>Esquema de configuración de Windows Workflow Foundation

Los elementos de configuración de Windows Workflow Foundation (WF) le permiten configurar aplicaciones de flujo de trabajo. Para una aplicación de flujo de trabajo puede configurar, entre otras cosas, el seguimiento y la traza. Para obtener más información sobre el seguimiento y la traza, vea [Workflow Tracking and Tracing](../../../windows-workflow-foundation/workflow-tracking-and-tracing.md) (Seguimiento y traza del flujo de trabajo). En el caso de los servicios de flujo de trabajo, también puede usar elementos de configuración de Windows Communication Foundation (WCF). Para obtener más información acerca de WCF, consulte [esquema de configuración de WCF](../wcf/index.md).  
  
 Puesto que los archivos de configuración tienen el formato XML, debe familiarizarse con el formato XML si desea modificarlos manualmente con un editor de texto. De lo contrario, puede encontrarse con problemas como no encontrar un atributo o una etiqueta de elemento XML. Esto se debe a que los atributos y las etiquetas del elemento XML distinguen entre mayúsculas y minúsculas.
