# <a name="tlsstreamm_worker-field"></a>TlsStream.m_Worker campo

Representa el estado de la secuencia SSL.

## <a name="syntax"></a>Sintaxis

```csharp
private SslState m_Worker;
```

## <a name="field-value"></a>Valor de campo

`System.Net.Security.SslState`  
El estado de la secuencia SSL.

## <a name="remarks"></a>Observaciones

> [!WARNING]
> El `TlsStream.m_Worker` campo es privado y no está diseñado para usarse directamente en el código.
>
> Microsoft no admite el uso de este campo en una aplicación de producción en cualquier circunstancia.

## <a name="requirements"></a>Requisitos

**Espacio de nombres:** <xref:System.Net>

**Ensamblado:** Sistema (en System.dll)

**.NET Framework versiones:** Disponible desde 2,0.
