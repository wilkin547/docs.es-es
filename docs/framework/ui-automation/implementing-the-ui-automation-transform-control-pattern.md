## <a name="implementation-guidelines-and-conventions"></a><span data-ttu-id="a9e2c-109">Directrices y convenciones de implementación</span><span class="sxs-lookup"><span data-stu-id="a9e2c-109">Implementation Guidelines and Conventions</span></span>
