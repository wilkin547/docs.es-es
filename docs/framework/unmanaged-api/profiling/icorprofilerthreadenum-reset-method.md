# <a name="icorprofilerthreadenumreset-method"></a>ICorProfilerThreadEnum::Reset (Método)
Mueve el cursor del enumerador a la posición inicial de la secuencia.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp  
HRESULT Reset();  
```
