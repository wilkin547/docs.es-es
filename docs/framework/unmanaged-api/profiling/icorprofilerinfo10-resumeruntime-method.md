# <a name="icorprofilerinfo10resumeruntime-method"></a>ICorProfilerInfo10:: ResumeRuntime (método)

Reanuda el tiempo de ejecución sin realizar un GC.

## <a name="syntax"></a>Sintaxis

```cpp
HRESULT ResumeRuntime();
```
