# <a name="icorprofilerfunctionenumreset-method"></a>ICorProfilerFunctionEnum::Reset (Método)

Mueve el cursor del enumerador a la posición inicial de la secuencia.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp  
HRESULT Reset();  
```
