# <a name="icordebugvariablehomegetlocationtype-method"></a>ICorDebugVariableHome:: GetLocationType (método)
Obtiene el tipo de la ubicación nativa de la variable.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp  
HRESULT GetLocationType(  
    [out] VariableLocationType *pLocationType  
);  
```  
  
## <a name="parameters"></a>Parámetros  
 `pLocationType`  
 enuncia Puntero al tipo de la ubicación nativa de la variable.  Vea la enumeración [VariableLocationType](../../../../docs/framework/unmanaged-api/debugging/variablelocationtype-enumeration.md) para obtener más información.
