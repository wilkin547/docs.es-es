# <a name="idebuggerinfo-interface"></a>IDebuggerInfo (Interfaz)

Proporciona métodos para obtener información sobre el estado de los servicios de depuración.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[Método IsDebuggerAttached](idebuggerinfo-isdebuggerattached-method.md)|Obtiene un valor que indica si un depurador administrado está asociado a este proceso.|
