# <a name="iobjecthandle-interface"></a>IObjectHandle (Interfaz)

Proporciona un método para desempaquetar objetos de cálculo de referencias por valor desde el direccionamiento indirecto.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[Método Unwrap](iobjecthandle-unwrap-method.md)|Desencapsula un objeto de cálculo de referencias por valor desde el direccionamiento indirecto.|
