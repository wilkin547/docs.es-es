# <a name="imetadatatablesgetrow-method"></a>IMetaDataTables::GetRow (Método)
Gets the row at the specified row index, in the table at the specified table index.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp  
HRESULT GetRow (   
    [in]  ULONG   ixTbl,  
    [in]  ULONG   rid,  
    [out] void    **ppRow  
);  
```  
  
## <a name="parameters"></a>Parámetros  
 `ixTbl`  
 [in] The index of the table from which the row will be retrieved.  
  
 `rid`  
 [in] The index of the row to get.  
  
 `ppRow`  
 [out] A pointer to a pointer to the row.  
  
## <a name="remarks"></a>Comentarios  
 We do not recommend the use of this method, because it does not return consistent results.
