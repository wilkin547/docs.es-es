# <a name="itypelibresolver-interface"></a>ITypeLibResolver (Interfaz)
Proporciona el [ResolveTypeLib (método)](../../../../docs/framework/unmanaged-api/tlbexp/resolvetypelib-method.md), que resuelve la ruta de acceso de archivo de una biblioteca de tipos.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[ResolveTypeLib (método)](../../../../docs/framework/unmanaged-api/tlbexp/resolvetypelib-method.md)|Resuelve el nombre sencillo de una biblioteca de tipos mediante la devolución de su ruta de acceso completa.|
