# <a name="iinstallreferenceitem-interface"></a>IInstallReferenceItem (Interfaz)

Representa un elemento instalado en la caché global de ensamblados.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[Método GetReference](iinstallreferenceitem-getreference-method.md)|Obtiene un puntero a la `FUSION_INSTALL_REFERENCE` estructura representada por este `IInstallReferenceItem` .|
