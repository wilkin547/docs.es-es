  
 <span data-ttu-id="fb0dd-185">Si desea mezclar código transparente y crítico en el mismo ensamblado, empiece marcando el ensamblado con el atributo <xref:System.Security.SecurityCriticalAttribute> para indicar que el ensamblado puede contener código crítico, como se indica a continuación:</span><span class="sxs-lookup"><span data-stu-id="fb0dd-185">If you want to mix critical and transparent code in the same assembly, start by marking the assembly with the <xref:System.Security.SecurityCriticalAttribute> attribute to indicate that the assembly can contain critical code, as follows:</span></span>
