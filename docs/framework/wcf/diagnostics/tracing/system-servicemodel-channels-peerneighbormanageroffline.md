# <a name="systemservicemodelchannelspeerneighbormanageroffline"></a>System.ServiceModel.Channels.PeerNeighborManagerOffline
PeerNode está ahora sin conexión, lo que indica que no hay vecinos actualmente conectados.  
  
## <a name="description"></a>Descripción  
 Este seguimiento se produce en la eliminación de un vecino de la lista de vecinos del nodo local.
