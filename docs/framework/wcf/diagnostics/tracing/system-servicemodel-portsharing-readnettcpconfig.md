System.ServiceModel.PortSharing.ReadNetTcpConfig  
  
## <a name="description"></a>Descripción  

 Los datos de configuración para NetTcpSection se han leído con los datos especificados:  listenBacklog, maxConnectionsPendingDispatch, maxPendingAccepts, receiveTimeout y teredoEnabled.
