# <a name="systemruntimeserializationobjectwithlargedepth"></a>System.Runtime.Serialization.ObjectWithLargeDepth

System.Runtime.Serialization.ObjectWithLargeDepth  
  
## <a name="description"></a>Descripción  

 Al escribir un objeto con una estructura recursiva, se tienen limitaciones cuando es de una gran profundidad. Considere la posibilidad de reducir la profundidad del objeto.
