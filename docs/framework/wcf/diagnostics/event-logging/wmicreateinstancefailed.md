# <a name="wmicreateinstancefailed"></a>WmiCreateInstanceFailed

Id: 123  
  
 Gravedad: error  
  
 Categoría: ServiceModel  
  
## <a name="description"></a>Descripción  

 Este evento indica que no se procesó la solicitud CreateInstance de WMI. El evento enumera el nombre de clase, excepción, nombre del proceso e identificador del proceso.
