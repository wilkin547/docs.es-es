# <a name="wmiunregistrationfailed"></a>WmiUnregistrationFailed

ID.: 127  
  
 Gravedad: error  
  
 Categoría: ServiceModel  
  
## <a name="description"></a>Descripción  

 Este evento indica que no se eliminó del registro al proveedor WMI. El evento enumera el objeto WMI, el error, el nombre de proceso y el id. de proceso.
