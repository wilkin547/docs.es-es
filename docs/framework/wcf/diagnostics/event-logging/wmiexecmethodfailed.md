# <a name="wmiexecmethodfailed"></a>WmiExecMethodFailed

ID.: 125  
  
 Gravedad: error  
  
 Categoría: ServiceModel  
  
## <a name="description"></a>Descripción  

 Este evento indica que no se procesó la solicitud ExecMethod de WMI. El evento enumera la excepción, el nombre del proceso y el id. del mismo.
