# <a name="wmigetobjectfailed"></a>WmiGetObjectFailed
Id.: 120  
  
 Gravedad: Error  
  
 Categoría: ServiceModel  
  
## <a name="description"></a>Descripción  
 Este evento indica que no se procesó la solicitud GetObject de WMI. El evento enumera el objeto WMI, excepción, nombre del proceso e identificador del proceso.
