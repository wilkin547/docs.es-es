# <a name="lafailedtolistenforapp"></a>LAFailedToListenForApp

ID.: 160  
  
 Gravedad: error  
  
 Categoría: SMSvcHost  
  
## <a name="description"></a>Descripción  

 Este evento indica que se ha producido un error en el servicio de activación especificado del protocolo especificado al intentar escuchar el sitio especificado, por lo que el protocolo está deshabilitado temporalmente para el sitio. El evento enumera la dirección URL, el estado, la excepción, el nombre del proceso y el identificador del proceso.
