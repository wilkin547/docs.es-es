# <a name="invariantassertionfailed"></a>InvariantAssertionFailed
Id.: 106  
  
 Gravedad: error  
  
 Categoría: seguimiento  
  
## <a name="description"></a>Descripción  
 Este evento indica que una aserción invariable es falsa. El evento enumera el mensaje, el nombre del proceso y el identificador del proceso.
