# <a name="3430---websocketuseversionfromclientwebsocketfactory"></a>3430 - WebSocketUseVersionFromClientWebSocketFactory

## <a name="properties"></a>Propiedades  
  
|||  
|-|-|  
|Id.|3430|  
|Palabras clave|HTTP|  
|Nivel|Verbose|  
|Canal|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Descripción  

 Este evento se genera cuando se usa WebSocketVersion de un generador de WebSocket cliente.  
  
## <a name="message"></a>Message  

 Usando el WebSocketVersion de un generador de WebSocket cliente de tipo '%1'.
