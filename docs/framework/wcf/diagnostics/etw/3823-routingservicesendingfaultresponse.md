## <a name="properties"></a>Propiedades  
  
|||  
|-|-|  
|Id.|3823|  
|Palabras clave|RoutingServices|  
|Nivel|Advertencia|  
|Canal|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Descripción  

 Este evento se genera cuando el servicio de enrutamiento está devolviendo un mensaje de respuesta de error.  
  
## <a name="message"></a>Message  

 El servicio de enrutamiento está devolviendo un mensaje de respuesta de error con la acción '%1'.
