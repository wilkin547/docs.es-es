# <a name="core-communications-transport-framework"></a>Comunicaciones básicas: marco de transporte

En este tema se hace una lista de todas las excepciones generadas por el marco de transporte.  
  
## <a name="exception-list"></a>Lista de excepciones  
  
|Código de recurso|Cadena de recurso|  
|-------------------|---------------------|  
|Hosting_TransportBindingNotFound|Ningún enlace de protocolo coincide con la dirección especificada. Los enlaces de protocolo se configuran en el nivel del sitio en la configuración de los servicios de Internet Information Server o del Servicio de activación de procesos de Windows.|
