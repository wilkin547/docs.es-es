## <a name="secure-channel-factory"></a><span data-ttu-id="2fb21-117">Generador de canales seguro</span><span class="sxs-lookup"><span data-stu-id="2fb21-117">Secure Channel Factory</span></span>

         [KeyId="xs:base64Binary"]?

  ...
