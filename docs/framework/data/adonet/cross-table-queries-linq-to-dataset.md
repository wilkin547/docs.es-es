data-stu-id="7f893-104">This is done by using a *join*.</span></span> <span data-ttu-id="7f893-105">Una combinación es la asociación de objetos en un origen de datos con objetos que comparten un source with objects that share a common attribute in another data source, such as a product or contact ID.</span></span> <span data-ttu-id="7f893-106">En la programación orientada a objetos, las <span data-ttu-id="7f893-107">Sin embargo, en tablas de bases de datos externas, navegar por relaciones no es tan sencillo.</span><span class="sxs-lookup"><span data-stu-id="7f893-107">In external class="sxs-lookup"><span data-stu-id="7f893-108">Database tables do not contain built-in relationships.</span></span> <span data-ttu-id="7f893-109">En estos casos, la operación de combinación se puede información de ventas y producto del mismo pedido de ventas.</span><span class="sxs-lookup"><span data-stu-id="7f893-110">For example, given two tables that contain product information and sales  <span data-ttu-id="7f893-111">El marco de trabajo [!INCLUDE[vbteclinqext](../../../../includes/vbteclinqext-md.md)] proporciona dos operadores de combinación, <xref:System.Linq.Enumerable.Join%2A> y <xref:System.Linq.Enumerable.GroupJoin%2A>.</span><span class="sxs-lookup"><span data-stu-id="7f893-111">The [!INCLUDE[vbteclinqext](../../../../includes/vbteclinqext-md.md)] framework provides two join operators, <xref:System.Linq.Enumerable.Join%2A> and <xref:System.Linq.Enumerable.GroupJoin%2A>.</span></span> <span data-ttu-id="7f893-112">Estos operadores realizan *combinaciones de igualdad*: es decir, combinaciones que hacen coincidir dos orígenes únicamente cuando sus claves son iguales.</span><span class="sxs-lookup"><span data-stu-id="7f893-112">These operators perform *equi-joins*: that is, joins that match two data sources only when their keys are equal.</span></span> <span data-ttu-id="7f893-113">(por el contrario, [!INCLUDE[tsql](../../../../includes/tsql-md.md)] admite operadores de combinación distintos de `equals`, como el operador `less than`).</span><span class="sxs-lookup"><span data-stu-id="7f893-113">(By contrast, [!INCLUDE[tsql](../../../../includes/tsql-md.md)] supports join operators other than `equals`, such as the `less than` operator.)</span></span>  
  
 <span data-ttu-id="7f893-114">En cuanto a las bases de datos relacionales, <xref:System.Linq.Enumerable.Join%2A> implementa una combinación interna.</span><span class="sxs-lookup"><span data-stu-id="7f893-114">In relational database terms, <xref:System.Linq.Enumerable.Join%2A> implements an inner join.</span></span> <span data-ttu-id="7f893-115">Una combinación interna es un tipo de combinación en la que solamente se devuelven los objetos que tienen una correspondencia en el conjunto de datos opuesto.</span><span class="sxs-lookup"><span data-stu-id="7f893-115">An inner join is a type of join in which only those objects that have a match in the opposite data set are returned.</span></span>  
  
 <span data-ttu-id="7f893-116">El <xref:System.Linq.Enumerable.GroupJoin%2A> operadores no tienen ningún equivalente directo en términos de bases de datos relacionales; implementan un superconjunto de combinaciones internas y combinaciones externas izquierdas.</span><span class="sxs-lookup"><span data-stu-id="7f893-116">The <xref:System.Linq.Enumerable.GroupJoin%2A> operators have no direct equivalent in relational database terms; they implement a superset of inner joins and left outer joins.</span></span> <span data-ttu-id="7f893-117">Una combinación externa izquierda es una combinación que devuelve cada elemento de la primera colección (izquierda), aunque éste no tenga elementos correlacionados en la segunda colección.</span><span class="sxs-lookup"><span data-stu-id="7f893-117">A left outer join is a join that returns each element of the first (left) collection, even if it has no correlated elements in the second collection.</span></span>  
  
 <span data-ttu-id="7f893-118">Para obtener más información acerca de las combinaciones, vea [operaciones de combinación](https://msdn.microsoft.com/library/442d176d-028c-4beb-8d22-407d4ef89107).</span><span class="sxs-lookup"><span data-stu-id="7f893-118">For more information about joins, see [Join Operations](https://msdn.microsoft.com/library/442d176d-028c-4beb-8d22-407d4ef89107).</span></span>
