### <a name="to-generate-the-school-model-for-a-visual-basic-project-using-edmgenexe"></a><span data-ttu-id="4024f-116">Para generar el modelo School para un proyecto de Visual Basic con EdmGen.exe</span><span class="sxs-lookup"><span data-stu-id="4024f-116">To generate the School model for a Visual Basic project using EdmGen.exe</span></span>

