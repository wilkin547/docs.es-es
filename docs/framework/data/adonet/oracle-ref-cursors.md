-   <span data-ttu-id="faabd-114">El orden de los cursores REF cursor en el **OracleDataReader** depende del orden de los parámetros en el **OracleParameterCollection**.</span><span class="sxs-lookup"><span data-stu-id="faabd-114">The order of REF CURSORs in the **OracleDataReader** depends on the order of the parameters in the **OracleParameterCollection**.</span></span> <span data-ttu-id="faabd-115">Se omite la propiedad <xref:System.Data.OracleClient.OracleParameter.ParameterName%2A>.</span><span class="sxs-lookup"><span data-stu-id="faabd-115">The <xref:System.Data.OracleClient.OracleParameter.ParameterName%2A> property is ignored.</span></span>  
  
-   <span data-ttu-id="faabd-116">PL/SQL **tabla** no se admite el tipo de datos.</span><span class="sxs-lookup"><span data-stu-id="faabd-116">The PL/SQL **TABLE** data type is not supported.</span></span> <span data-ttu-id="faabd-117">No obstante, los cursores REF CURSOR resultan más eficientes.</span><span class="sxs-lookup"><span data-stu-id="faabd-117">However, REF CURSORs are more efficient.</span></span> <span data-ttu-id="faabd-118">Si debe utilizar un **tabla** tipo de datos, utilice el proveedor de datos OLE DB .NET con MSDAORA.</span><span class="sxs-lookup"><span data-stu-id="faabd-118">If you must use a **TABLE** data type, use the OLE DB .NET Data Provider with MSDAORA.</span></span>
