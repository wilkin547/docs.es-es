## <a name="user-instances"></a><span data-ttu-id="c1252-114">Instancias de usuario</span><span class="sxs-lookup"><span data-stu-id="c1252-114">User Instances</span></span>

 <span data-ttu-id="c1252-118">Una instancia de usuario se genera desde una instancia principal de SQL Server o SQL Server Express en nombre de un usuario.</span><span class="sxs-lookup"><span data-stu-id="c1252-118">A user instance is generated from a primary instance of SQL Server or SQL Server Express on behalf of a user.</span></span> <span data-ttu-id="c1252-119">Se ejecuta como un proceso del usuario en el contexto de seguridad de Windows del usuario, no como un servicio.</span><span class="sxs-lookup"><span data-stu-id="c1252-119">It runs as a user process under the Windows security context of the user, not as a service.</span></span> <span data-ttu-id="c1252-120">No se permiten inicios de sesión de SQL Server; únicamente se admiten los inicios de sesión de Windows.</span><span class="sxs-lookup"><span data-stu-id="c1252-120">SQL Server logins are disallowed; only Windows logins are supported.</span></span> <span data-ttu-id="c1252-121">De esta manera se evita que haya software que se ejecuta en una instancia de usuario que efectúe cambios en todo el sistema para los que el usuario no debería tener permisos.</span><span class="sxs-lookup"><span data-stu-id="c1252-121">This prevents software executing on a user instance from making system-wide changes that the user would not have permissions to make.</span></span> <span data-ttu-id="c1252-122">La instancia de usuario también es conocida como instancia secundaria o de cliente, y a veces es denominada utilizando el acrónimo RANU (del inglés "run as normal user", ejecutar como usuario normal).</span><span class="sxs-lookup"><span data-stu-id="c1252-122">A user instance is also known as a child or client instance, and is sometimes referred to by using the RANU acronym ("run as normal user").</span></span>
