 <span data-ttu-id="d6878-108">En el nivel intermedio, las entidades son creadas por el contexto de los datos, el cual realiza el seguimiento de su estado, y administra la carga aplazada desde, y el class="sxs-lookup"><span data-stu-id="d6878-109">These entities are "attached" to the `DataContext`.</span></span> <span data-ttu-id="d6878-110">Sin embargo, una vez que las entidades se han enviado a <span data-ttu-id="d6878-111">Las entidades que el cliente devuelve para las actualizaciones se deben volver a asociar al contexto de los datos antes de que optimista.</span><span class="sxs-lookup"><span data-stu-id="d6878-112">The client is responsible for providing original values and/or timestamps back to the middle tier if those are required for
