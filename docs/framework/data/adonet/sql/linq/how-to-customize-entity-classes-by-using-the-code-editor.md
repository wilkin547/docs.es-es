# <a name="how-to-customize-entity-classes-by-using-the-code-editor"></a>Procedimiento para personalizar clases de entidades con el editor de código

Los desarrolladores que usan Visual Studio pueden usar el Object Relational Designer para crear o personalizar las clases de entidad.  
  
 También puede usar el editor de código de Visual Studio para escribir su propio código de asignación o para personalizar el código que ya se ha generado. Para obtener más información, consulte [asignación basada en atributos](attribute-based-mapping.md).
