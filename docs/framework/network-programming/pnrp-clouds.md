 <span data-ttu-id="e6035-105">La comunicación entre los nodos nunca debería cruzar de una nube a otra.</span><span class="sxs-lookup"><span data-stu-id="e6035-105">Communication between nodes should never cross from one cloud to another.</span></span> <span data-ttu-id="e6035-106">Una instancia <xref:System.Net.PeerToPeer.Cloud> se identifica por su nombre, que distingue mayúsculas de minúsculas.</span><span class="sxs-lookup"><span data-stu-id="e6035-106">A <xref:System.Net.PeerToPeer.Cloud> instance is uniquely identified by its name, which is case-sensitive.</span></span> <span data-ttu-id="e6035-107">Un único nodo o elemento del mismo nivel puede estar conectado a más de una nube.</span><span class="sxs-lookup"><span data-stu-id="e6035-107">A single peer or node may be connected to more than one cloud.</span></span>  
  
 <span data-ttu-id="e6035-108">Las nubes están estrechamente unidas a las interfaces de red.</span><span class="sxs-lookup"><span data-stu-id="e6035-108">Clouds are tied very closely to network interfaces.</span></span>  <span data-ttu-id="e6035-109">En un equipo de hosts múltiples con dos tarjetas de red asociadas a subredes diferentes, se devolverán tres nubes: una para cada una de las direcciones locales de vínculo por interfaz y una nube de ámbito global única.</span><span class="sxs-lookup"><span data-stu-id="e6035-109">On a multi-homed machine with two network cards attached to different subnets, three clouds will be returned: one for each of the link local addresses per interface, and a single global scope cloud.</span></span>  
  
 <span data-ttu-id="e6035-110">PNRP usa tres "ámbitos" de nube, donde un ámbito es una agrupación de equipos que se pueden encontrar los unos a los otros:</span><span class="sxs-lookup"><span data-stu-id="e6035-110">PNRP uses three cloud "scopes", in which a scope is a grouping of computers that are able to find each other:</span></span>  
  
-   <span data-ttu-id="e6035-111">La nube global se corresponde con el ámbito de dirección IPv6 global y las direcciones globales y representa todos los equipos de Internet IPv6.</span><span class="sxs-lookup"><span data-stu-id="e6035-111">The global cloud corresponds to the global IPv6 address scope and global addresses and represents all the computers on the entire IPv6 Internet.</span></span> <span data-ttu-id="e6035-112">Solo hay una nube global.</span><span class="sxs-lookup"><span data-stu-id="e6035-112">There is only a single global cloud.</span></span>  
  
-   <span data-ttu-id="e6035-113">La nube local de vínculo se corresponde con el ámbito de dirección IPv6 local del vínculo y las direcciones locales del vínculo.</span><span class="sxs-lookup"><span data-stu-id="e6035-113">The link-local cloud corresponds to the link-local IPv6 address scope and link-local addresses.</span></span> <span data-ttu-id="e6035-114">Las nubes locales de vínculo están destinadas a un vínculo específico, que suele ser el mismo que la subred asociada localmente.</span><span class="sxs-lookup"><span data-stu-id="e6035-114">A link-local cloud is for a specific link, which is typically the same as the locally attached subnet.</span></span> <span data-ttu-id="e6035-115">Puede haber varias nubes locales de vínculo.</span><span class="sxs-lookup"><span data-stu-id="e6035-115">There can be multiple link-local clouds.</span></span>  
  
 <span data-ttu-id="e6035-116">La tercera nube, que es la nube específica del sitio, se corresponde con el ámbito de dirección IPv6 de sitio y las direcciones locales del sitio.</span><span class="sxs-lookup"><span data-stu-id="e6035-116">A third cloud, the site-specific cloud, corresponds to the site IPv6 address scope and site-local addresses.</span></span> <span data-ttu-id="e6035-117">Esta nube está en desuso, aunque todavía se admite en PNRP.</span><span class="sxs-lookup"><span data-stu-id="e6035-117">This cloud has been deprecated, although it is still supported in PNRP.</span></span>  
  
## <a name="clouds"></a><span data-ttu-id="e6035-118">Nubes</span><span class="sxs-lookup"><span data-stu-id="e6035-118">Clouds</span></span>
