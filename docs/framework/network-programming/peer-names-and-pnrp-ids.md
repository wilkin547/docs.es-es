## <a name="peer-names"></a><span data-ttu-id="a4d87-106">Nombres de mismo nivel</span><span class="sxs-lookup"><span data-stu-id="a4d87-106">Peer Names</span></span>  

 <span data-ttu-id="a4d87-107">Los nombres de mismo nivel se pueden registrar como no seguros o seguros.</span><span class="sxs-lookup"><span data-stu-id="a4d87-107">Peer names can be registered as unsecured or secured.</span></span> <span data-ttu-id="a4d87-108">Los nombres no seguros son simplemente cadenas de texto que están sujetas a la suplantación de identidad, dado que cualquier usuario puede registrar un nombre no seguro duplicado.</span><span class="sxs-lookup"><span data-stu-id="a4d87-108">Unsecured names are just text strings that are subject to spoofing, as anyone can register a duplicate unsecured name.</span></span> <span data-ttu-id="a4d87-109">Los nombres no seguros son útiles en redes privadas o bien, en otros casos, protegidas.</span><span class="sxs-lookup"><span data-stu-id="a4d87-109">Unsecured names are best used in private or otherwise protected networks.</span></span> <span data-ttu-id="a4d87-110">Los nombres seguros están protegidos con un certificado y una firma digital.</span><span class="sxs-lookup"><span data-stu-id="a4d87-110">Secured names are protected with a certificate and a digital signature.</span></span> <span data-ttu-id="a4d87-111">Solo el publicador original podrá demostrar la propiedad de un nombre seguro.</span><span class="sxs-lookup"><span data-stu-id="a4d87-111">Only the original publisher will be able to prove ownership of a secured name.</span></span>  
  
 <span data-ttu-id="a4d87-112">La combinación de la nube y el ámbito proporciona un entorno razonablemente seguro para equipos del mismo nivel que participan en la actividad de PNRP.</span><span class="sxs-lookup"><span data-stu-id="a4d87-112">The combination of cloud and scope provides a reasonably secure environment for peers that participate in PNRP activity.</span></span> <span data-ttu-id="a4d87-113">Sin embargo, el uso de un nombre seguro de mismo nivel no garantiza la seguridad global de la aplicación de red.</span><span class="sxs-lookup"><span data-stu-id="a4d87-113">However, using a secured peer name does not ensure the overall security of the networking application.</span></span> <span data-ttu-id="a4d87-114">La seguridad de la aplicación depende de la implementación.</span><span class="sxs-lookup"><span data-stu-id="a4d87-114">Security of the application is implementation-dependent.</span></span>  
  
 <span data-ttu-id="a4d87-115">Los nombres de mismo nivel seguros solo se registran por su propietario y están protegidos con criptografía de clave pública.</span><span class="sxs-lookup"><span data-stu-id="a4d87-115">Secured peer names are only registered by their owner and are protected with public key cryptography.</span></span> <span data-ttu-id="a4d87-116">Un nombre seguro de mismo nivel se considera que pertenecen a la entidad de mismo nivel que tiene la clave privada correspondiente.</span><span class="sxs-lookup"><span data-stu-id="a4d87-116">A secured peer name is considered owned by the peer entity having the corresponding private key.</span></span> <span data-ttu-id="a4d87-117">La propiedad se puede probar a través de la dirección de mismo nivel certificada (CPA), que se firma con la clave privada.</span><span class="sxs-lookup"><span data-stu-id="a4d87-117">Ownership can be proved via the certified peer address (CPA), which is signed using the private key.</span></span> <span data-ttu-id="a4d87-118">Un usuario malintencionado no puede falsificar la propiedad de un nombre de mismo nivel sin la clave privada correspondiente.</span><span class="sxs-lookup"><span data-stu-id="a4d87-118">A malicious user cannot forge ownership of a peer name without the corresponding private key.</span></span>  
  
## <a name="pnrp-ids"></a><span data-ttu-id="a4d87-119">Identificadores PNRP</span><span class="sxs-lookup"><span data-stu-id="a4d87-119">PNRP IDs</span></span>

PNRP](./media/fdc9e8a0-4a1c-488d-a019-bc3a1973220c.gif "fdc9e8a0-4a1c-488d-a019-bc3a1973220c")</span><span class="sxs-lookup"><span data-stu-id="a4d87-120">![PNRP ID](./media/fdc9e8a0-4a1c-488d-a019-bc3a1973220c.gif "fdc9e8a0-4a1c-488d-a019-bc3a1973220c")</span></span>  
  
 <span data-ttu-id="a4d87-121">Los identificadores PNRP se componen de lo siguiente:</span><span class="sxs-lookup"><span data-stu-id="a4d87-121">PNRP IDs are composed of the following:</span></span>
