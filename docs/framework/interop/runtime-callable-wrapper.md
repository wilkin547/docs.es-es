# <a name="runtime-callable-wrapper"></a><span data-ttu-id="59bf1-102">Contenedor al que se puede llamar en tiempo de ejecución</span><span class="sxs-lookup"><span data-stu-id="59bf1-102">Runtime Callable Wrapper</span></span>

<span data-ttu-id="59bf1-110">Acceso a objetos COM mediante el contenedor RCW</span><span class="sxs-lookup"><span data-stu-id="59bf1-110">Accessing COM objects through the runtime callable wrapper</span></span>  
  
 <span data-ttu-id="59bf1-111">Usando los metadatos derivados de una biblioteca de tipos, CLR crea el objeto COM al que se está llamando y un contenedor para dicho objeto.</span><span class="sxs-lookup"><span data-stu-id="59bf1-111">Using metadata derived from a type library, the runtime creates both the COM object being called and a wrapper for that object.</span></span> <span data-ttu-id="59bf1-112">Cada contenedor RCW mantiene una memoria caché de punteros de interfaz en el objeto COM que contiene y libera su referencia en el objeto COM cuando el contenedor RCW ya no es necesario.</span><span class="sxs-lookup"><span data-stu-id="59bf1-112">Each RCW maintains a cache of interface pointers on the COM object it wraps and releases its reference on the COM object when the RCW is no longer needed.</span></span> <span data-ttu-id="59bf1-113">CLR realiza la recolección de elementos no utilizados en el contenedor RCW.</span><span class="sxs-lookup"><span data-stu-id="59bf1-113">The runtime performs garbage collection on the RCW.</span></span>
