## <a name="application-and-component-isolation"></a><span data-ttu-id="7ed26-116">Aislamiento de aplicación y componente</span><span class="sxs-lookup"><span data-stu-id="7ed26-116">Application and Component Isolation</span></span>  
 <span data-ttu-id="7ed26-117">El aislamiento es una de las claves en el diseño de una aplicación o componente para la ejecución en paralelo.</span><span class="sxs-lookup"><span data-stu-id="7ed26-117">One key to successfully designing an application or component for side-by-side execution is isolation.</span></span> <span data-ttu-id="7ed26-118">La aplicación o componente debe administrar todos los recursos —especialmente la E/S de archivos— de forma aislada.</span><span class="sxs-lookup"><span data-stu-id="7ed26-118">The application or component must manage all resources, particularly file I/O, in an isolated manner.</span></span> <span data-ttu-id="7ed26-119">Siga estas instrucciones para asegurarse de que su aplicación o componente se ejecuta de forma aislada:</span><span class="sxs-lookup"><span data-stu-id="7ed26-119">Follow these guidelines to make sure your application or component runs in isolation:</span></span>  
  
-   <span data-ttu-id="7ed26-120">Escriba en el registro de una manera específica de la versión.</span><span class="sxs-lookup"><span data-stu-id="7ed26-120">Write to the registry in a version-specific way.</span></span> <span data-ttu-id="7ed26-121">Almacene los valores en subárboles o claves que indiquen la versión y no comparta información ni el estado entre las versiones de un componente.</span><span class="sxs-lookup"><span data-stu-id="7ed26-121">Store values in hives or keys that indicate the version, and do not share information or state across versions of a component.</span></span> <span data-ttu-id="7ed26-122">Esto evita que dos aplicaciones o componentes que se ejecutan al mismo tiempo sobrescriban la información.</span><span class="sxs-lookup"><span data-stu-id="7ed26-122">This prevents two applications or components running at the same time from overwriting information.</span></span>
