  - <span data-ttu-id="43ede-115">[dotnet-sql-cache](https://github.com/dotnet/aspnetcore/blob/master/src/Tools/dotnet-sql-cache/README.md) (Microsoft.Extensions.Caching.SqlConfig.Tools)</span><span class="sxs-lookup"><span data-stu-id="43ede-115">[dotnet-sql-cache](https://github.com/dotnet/aspnetcore/blob/master/src/Tools/dotnet-sql-cache/README.md) (Microsoft.Extensions.Caching.SqlConfig.Tools)</span></span>
