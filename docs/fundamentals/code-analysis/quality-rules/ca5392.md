## <a name="pseudo-code-examples"></a><span data-ttu-id="da360-125">Ejemplos de pseudocódigo</span><span class="sxs-lookup"><span data-stu-id="da360-125">Pseudo-code examples</span></span>

{
    [DllImport("The3rdAssembly.dll")]

### <a name="solution"></a><span data-ttu-id="da360-126">Solución</span><span class="sxs-lookup"><span data-stu-id="da360-126">Solution</span></span>

```csharp

class ExampleClass
{
    [DllImport("The3rdAssembly.dll")]
    [DefaultDllImportSearchPaths(DllImportSearchPath.UserDirectories)]
    public static extern int MessageBox(IntPtr hWnd, String text, String caption, uint type);

    public void ExampleMethod()
    {
        MessageBox(new IntPtr(0), "Hello World!", "Hello Dialog", 0);
    }
}
```
